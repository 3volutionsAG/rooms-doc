Knowledge Base (Deutsch) : Buchung löschen

Verwandte Seiten

 

Einleitung

Ziel des Löschens ist es, eine Buchung komplett aus dem System zu entfernen (vergl. dazu Buchung stornieren (annullieren).

Vorbedingungen

Benutzer: Benötigt Rechte um das gewählte Menü zu sehen und mindestens Leserechte für die darin verfügbaren Ressourcen.

Administratoren: Finden einen detaillierten Beschrieb im KB Artikel Rechte.

Vorgehen

In den drei Menüs PLAN, KALENDER und LISTEN ist es möglich, eine Buchung zu löschen.

 

Löschen im Menü PLAN

  • Über den Standortbaum oder die Plansymbole zur gewünschten Buchung navigieren.
  • Klick auf das rote Symbol.
  • Im anschliessend erscheinenden Popup Klick auf Button Löschen.

  • Es erscheint ein Popup, welches nachfragt ob man die Buchung wirklich löschen möchte.

  • Bestätigung durch Klick auf Button OK.

  • Oben rechts erscheint eine Hinweisbox, welche das erfolgreiche Löschen bestätigt.


Löschen im Menü KALENDER

  • In der Kalenderansicht zur gewünschten Buchung navigieren.
  • Klick auf das „+“-Symbol.
  • Im anschliessend erscheinenden Popup Klick auf Button Löschen.


  • Es erscheint ein Popup, welches nachfragt ob man die Buchung wirklich löschen möchte.

  • Bestätigung durch Klick auf Button OK.

  • Oben rechts erscheint eine Hinweisbox, welche das erfolgreiche Löschen bestätigt.

 

 

Löschen im Menü LISTEN

  • In der Listenansicht zur gewünschten Buchung navigieren.
  • Wenn in der Spaltenreihenfolge so eingestellt, erscheint die Spalte "Löschen" mit dem Icon Löschen.
  • Klick auf das Icon Löschen.

  • Es erscheint ein Popup, welches nachfragt ob man die Buchung wirklich löschen möchte.

  • Bestätigung durch Klick auf Button OK.

  • Oben rechts erscheint eine Hinweisbox, welche das erfolgreiche Löschen bestätigt.
