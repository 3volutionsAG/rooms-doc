Einleitung

Falls das System entsprechend konfiguriert ist, kann ein Benutzer sich am System registrieren und danach mit Benutzernamen und Passwort anmelden.

Vorbedingungen

System muss entsprechend konfiguriert sein - Benutzer müssen das System erreichen können (richtige Serverzone muss gewählt werden)

In der Systemkonfiguration sollte ein Standardbenutzer (siehe KB Konfigurationsparameter Parameter: Vorlagebenutzer für Anmeldeformular) hinterlegt werden, von welchem die neu registrierten Benutzer das Profil  und die Benutzergruppenzuteilung "erben"
