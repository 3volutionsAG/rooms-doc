Vorgehen / Beschrieb

Darstellung Feiertage:

Per GARAIO-Default ein hell-gelber Balken resp. Symbol (die Farbe kann global geändert werden), ohne Text im Balken, aber mit Tooltip (bei Mouse-Over über den Balken resp. Symbol). Das Tooltip beinhaltet folgende Informationen in formatierter Form (untereinander):

  • Bezeichnung (Feiertag)
  • Von Datum / Von Zeit
  • Bis Datum / Bis Zeit

    Klick auf den Balken mit linker Maustaste und Buchung möglich (wird ein Feiertag gebucht, muss zusätzlich eine Meldung im Action Panel ausgegeben werden) Darstellung Wochenende:
    Per GARAIO-Default ein hell-gelber Balken resp. Symbol (die Farbe kann global geändert werden – d.h. hier muss ein globaler Konfigurationseintrag erstellt werden), ohne Text im Balken, aber mit Tooltip (bei Mouse-Over über den Balken resp. Symbol).
  • Bezeichnung (Wochenende resp. Wochenend-Tag)

    Klick auf den Balken mit rechter Maustaste und Buchung möglich (wird an einem Wochenende gebucht, muss zusätzlich eine Meldung im Action Panel ausgegeben werden).

Darstellung Sperrzeiten:

Per Standard ein anthrazit-grauer Balken resp. Symbol (die Farbe kann global geändert werden - d.h. hier muss ein globaler Konfigurationseintrag erstellt werden), ohne Text im Balken, aber mit Tooltip (bei Mouse-Over über den Balken resp. Symbol). Das Tooltip beinhaltet folgende Informationen in formatierter Form (untereinander):

  • Bezeichnung (Sperrzeiten)
  • Von Datum / Von Zeit
  • Bis Datum / Bis Zeit

    Klick auf den Balken mit rechter Maustaste und Buchung NICHT möglich.

Darstellung inaktive Ressource:

Eine inaktive Ressource wird nicht dargestellt. 

Darstellung aktive (freie) Ressource

Im Kalender zeigt der weisse Bereich an, ob die Ressource frei ist. Im Plan wird die freie Ressource durch ein grünes Symbol angezeigt (die Farbe kann global geändert werden), ohne Text im Balken (Kalender) und ohne Tooltip (bei Mouse-Over).

Darstellung der Reservation (besetzte Ressource):

Die Status (u.a. Reservation, Provisorische Reservation, Besetzt, etc.) werden mittels Balkenfarbe resp. Symbol/Farbe (PLAN) deklariert.
Ob eine Reservation Teil eines Anlasses, einer Serie, Buchungen mit fixem Equipment, Buchungen mit mobilem Equipment, Buchungen mit Bestellungen, Buchungen mit Parkplätzen oder Buchungen in teilbaren Räumen ist, wird mit einem entsprechenden Symbol visualisiert. BILDER GEM. 4.2



Die Symbole können mittels einer Symbolleiste im Fussbereich ein- und ausgeblendet werden.
