Bestuhlung erfassen

Navigation zu >Einstellungen>Ressourcen>Bestuhlung verwalten.
Klick auf den Button Anzeigen öffnet im Ansichtsbereich alle bereits erfassten Bestuhlungsoptionen.

Durch Angabe eines Wertes grösser 0 im Feld "Anzahl Personen" werden alle bereits erfassten Bestuhlungsoptionen angegeben, bei der sich dieser Wert zwischen Minimum und Maximum Personenanzahl bewegt.

Mittels Button Neu öffnet sich die Eingabemake zur Erfassung der Bestuhlung. Die Felder und deren Funktionen sind der untenstehenden Tabelle zu entnehmen:

Bezeichnung

Sprechende Bezeichnung der Bestuhlung

 

 

Max. Personenanzahl

Maximale Kapazität (Fassungsvermögen) eines Raums. Dieser Wert wird später im Buchungsprozess dazu verwendet, um auf Grund der gewünschten Kapazität die passende Raumgrössen zu finden.

0

0 = keine Einschränkung

Min. Personenanzahl

Minimale Kapazität (Fassungsvermögen) eines Raums. Dieser Wert wird später im Buchungsprozess dazu verwendet, um auf Grund der gewünschten Kapazität die passende Raumgrössen zu finden.

0

0 = keine Einschränkung

Vorlaufzeit

Die Angabe der Vorlaufzeit in Minuten, dient später im Buchungsprozess dazu, die Buchung um diesen Wert zu erweitern z.B. 09.00 Uhr Terminbeginn mit Vorlaufzeit 15 Minuten, somit ist der Raum bereits ab 08.45 Uhr gebucht.

0

0 = keine Einschränkung

Nachlaufzeit

Die Angabe der Nachlaufzeit in Minuten, dient später im Buchungsprozess dazu, die Buchung um diesen Wert zu erweitern z.B. 10.00 Uhr Terminende mit Nachlaufzeit 15 Minuten, somit ist der Raum bis 10.15 Uhr gebucht.

0

0 = keine Einschränkung

Datei-/Verzeichnis-Browser (für Bild)

Mit dem Button Browse kann ein gewünschtes Bild z.B. einer Blockbestuhlung in der eigenen Dateiablage gesucht werden. Sobald das Bild ausgewählt wurde, genügt ein Klick auf den Button Upload, um dieses hochzuladen

 

 

 

Zugewiesenes Bild

Bild zur visuellen Darstellung der Bestuhlung. Es kann pro Bestuhlungsoption nur jeweils 1 Bild hochgeladen werden. Mittels Klick auf das Bild wird dieses im Zoom als Popup geöffnet.

Bestuhlung bearbeiten

Navigation zu >Einstellungen>Ressourcen>Bestuhlung verwalten.
Klick auf den Button Anzeigen öffnet im Ansichtsbereich alle bereits erfassten Bestuhlungsoptionen.

Mittels Editieren-Symbol öffnet sich die Eingabemaske zur Anpassung der Bestuhlung.

Eine bereits erfasste Bestuhulung kann im Anzeigemodus dargestellt werden, in dem die Bezeichnung angeklickt wird.

Bestuhlung Löschen

Navigation zu >Einstellungen>Ressourcen>Bestuhlung verwalten.
Klick auf den Button Anzeigen öffnet im Ansichtsbereich alle bereits erfassten Bestuhlungsoptionen.

Durch Selektieren einer oder mehrerer Checkboxen von Bestuhlungsoptionen und anschliessendem Klick auf den Button Entfernen werden diese gelöscht.

Hinweis

Bestuhlungen können nur gelöscht werden, wenn keine Verlinkung zu einer Ressource mehr besteht bzw. die Bestuhlung keiner Ressource mehr zugeordnet ist. Ansonsten wird eine Fehlermeldung im Action Message ausgegeben.
