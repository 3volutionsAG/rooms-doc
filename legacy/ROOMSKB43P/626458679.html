Einleitung

Die Oberfläche kann dem CI/CD des Kunden angepasst werden. Die Anpassungen erfolgen in Form von Änderungen von Schriftgrösse, Bildern, Icons, Farben, etc.

Änderungen an der Struktur sind nicht möglich

Vorbedingungen

Keine

Benötigte Rechte

  • Darf das Thema verwalten

Vorgehen / Beschrieb

Benutzer navigiert auf Einstellungen >System >Oberfläche
Benutzer erfasst optional Suchkriterien und klickt Button Finden

Ein Suchresultat über alle verfügbaren Oberflächenanpassungsdateien wird in einer Smartlist dargestellt (Spalten: Filepfad). Eine oder mehrere Dateien können mit Klick auf Button Entfernen gelöscht werden.

Klick auf "Filepfad" zeigt den Inhalt im Anzeigemodus (Viewmode)

Klick auf Button Bearbeiten oder Neu öffnet den Bearbeitungsmodus

Folgende Felder stehen zur Verfügung:

Feld

Funktion

Standardeinstellung

Eigenschaften

Regeln

Dateipfad

Exakter Ablagepfad der zu übersteuernden CSS oder Bilddatei

 

  • Pfad ab (nach)  "Content" erfassen
  • Ohne Slash (/) starten
  • Gross-Kleinschreibung irrelevant

File Daten

Über Browse Button wird die zu ersetzende Datei selektiert
Über Upload Button wird die Datei auf den Server geladen

 

  • Der Upload ist begrenzt. Begrenzung wird global festgelegt (siehe Globale System Parameter konfigurieren)
  • Der Upload Status wird visualsiert
  • Die Upload Datei wird nicht überprüft ob es sich um eine gültige Datei handelt

= Pflichtfelder
= Daten mehrsprachig erfassen

Unterstützte Formate:

  • CSS
  • HTML
  • JS
  • JPG, JPEG, PNG, GIF
  • ZIP (beinhaltet alle Themenfiles)

 

 

Klick auf Button Speichern speichert alle Änderungen. Die Änderungen werden sofort wirksam für alle Benutzer.
