Einleitung

Bestehende Anlässe (siehe Anlass erstellen & bearbeiten) werden unter dem Menü LISTEN wiedergefunden und neben der Möglichkeit diese so zu löschen, bieten sie auch verschiedene Kopiermöglichkeiten.

Anlass kopieren

Ausgehend von dem gespeicherten Anlass der im Anzeigemodus (Viewmode) dargestellt wird, kann mittels Button Kopieren eine Kopie des Anlasses erstellt werden.


Folgende Regeln sind zu beachten:

  • die Kopie enthält die Stammdaten aus dem Originalanlass
  • die Kopie enthält die Teilnehmerdaten aus dem Originalanlass
  • die Kopie enthält die Daten Catering&Services aus dem Originalanlass
  • die Kopie enthält KEINE Reservationen aus dem Originalanlass - diese können nachträglich hinzugefügt werden (siehe unten)
  • die Kopie übernimmt keine Reservationsanfragen

Ist die Anlasskopie erstellt, besteht wie bereits erwähnt die Möglichkeit, Buchungen aus dem Originalanlass mittels Button Reservationen aus Originalanlass kopieren ebenfalls als Kopie dem neuen Anlass hinzuzufügen. Selektieren einer Checkbox und bestätigen über den Button Als Kopie hinzufügen. Das System öffnet die Kopie der Buchung im Editiermodus (siehe Detailbuchung in Handbücher 4.0 - 4.2) und es muss zwingend der Raum oder das Datum/Zeit angepasst werden, ansonsten kann die Buchung nicht gespeichert werden.


Nach Abschluss der Buchung wird die Verknüpfung zum Anlass erstellt und es kann entweder ein Rücksprung auf den Anlass erfolgen oder der Vorgang wird für andere Buchungen wiederholt.

Anlass löschen

Navigation zu >Listen>Anlässe


In der Smartlist werden alle Anlässe gem. Such- und Filterkriterien angezeigt. Sofern dem Anlass noch keine Buchungen von Ressourcen hinzugefügt wurden, kann dieser über das Löschen-Symbol gelöscht werden. Im Action Message wird die Anzahl der entfernten Einträge ausgegeben.
