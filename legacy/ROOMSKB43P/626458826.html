Einleitung

Zur visuellen Unterscheidung (siehe auch Farbgebung im Kalender) und auch statistischen Erhebungen können Buchungen mit bestimmten Kriterien bzw. Klassifikationen (z.B. Kundentermine, interne Meetings etc.) versehen werden.

Vorbedingungen

Keine

Benötigte Rechte

  • Darf Klassifikationen verwalten
  • Darf Sektion 'Einstellungen' nutzen

Vorgehen / Beschrieb

Klassifikationstyp erfassen

Navigation zu >Einstellungen>Erweiterte Basisdaten für Ressourcen>Klassifikationen

Über den Button Klassifikationstyp erstellen, öffnet die Eingabemaske der Hauptgliederung. Die Felder und deren Funktionen sind der untenstehenden Tabelle zu entnehmen.

Funktion

Standardeinstellung

Eigenschaften

Regeln

Bezeichnung

Sprechende Bezeichnung des Klassifikationstyps z.B. Verwendungszwecke

 

 

Code

Freitextfeld (kann frei übersetzt werden siehe Übersetzungen verwalten) zur Angabe z.B. eines Codes wie VZ für Verwendungszweck

 

 

 

Ressourcenart

Auswahl der Ressourcenarten (Sitzungszimmer, Equipment, etc.) zur Definition, bei welcher davon der Klassifikationstyp später in der Detailbuchung zur Verfügung steht.

 

Es stehen nur die Ressourcenarten zur Verfügung, auf denen das nötige Recht vorhanden ist z.B. _Darf Ressourcentyp Raum verwalten_

Mehrfachauswahl erlaubt

Selektieren der Checkbox ermöglicht im späteren Buchungsprozess die Auswahl von mehreren Klassifikationen.

nicht selektiert = im Buchungsprozess kann einer Buchung maximal 1 Klassifikation mitgegeben werden

 

In Listen sichtbar

Selektieren der Checkbox definiert, ob der Klassifikationstyp auch als Filterkriterium in den Listen im Sidepanel sichtbar ist.

nicht selektiert = nicht sichtbar im Sidepanel in den Listen

In Outlook sichtbar

Selektieren der Checkbox definiert, ob der Klassifikationstyp auch als Filterkriterium im Outlook (Handbücher 4.0 - 4.2) verfügbar ist  (Bereich Suchkriterien)

nicht selektiert = nicht sichtbar im Outlook

Standartwert(e)

Möglichkeit alle unter diesem Klassifikationstyp angelegten Klassifikationen auszuwählen, damit diese im Buchungsprozess auf den entsprechenden Ressourcen jeweils bereits enthalten sind

 

 

 

AnlasstypMöglichkeiten alle unter diesem Klassifikationen angelegte Anlasstypen auszuwählen.   

= Pflichtfelder
= Daten mehrsprachig erfassen

Im Register "Ressourcen" werden die Ressourcen hinzugefügt, auf denen im Buchungsprozess die erfassten Klassifikationen zur Auswahl angeboten werden sollen. Über den Button Ressourcen hinzufügen öffnet ein Modales Popup zur Auswahl der gewünschten Ressourcen.

Es werden nur die Ressourcen aufgelistet, für die genügend Berechtigung (z.B. Darf Ressourcentyp Raum verwalten) vorliegen.



Selektieren einer oder mehrerer Checkboxen und Klick auf den Button Ressource hinzufügen schliesst den Vorgang ab.

Im Register "Gruppe" werden die Gruppen hinzugefügt, für die die Klassifikation verfügbar sein soll. Damit wird der Benutzerkreis eingeschränkt, der diese Klassifikation später nutzen kann. Über den Button Hinzufügen öffnet ein Modales Popup zur Auswahl der gewünschten Gruppen.


Selektieren einer oder mehrerer Checkboxen und Klick auf den Button Gruppe hinzufügen schliesst den Vorgang ab.

Klassifikationstyp speichern
Nach Eingabe aller Pflichtdaten und der gewünschten optionalen Daten wird der Klassifikationstyp mittels Button Speichern gesichert und erscheint in Baumstruktur.

Klassifikationstyp bearbeiten

Navigation zu >Einstellungen>Erweiterte Basisdaten für Ressourcen>Klassifikationen
In der Smartlist werden alle bereits erfassten Klassifikationstypen angezeigt. Mittels Klick auf die Bezeichnung erscheinen die Icons zur Bearbeitung, Ansicht und Löschung. Über das Ansichts-Symbol wird der Gliederungstyp im Viewmode dargestellt.

Über das Editieren-Symbol wechselt das System in den Bearbeiten Modus. Es können alle Felder verändert werden.

Klassifikationstyp löschen

Navigation zu >Einstellungen>Erweiterte Basisdaten für Ressourcen>Klassifikationen
In der Smartlist werden alle bereits erfassten Klassifikationstypen angezeigt. Mittels Klick auf die Bezeichnung erscheinen die Icons zur Bearbeitung, Ansicht und Löschung. Über das Löschen-Symbol erscheint eine Sicherheitsabfrage, ob der Klassifikationstyp wirklich entfernt werden soll. Quittieren dieser Meldung entfernt den Status. Im Action Message wird die Anzahl der entfernten Einträge ausgegeben.

Klassifikationstypen können nur gelöscht werden, sofern keine Klassifikationen mehr angehängt sind und/oder dem Klassifikationstyp Ressourcen zugewiesen sind.

Klassifikation erfassen

Navigation zu >Einstellungen>Erweiterte Basisdaten für Ressourcen>Klassifikationen
In der Smartlist werden alle bereits erfassten Klassifikationstypen angezeigt. Mittels Klick auf die Bezeichnung erscheinen die Icons zur Bearbeitung, Ansicht und Löschung.

Über das Plus-Symbol öffnet die Eingabemaske im Register "Stammdaten". Die Felder und deren Funktionen sind der untenstehenden Tabelle zu entnehmen.

Feld

Funktion

Standardeinstellung

Eigenschaften

Regeln

Bezeichnung

Sprechende Bezeichnung der Klassifikation z.B. Beratung

 

 

Sortierung

Angabe eines alphanummerischen Wertes damit die Position in der Baumstruktur vorgeben werden kann d.h. an welcher Stelle die Klassifikation aufgelistet wird in der Smartlist

 

 

 

Farbe

Auswahl einer Farbe für die Klassifikation, die im Kalender visualisiert wird (siehe Farbgebung im Kalender und Farbpicker).

 

 

  • Werden im Buchungsprozess mehrere Klassifikationen vom selben Klassifikationstyp gewählt, wird die Farbe mit der höchsten Sortierpriosität angezeigt.
  • Werden im Buchungsprozess mehrere Klassifikationen von unterschiedlichen Klassifikationstypen gewählt, wird die Farbe des Klassifikationstyps mit der höchsten Sortierpriorität angezeigt.

Nach Eingabe aller Pflichtdaten und der gewünschten optionalen Daten wird die Klassifikation mittels Button Speichern gesichert und erscheint in Baumstruktur.

Klassifikation bearbeiten

Navigation zu >Einstellungen>Erweiterte Basisdaten für Ressourcen>Klassifikationen
In der Smartlist werden alle bereits erfassten Klassifikationstypen angezeigt. Über das Plus-Symbol öffnet sich die zweite Hirarchiestufe und zeigt alle untergeorneten Klassifikationen an. Mittels Klick auf die Bezeichnung erscheinen die Icons zur Bearbeitung, Ansicht und Löschung.



Über das Editieren-Symbol öffnet sich die Klassifikation im Bearbeiten Modus und alle Daten können verändert werden.

Klassifikation löschen

Navigation zu >Einstellungen>Erweiterte Basisdaten für Ressourcen>Klassifikationen
In der Smartlist werden alle bereits erfassten Klassifikationstypen angezeigt. Über das Plus-Symbol öffnet sich die zweite Hirarchiestufe und zeigt alle untergeorneten Klassifikationen an. Mittels Klick auf die Bezeichnung erscheinen die Icons zur Bearbeitung, Ansicht und Löschung.



Mittels Klick auf das Löschen-Symbol erscheint eine Sicherheitsabfrage, ob die Klassifikation wirklich entfernt werden soll. Quittieren dieser Meldung mit OK löscht die Klassifikation. Im Action Message wird eine Meldung ausgegeben, dass der Eintrag erfolgreich gelöscht wurde. Die Löschung kann nur erfolgen, wenn keine Abhängigkeit zu Buchungen mehr bestehen.
