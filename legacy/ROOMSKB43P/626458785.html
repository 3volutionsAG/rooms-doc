Verwandte Seiten

Einleitung

Unter einem Anlass wird in ROOMS grundsätzlich ein “Pool” verstanden, welcher verschiedene Buchungen, Teilnehmer und Bestellungen zusammenfasst. In diesem “Pool” können Buchungen von Ressourcen, Einladungen von Teilnehmern sowie Bestellungen verschiedener Dienstleistungen erfasst werden.

Vorbedingungen

Es muss min. 1 Anlasstyp erfasst worden sein (siehe Anlasstypen erfassen & bearbeiten)

Benötigte Rechte

  • Darf Sektion Reservieren nutzen
  • Darf Sektion Listen nutzen
  • Darf Anlässe als Kalender sehen
  • Darf Anlässe buchen
  • Darf Anlässe für Dritte buchen ist nur notwendig, wenn für einen anderen Organisator gebucht werden soll
  • Darf Anlässe mit Bestellungen buchen ist nur notwendig, wenn Dienstleistungen bestellt werden sollen
  • Darf Anlässe mit Reservationen buchen
  • Darf Anlässe mit Teilnehmernotifikationen editieren
  • Darf Teilnehmer editieren ist nur notwendig, wenn Teilnehmer hinzugefügt werden sollen

Über den Button Speichern wird der Vorgang abgeschlossen und der Anlass wird im Anzeigemodus (Viewmode) dargestellt.

Bestehende Reservationen zu Anlass hinzufügen

Ausgehend von dem gespeicherten Anlass der im Anzeigemodus (Viewmode) dargestellt wird, kann mittels Button Bestehende Reservationen hinzufügen eine bereits vorgängig getätigte Buchung dem Anlass hinzugefügt werden.

Sowohl Einzelbuchungen wie auch Serienbuchungen (Handbücher 4.0 - 4.2) können dem Anlass hinzugefügt werden.

Weitere Regeln:

  • Button nur sichtbar mit Recht: Darf Anlässe mit Reservationen buchen
  • Durch das Absetzen einer Suche mit entsprechenden Kriterien, werden die noch nicht dem aktuellen Anlass zugeordneten Buchungen angezeigt (Dazu gehören auch Buchungen, die bereits einem anderen Anlass zugeordnet sind)
  • Die Buchungen, die bereits einem anderen Anlass zugeordnet sind, werden im Suchergebnis entsprechend gekennzeichnet
  • Alle Einstellungen, sowie allfällig gebuchte Dienstleistungen, zusätzliches Equipment und Teilnehmer werden übernommen
  • Werden Buchungen von einem anderen Anlass zum aktuellen hinzugefügt, wird die Verbindung zum alten Anlass getrennt und eine entsprechende Meldung wird ausgegeben
  • Müssen Einstellungen auf dieser Buchung verändert werden (über Detail), muss dies separat, nach dem Hinzufügen der Buchung zum Anlass erfolgen.
  • Suchkriterien Sidepanel: Als Vorgabe dient das Anlass von und Bis Datum

Ressourcen zu Anlass buchen über das Menü LISTEN

Ausgehend von dem gespeicherten Anlass der im Anzeigemodus (Viewmode) dargestellt wird, kann mittels Button Ressourcen zu Anlass reservieren (Listen) eine verfügbare Ressource dem Anlass hinzugefügt werden.


Sowohl Einzelbuchungen wie auch Serienbuchungen (siehe Handbücher 4.0 - 4.2) können dem Anlass hinzugefügt werden. Mittels Selektieren der Checkbox einer oder mehrerer Buchungen wird über den Button Schnellreservation zu Anlass (siehe Handbücher 4.0 - 4.2) eine Ressource/n dem Anlass hinzugefügt. Die Schnellreservation von nur 1 Ressource kann auch über das entsprechende Icon erfolgen (siehe Abbildungen).

Mittels Button Detailreservation wird eine detaillierte Buchung erstellt und dem Anlass hinzugefügt. Innerhalb der Detailreservation können wiederum weitere Ressourcen wie Equipment, Fahrzeuge etc. dazu genommen, sowie Teilnehmer und Catering&Services hinzugefügt werden. Sind keine Ergänzungen gewünscht, kann auch eine Schnellbuchung erfolgen.

Ressourcen zu Anlass buchen über das Menü KALENDER

Ausgehend von dem gespeicherten Anlass der im Anzeigemodus (Viewmode) dargestellt wird, kann mittels Button Ressourcen zu Anlass reservieren (Kalender) eine verfügbare Ressource dem Anlass hinzugefügt werden.


Sowohl Einzelbuchungen wie auch Serienbuchungen (siehe Handbücher 4.0 - 4.2) können dem Anlass hinzugefügt werden. Die Serie wird über die Detailbuchung erstellt. Dazu wird im Kalender zum gewünschten Zeitpunkt mit Mausklick und dann über den Button Detailreservation diese geöffnet. Sind keine Ergänzungen gewünscht, kann auch eine Schnellbuchung erfolgen.

Hinzugefügte Reservationen bearbeiten

Ausgehend von dem gespeicherten Anlass der im Anzeigemodus (Viewmode) dargestellt wird, kann jede hinzugefügte Buchung über das Bearbeiten-Symbol oder durch selektieren der Checkbox und klicken auf den Button Bearbeiten editiert werden.

Massenmutation im Anlass (nicht verfügbar für VC-Anlässe)

Eine Mehrfachselektion d.h. die Bearbeitung mehrerer Buchungen gleichzeitig ist möglich. Dazu werden mehrere Checkboxen selektiert und es öffnet sich die Bearbeitungsmaske der Massenbearbeitung.

Die Felder für die Massen-Bearbeitung und deren Funktionen sind der unten stehenden Tabelle zu entnehmen. Alle Felder die angepasst werden wollen, müssen mit der entsprechenden Checkbox selektiert werden, damit der ausgegraute Bereich zur Bearbeitung "freigeschaltet" wird.

Anlass bearbeiten (Buchungen vom Anlass entfernen)

Ausgehend von dem gespeicherten Anlass der im Anzeigemodus (Viewmode) dargestellt wird, kann mittels Button Bearbeiten der Anlass editiert und zusätzlich auch Buchungen davon entfernt werden. Alle bereits vorgängig beschriebenen Register (Stammdaten, Catering & Services, Teilnehmer) sind editierbar.


Über Icons werden Einzelbuchungen wie auch Serienbuchungen vom Anlass entfernt.

Regeln für die Entfernung von Buchungen

  • Verknüpfung zu Anlass wird entfernt
  • Buchung bleibt bestehen (ausserhalb des Anlass)
  • Bestätigungsmeldung in Action Message

Anlasskalender

Ausgehend von dem gespeicherten Anlass der im Anzeigemodus (Viewmode) dargestellt wird, wechselt man ins Register "Anlass Kalender". Alle bereits hinzugefügten Buchungen werden in Form des Kalenders (gleiche Funktionalität wie im Menü KALENDER, siehe Handbücher 4.0 - 4.2) visualisiert und sind editierbar. Ausserdem ist eine Zeitlinie, die sich über die gesamte Anlassdauer erstreckt, angezeigt. Auf diese Weise ist auf einen Blick jedes Datum ersichtlich, an dem eine Ressourcenbuchung vorliegt für den Anlass.

Generelle Regeln beim Anlasskalender

Zeitlinine

  • Das Startdatum entspricht dem Anlassstartdatum aus den Stammdaten
  • Das Enddatum entspricht dem Anlassenddatum aus den Stammdaten

Folgendes wird in Form einer Sprechblase auf der Zeitlinie dargestellt:

  • Buchungen von Ressourcen
  • Nur die Masterbuchung wird dargestellt. Wird also ein Raum mit Dienstleistungen und Equipment gebucht, wird nur die Raumbuchung dargestellt.
  • MouseOver über das Symbol zeigt ein erweitertes Tooltip
  • Klick auf Sprechblase wechselt den Kalender auf die Kalenderwoche in welcher sich die aktuelle Buchung befindet.

Nicht dargestellt werden:

  • Buchungen von Arbeitsplätzen
  • Teilnehmer
  • Wochenendtage Samstag/Sonntag

Kalender

  • Der Kalender zeigt alle gebuchten Ressourcen (Masterbuchungen) innerhalb des Anlass an.
  • Es wird jeweils nur eine Kalenderwoche visualisiert.
  • Der Kalender verfügt über eine Navigation um eine Woche vor und zurück zu navigieren.
  • Der Kalender verfügt über dieselbe Funktionen wie der Hauptkalender (Drag&Drop, Verlängern, Tooltip, Schnellreservation)


Das Wiederfinden eines bereits erstellten Anlasses, dessen Löschung, sowie die möglichen Kopierfunktionen sind im KB Artikel Anlass suchen, kopieren, löschen beschrieben.
