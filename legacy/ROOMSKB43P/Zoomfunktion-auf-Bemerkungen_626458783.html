Knowledge Base (Deutsch) : Zoomfunktion auf Bemerkungen

Verwandte Seiten

 

Einleitung

Da teilweise Bemerkungen mit viel Text erfasst werden müssen, steht eine Zoomfunktion zur Verfügung.

Vorbedingungen

Keine

Vorgehen / Beschrieb

  • Neben jedem Bemerkungsfeld ist ein Zoom Icon ersichtlich.
  • Mit Klick auf das Zoom Icon wird ein editierbares Popup geöffnet.
  • Der Benutzer kann den Text ändern und mittels Button Übernehmen zurückkehren in das Originalfeld.

 

Regeln:

  1. Gilt für alle Bemerkungsfelder in der Reservation, Anlass, Reservationsanfrage, etc.
  2. Explizit ausgenommen sind folgende Bereiche:
    1. Notifikationen
    2. Informationen
    3. Action Popup (Links einfügen)
