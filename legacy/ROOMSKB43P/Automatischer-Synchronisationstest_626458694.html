Automatischer Synchronisationstest

Einleitung

Dieser Artikel beschreibt den automatischen Synchronisationstest durch den geprüft wird ob die Einstellungen und die Konfiguration für den jeweiligen Benutzer korrekt sind und funktionieren.

Vorbedingungen

Initial muss die Synchronisation im Profil (siehe Profil bearbeiten) aktiviert sein.

Benötigte Rechte

Keine

Vorgehen / Beschrieb

Wenn im Benutzerprofil die Synchronisation konfiguriert/aktiviert ist, wird im Viewmodus des angemeldeten Benutzers (Menü EINSTELLUNGEN>Persönliche Einstellungen) das Register "Synchronisationstest" eingeblendet.

Unter einem kurzen Informationstext befindet sich der Button Test durchführen. Ein Klick auf den Button startet den Test. Dabei werden zeilenweise die Testschritte aufgeführt.

Schlägt ein Testschritt fehl oder sind alle Testschritte erfolgreich abgearbeitet, erscheint eine kurze Zusammenfassung.

Der Button Test durchführen ist während des Tests deaktiviert und wird nach Abschluss des Tests wieder aktiviert. Der Benutzer kann das Register "Synchronisationstest" jederzeit verlassen und weiterarbeiten. In diesem Fall läuft der Test im Hintergrund weiter. Ruft der Benutzer das Register "Synchronisationstest" erneut auf, werden ihm die Resultate des letzten Tests oder der Stand des aktuellen Durchlaufs angezeigt.
