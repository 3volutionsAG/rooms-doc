Einleitung

Der folgende Artikel enthält alle relevanten Informationen über die Erfassung und Bearbeitung von Angeboten, die von den Dienstleistern bereit gestellt werden.

Vorbedingungen

Es muss mindestens 1 Angebotskategorie bestehen Angebotskategorie erfassen & bearbeiten.

Benötigte Rechte

  • Darf Dienstleister verwalten
  • Darf Angebote verwalten
  • Darf Dienstleister Angebote verwalten
  • Darf Angebot im Bestellprozess ändern (nur relevant zur freien Erfassung von Dienstleistungen d.h. Überschreiben von ausgewählten Angeboten im Bestellprozess)

Vorgehen / Beschrieb

Angebot erfassen/löschen

Navigation zu >Einstellungen>Catering & Services>Dienstleister verwalten

Klick auf das Angebots-Symbol öffnet im Anzeigebereich den Kategorienbaum.



Es besteht die Möglichkeit entweder Einzelangebote oder Angebotspakete zu erfassen. Damit steht später im Reservationsprozess beispielsweise ein Apfel als Einzelangebot oder ein Früchtekorb als Angebotspaket zur Auswahl (Angebotszone erfassen & bearbeiten).

Unterscheidung Angebotspaket und Angebot

•Sobald einem Angebot, weitere Unterangebote zugeordnet werden, handelt es sich um ein Paket (Bewirkt im Reservationsprozess, dass die Unterangebote als selektierbare Optionen des Pakets angezeigt wird)
•Dabei gibt es nur eine Hierarchiestufe (Erfassung von Subangebot auf einem Angebot möglich, Erfassung von Subangebot auf Subangebot NICHT möglich)


Klick auf eine Angebotskategorie zeigt die 4 Icons Angebotskategorie hinzufügen, Angebot hinzufügen, Angebot editieren, Löschen. Zur Erfassung eines Einzelangebots wird das zweite Plus-Symbol gewählt und damit der Editiermodus des Angebots geöffnet. Die Felder und deren Funktionen sind der untenstehenden Tabelle zu entnehmen.

Die Erfassung muss mehrsprachig erfolgen, ansonsten erscheint ein rotes Warndreieck mit dem Hinweis auf die fehlende Übersetzung bzw. Mehrsprachigkeit und das Angebot (oder Paket) kann nicht gespeichert werden.

Angebotskategorie (Angebotspicker)

Über den Angebotspicker kann ein Angebot (oder Paket) in die gewünschte Kategorie geschoben werden. Klick auf Plus-Symbol öffnet den Kategorienbaum.

Kategorie in der das Angebot (oder Paket) erstellt wurd ist ausgewählt

 

Feld: Vorlaufdauer

Gibt an wie viele Minuten vor Reservationsbeginn das Angeobt (oder Paket) noch bestellt werden kann.

0 (=keine Vorlaufdauer)

Im Reservationsprozess werden nicht mehr bestellbare Angebote ausgeblendet. Bei der Berechnung wird die Arbeitszeit des Dienstleisters berücksichtigt.

Feld: Nachlaufdauer

Gibt rein informativ eine Nachlaufzeit in Minuten nach Buchungsende an.

0 (=keine Nachlaufdauer)

Überschreibbar

Selektieren dieser Checkbox bewirkt im Reservationsprozess, dass der Benutzer ein gewähltes Angebot überschreiben kann (freie Erfassung von Dienstleistungen)

Möglichkeit zur Erfassung von Angebotskosten

Annullationskosten

Möglichkeit zur Erfassung von Annullationskosten, die bei Annullation einer Buchung erhoben werden.

leer (= keine Annullationskosten)

 

Wert > 0 entspricht dem Wert, der bei einer Annullation pauschal erhoben wird

Annullation vor Beginn (in Minuten)

Gibt an wie viele Minuten vor Beginn der Buchung Annullationskosten erhoben weden. Als Beispiel: Bei einem Wert von 60, werden bis 60 Minuten vor Buchungsstart KEINE Annullationskosten erhoben. Zwischen 60 Minuten und der Endzeit der Buchung wird der Pauschalbetrag, der unter "Annullationskosten" definiert wurde in Rechung gestellt.

leer (= ausgeschalten)

 

0 = Annullationskosten werden zur Laufzeit erhoben

Sortierung

Möglichkeit mit Angabe eines nummerischen Werts (1, 2, 3 etc.) die Sortierung in der Baumstruktur auf gleicher Hierarchiestufe einzustellen.

Ist die Erfassung abgeschlossen, wird mit Klick auf den Button Speichern das neue Angebot (oder Paket) gespeichert und erscheint in der Baumstruktur.

Zum Löschen wird das Angebots (oder Paket) in der Baumstruktur angeklickt und mit über das Löschen-Symbol entfernt. Die Löschung kann nur erfolgen, wenn keine Abhängigkeiten zu Angebotszonen mehr bestehen.

Angebot bearbeiten

Navigation zu >Einstellungen>Catering & Services>Dienstleister verwalten


Mit Klick auf das gewünschte Angebot (oder Paket) und anschliessendem wählen des Editieren-Symbol öffnet sich das Angebot im Editiermodus und die Daten können verändert werden
