Einleitung

Im gesamten System trifft der Benutzer auf Felder, welche auf Grund von Regeln validiert werden.

Vorbedingungen

Systemvorgaben (Feldregeln)

Vorgehen / Beschrieb

Nutzer befindet sich in einer Eingabemaske in einem mit einer Regeln hinterlegten Feld. Beim Verlassen des Feldes wird die Regel geprüft. Bei Regelverletzung wird der Benutzer umgehend informiert

Resultat:

  1. Das Feld wird eingefärbt und mit einem Warndreieck versehen, als Hinweis auf die Regel Verletzung .
  2. Bei Mouseover über das Warndreieck, wird ein Hilfetext eingeblendet

Anmerkung: Feldvalidierungen können auch in Kombination mit Pflichtfelder stehen

Fehlerbehandlungen

  1. Kein Suchresultat
    1. Anzeigen einer  Fehlermeldung in der Smartlist: Es wurden keine Einträge gefunden
  2. Felder werden spezifische validiert.
    1. Verletzung einer Validierungsregel
      1. anzeigen einer feldbezogene Meldung (Error Dreieck und rote Umrandung des Feldes) => Bei MouseOver wird ein Hilfetext angezeigt
      2. Anzeigen einer globalen Information im Action Message
      3. Anzeigen einer Meldung auf dem entsprechenden Tab (rote Schrift)
