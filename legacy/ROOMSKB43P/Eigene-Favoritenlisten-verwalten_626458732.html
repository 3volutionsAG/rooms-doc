Einleitung

Die persönlichen Listen (Favoriten) einer Person können gelöscht oder publiziert (für andere Zugänglich) gemacht werden.

Listen von Dritten können abonniert werden.

Vorbedingungen

Eine Person muss im Bearbeiten Modus sein - siehe KB Person bearbeiten

Eigene Listen wurden bereits gespeichert (siehe Handbücher 4.0 - 4.2) oder globale Listen stehen zur Verfügung (siehe KB Globale und öffentliche Listen)

Benötigte Rechte

  • keine

Vorgehen / Beschrieb

Register "Eigene Listen" wird angewählt

Liste aller eigenen oder bereits abonnierten Listen wird dargestellt. Folgende Spalten stehen zur Verfügung.

  • Bezeichnung
  • Listenart
    • privat => Liste ist nur für den privaten Gebrauch - Dritte können diese nicht einsehen
    • public => Liste wurde durch Dritten zur Publikation publiziert (Man können Liste abonnieren oder sich eine Kopie erstellen lassen
    • shared => Liste wurde durch einen Dritten publiziert und der angemeldete Benutzer hat sich diese abonniert 
  • Besitzer => zeigt, wem die Liste gehört
  • Abos => zeigt an wieviele Personen eine publizierte Liste abonniert haben
  • Suchbereich => zeigt den Verwendungsbereich (z.B. Kalender oder Plan)

Der Button "Publikation stoppen"  ändert eine bereits  publizierte Liste (Public) wieder zu Privat. Es können mehrere Listen gleichzeitig angewählt und gestoppt werden. Dabei werden alle Abonemente zu dieser Liste entfernt  und die Liste steht den Abonementen ab sofort nicht mehr zur Verfügung

Der Button "Entfernen" entfernt eine "private" Liste unwiederruflich. Es können mehrere Listen gleichzeitig entfernt werden.

Der Button "Hinzufügen oder "Abonnieren" öffnet einen modalen Dialog zur Zuweisung einer fremden (Public oder Global) Liste zu den eigenen:
