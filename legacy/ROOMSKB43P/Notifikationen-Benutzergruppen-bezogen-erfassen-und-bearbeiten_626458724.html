Knowledge Base (Deutsch) : Notifikationen Benutzergruppen bezogen erfassen und bearbeiten

Created by Former user, last modified by Former user (Deleted) on Jul 24, 2013

Artikel Nummer

Gültig ab Release

4.2

Verwandte Seiten

Einleitung

Damit im späteren Buchungsprozess Notifikationen (siehe auch Notifikationsmatrix) per Mail verschickt werden (z.B. neu registrierter Benutzer, Statuswechsel Reservation etc.), müssen die Vorlagen dazu erstellt werden. Nebst den Benutzergruppen bezogenen Vorlagen, sind ebenfalls auch Standort spezifische Notifikationsvorlagen (Notifikationen standortbezogen erfassen & bearbeiten) vorhanden.

Vorbedingungen

Initial ist eine Benutzergruppe erfasst

Benötigte Rechte

  • Darf Notifikationen verwalten
  • Darf Sektion Einstellugen nutzen
  • Darf Benutzergruppen verwalten

Vorgehen / Beschrieb

Navigation zu >Einstellungen>Sicherheitsdaten für Personen>Benutzergruppen
Mittels Klick auf das Bearbeiten-Symbol der gewünschten Benutzergruppe und Wechsel in das Registers "Notifikationen". Mittels Button Hinzufügen öffnet das modale Popup zur Auswahl der gewünschten Notifikationsvorlage. Die Felder und deren Funktionen sind er unten stehenden Tabelle zu entnehmen:

 

Damit die Bearbeitung des E-Mailbody benutzerfreundlicher wird, kann die Box maximiert und wieder minimiert werden (ab 4.2.4).

Die Notifikationsplatzhalter können nicht nur zur Konfiguration des E-Mailbodys verwendet werden, sondern ebenfalls für den Titel (Subject)

Feld

Funktion

Standardeinstellung

Eigenschaften

Regeln

Notfikationstyp

Bezeichnet die Vorlage z.B. Vorlage für das Bestätigungsmail bei einer neuen Buchung

 

Jeder Notifikationsvorlagen Typ kann nur einmal verwendet werden

Benachrichtigung im CC an

Erfassung von gültigen E-Mail Adressen, damit beim Versenden der Notifikation  ein CC an die entsprechenden Adressaten verschickt wird.

 

 

  • leer = keine CC Benachrichtigung
  • Mehrere E-Mail Adressen werden mittels Semikolon separiert

Sprache

Es stehen alle im System aktivierten Sprachen zur Verfügung

 

 

 

Titel (Subject)

Eingabe eines Sprechenden Titels (Subject) für die Notifikation

 

 

E-Mail Body

Erfassung des Inhalts der E-Mailnachricht. Diese kann mehrsprachig erfasst werden, dazu wird jeweils im Dropdown die gewünschte Sprache gewählt und danach der E-Mail Body eingerichtet. Für die Einstellung des Inhalts stehen unterschiedliche Platzhalter zur Verfügung siehe hierzu Notifikationsplatzhalter.

 

 

  • Wurde kein E-Mail-Server eingetragen (siehe Globale System Parameter konfigurieren), erscheint dieses Feld nicht. Klick auf einen Platzhalter fügt diesen an der aktuellen Cursor Position ein. Ist kein Cursor positioniert wird der Platzhalter am Anfang des Body Textes eingefügt
  • Es ist immer nur eine Platzahlter Sektion gleichzeitig geöffnet.
  • Die Platzhalter übernehmen die Formatierung des Body Editors

= Pflichtfelder
= Daten mehrsprachig erfassen

Nach Erfassung alles optionalen und Pflichtdaten, wird der Vorgang über den Button Speichern abgeschlossen.
