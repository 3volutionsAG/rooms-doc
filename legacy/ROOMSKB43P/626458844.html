Knowledge Base (Deutsch) : Gespeicherte Listen (Erstellen personalisierte Suchlisten)

Created by Former user (Deleted), last modified on Nov 15, 2013

Artikel Nummer

Gültig ab Release

4.3

Verwandte Seiten

 

Einleitung

Ausgenommen vom Menü REPORTS findet man unter jedem Menü (nicht in jedem Subpanel) die Möglichkeit Suchlisten zu speichern.

Vorbedingungen

Keine

Vorgehen

Stellvertretend wird hier das Vorgehen bei der Erstellung einer gespeicherten Liste im Menü LISTEN aufgezeigt.

 

Klick auf Disketten-Symbol im Menü LISTEN

 

Im geöffneten Fenster "Filter speichern" eine sprechende Bezeichnung für die Suchliste eingeben und den Vorgang mit dem Button Speichern abschliessen. 

 

Die Suchliste wird gespeichert und ist ab diesem Zeitpunkt im Dropdown verfügbar. Wird diese vom Benutzer ausgewählt, lädt das System die entsprechenden Such- und Filterkriterien automatisch im Anzeigebereich.

 

Wird die Suchliste so gespeichert, dass die Checkbox "Favorit" selektiert ist, wird bei jedem Öffnen des entsprechenden Menü bzw. Subpanels diese gespeicherte Liste als Suchresultat geladen. Wenn diese gelöscht oder publiziert werden will, kann das Vorgehen dazu unter folgendem Artikel Eigene Favoritenlisten verwalten nachgelesen werden.
