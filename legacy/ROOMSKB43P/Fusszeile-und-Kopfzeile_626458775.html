Einleitung

Der Footer kann individuell gestaltet werden. Standarmässig wird er vorkonfiguriert ausgeliefert.

Vorbedingungen

Keine

Vorgehen / Beschrieb

Grundsätzlich kann jeder Kunde diesen Bereich individuell gestalten können (keine Funktionalität nur Inhalte und Links). Dies kann via HTML erfolgen. Unter Einstellungen>System>Thema verwalten und Klick auf den Button Finden erscheint das derzeit aktuelle Thema.



Mittels Button Hinzufügen öffnet die Eingabemaske zum Hochladen der neuen HTML Datei, damit der kundenspezifische Fusszeile bzw. Kopfzeile.


Legende

1= Datei im Format htm
2= Button Browse zur Suche im Fileverzeichnis
3= Button Upload zum Hochladen der Datei

Benötigte Rechte

  • Darf das Thema verwalten
  • Darf Sektion Einstellungen nutzen
