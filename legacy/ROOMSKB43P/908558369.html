Knowledge Base (Deutsch) : FTP import/export

Created by Joel Frutiger, last modified on Oct 16, 2018

Rooms unterstützt (S)FTP import/export von Txt sowie XML Dateien.

Im DataDestination / DataSource Block muss der Korrekte ConnectionString angegeben werden:

Der Inhalt des Connectionstrings muss eine URI sein:

Beispiele: ftp://test.example.com:3333/path/to/file, sftp://test.example.com/path/

Zusätzlich kann der FTP Zugang mit Attributen im ConnectionString konfiguriert werden:

AttributMögliche WerteDefault
UserNameZeichenfolge
PasswordZeichenfolge
FtpModeActive / PassivePassive
FtpSecureImplicit, Explicit, NoneNone
PrivateKeyPassphraseZeichenfolge
TlsClientCertificatePathPfad
SshHostKeyFingerprintZeichenfolge
AcceptSshHostKeyTrue, FalseFalse
AcceptTlsCertTrue, FalseFalse
TransferModeAutomatic, Binary, AsciiBinary

Hier noch einige Beispiele:

Example FTP
  <DataDestination BaseDataTyp="Xml">
    <ConnectionString UserName="ftpuser" Password="pw">ftp://localhost/</ConnectionString>
  </DataDestination>
Example SFTP
<DataDestination BaseDataTyp="Txt" Delimiter=";">
<ConnectionString UserName="ftpuser" Password="pw" AcceptSshHostKey="True">sftp://ftp.example.com/home/ftpuser/*</ConnectionString>
</DataDestination>
