Einleitung

Im gesamten System trifft der Nutzer s.g. modale Popups an, die die Eingabe von einzelnen Einträgen oder ein Zuordnung von verschiedenen Einträgen ermöglicht.

Vorbedingungen

Keine

Vorgehen / Beschrieb

Wird ein modales Popup geöffnet, so gelten die folgenden Bestimmungen:

  • Der ganze Applikationsbereich wird mit einem transparenten Schwarz eingefärbt
  • Der schwarze Bereich ist nicht klickbar
  • Auf der Fläche des Inhaltsbereich hat das modale Popup einen editerbaren Bereich (weisse Darstellung), darin werden die editierbaren Felder gelistet
  • Der Editierbare Bereich ist scrollbar
  • Das modale Popup schliesst sich auf Knopfdruck (Abbrechen oder Funktionsknopf)

    In folgenden Fällen erfolgt ein Absprung in ein modales Popup:

1. Einzelner Eintrag wird geändert oder hinzugefügt z.B. offene Menge wird mit effektiver Menge abgefüllt (Bearbeiten von offenen Mengen auf Bestellungen durch Dienstleister).


2. Zuordnung von verschiedenen Records findet statt (z.B. mehrere Benutzer der aktuellen OE zuordnen). Innerhalb dieses Popups sind zusätzlich Suchfunktionen bzw. Suchfelder verfügbar.
