Knowledge Base (Deutsch) : Reservationsanfrage erfassen

Created by Former user, last modified on Jul 05, 2013

Artikel Nummer

Gültig ab Release

4.2

Verwandte Seiten

Einleitung

Besitzt ein User ungenügende Rechte oder handelt es sich um eine Spezialbuchung, können diese User eine Reservationsanfrage tätigen. Aus dieser wird durch eine autorisierte Instanz ein Anlass erstellt. Die Funktion wird sehr häufig für Videokonferenzen oder auserwählte Räume z.B. Aula verwendet.

Vorbedingungen

Initial müssen Anlasstypen (Anlasstypen erfassen & bearbeiten) erfasst worden sein.

Benötigte Rechte

  • Darf Reservationsanfragen einreichen

Vorgehen / Beschrieb

Navigation zu >Reservieren>Reservationsanfrage
Auswahl des Standortes an dem die Buchung (Anlass, Videokonferenz etc.) gewünscht ist. Als Standardwert wird der Standard Standort aus dem Profil übernommen (Profil bearbeiten). In der Dropdown Anlasstypen stehen alle verfügbaren Anlasstypen auf denen der Benutzer berechtigt ist (siehe Anlasstypen erfassen & bearbeiten).


Mittels Button Erstellen öffnet die Eingabemaske das sogenannte Reservationsformular. Da dieses Formular individuell von jedem Kunden gestaltet werden kann (siehe Anlasstypen erfassen & bearbeiten) wird hier nicht auf die einzelnen Felder eingegangen.
Sind alle Pflicht- und optionalen Felder wunschgemäss ausgefüllt, wird der Vorgang mit dem Button Anfrage senden abgeschlossen. Es wird eine E-Mailnotifikation an die Benutzergruppe verschickt, die beim Erstellen des Anlasstyps hinterlegt worden ist (hierbei muss zwingend die Notifikationsvorlage ReservationsanfrageNeu hinterlegt worden sein siehe Notifikationen Benutzergruppen bezogen erfassen und bearbeiten).
