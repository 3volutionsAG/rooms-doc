Einleitung

Smart ROOMS dient dazu Räume effizienter zu nutzen. Dazu werden Sensoren in Räumen installiert, welche mittels Funkübertragen den Besetztstatus eines Raumes an den Smart ROOMS Service übermitteln. Der Smart ROOMS Service nutzt diese Informationen um Buchungen als "No Show" zu markieren, zu terminieren, zu erstellen oder zu verlängern.

Vorbedingungen

Sensoren Empfänger und Signalverstärker wurden so platziert, dass sämtliche Sensoren ihren Status an den Smart ROOMS Service übermitteln können.

Lizenzen für Türschilder wurden erworben und installiert.

Benötigte Rechte

Der Smart ROOMS Service wird so konfiguriert, dass er einen spezifischen (Service-) User verwendet um Buchungen in ROOMS zu manipulieren.

Dieser Benutzer: Benötigt Rechte um den entsprechenden zu buchen (Checkout und Freigeben Rechte ebenfalls beachten)

Administratoren: Finden einen detaillierten Beschrieb im KB Artikel Rechte.

Vorgehen / Beschrieb

Installation

Der Smart ROOMS Service wird als Windows Service auf einem Server mit Netzwerkverbindung zum ROOMS Server installiert und konfiguriert.
