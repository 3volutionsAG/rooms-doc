Knowledge Base (Deutsch) : Gliederungen erfassen & bearbeiten

Verwandte Seiten

Einleitung

Zu Such- und Filterzwecken können individuelle Gliederungen erstellt werden. Eine Gliederung ist in eine Haupt- und mehrere Subgliederungen, wie z.B Gliederung nach Equipmenttyp beinhaltet Beamer, Laptop, Flipchart etc., aufgeteilt. Es werden von Garaio keine Gliederungen als Standard mitgeliefert.

  • Darf Gliederungen verwalten
  • Darf Sektion Einstellungen nutzen

Vorgehen / Beschrieb

Gliederungstyp erfassen

Navigation zu >Einstellungen>Erweiterte Basisdaten für Ressourcen>Gliederungen


Über den Button Gliederungstyp erstellen öffnet die Eingabemaske der Hauptgliederung. Die Felder und deren Funktionen sind der untenstehenden Tabelle zu entnehmen.

Bezeichnung

Eingabe einer sprechenden Bezeichnung für die Hauptgliederung wie z.B. Raumtypen

 


Ressourcenart

Auswahl (mehrfach Auswahl erlaubt) der Ressourcenart, für die die Gliederung später als Such- und Filterkriterium dienen soll

 

Es stehen nur die Ressourcenarten zur Verfügung auf denen die nötige Berechtigung (Rolle zu Benutzergruppe und Standort, bzw. Ressource zuweisen) vorhanden sind.

Aggregiert für

Wahl einer Ressourcenart, für die der Gliederungstyp zusammengezogen wird. Beispiel: Gliederungstyp "Equipmenttyp" von der Ressourcenart "Equipment" wird auf dem "Raum" aggregiert. Somit stehen dem Benutzer im Buchungsprozess alle Gliederungen - Beamer, Flipchart, etc. - auf dem Raum zur Verfügung. Dies erlaubt die Suche nach einem freien Sitzungszimmer mit Beamer.

Nach Eingabe aller Pflicht- und gewünschten optionalen Feldern wird der Gliederungstyp mit dem Button Speichern gesichert und erscheint in der Smartlist.

Gliederungstyp bearbeiten

Navigation zu >Einstellungen>Erweiterte Basisdaten für Ressourcen>Gliederungen
In der Smartlist werden alle bereits erfassten Gliederungstypen angezeigt. Mittels Klick auf die Bezeichnung erscheinen die Icons zur Bearbeitung, Ansicht und Löschung. Über das Ansichts-Symbol wird der Gliederungstyp im Viewmode dargestellt.


Über das Editieren-Symbol wechselt das System in den Bearbeiten Modus. Es können alle Felder verändert werden.

Gliederungstyp löschen

Navigation zu >Einstellungen>Erweiterte Basisdaten für Ressourcen>Gliederungen
In der Smartlist werden alle bereits erfassten Gliederungenstypen angezeigt. Mittels Klick auf die Bezeichnung erscheinen die Icons zur Bearbeitung, Ansicht und Löschung. Über das Löschen-Symbol erscheint eine Sicherheitsabfrage, ob der Gliederungstyp wirklich entfernt werden soll. Quittieren dieser Meldung entfernt den Status. Im Action Message wird die Anzahl der entfernten Einträge ausgegeben.


Die Löschung des Gliederungstyps kann nur erfolgen, sofern diesem keine Gliederungen mehr angehängt sind.

Navigation zu >Einstellungen>Erweiterte Basisdaten für Ressourcen>Gliederungen
In der Smartlist werden alle bereits erfassten Gliederungenstypen angezeigt. Mittels Klick auf die Bezeichnung erscheinen die Icons zur Bearbeitung, Ansicht und Löschung.


Über das Plus-Symbol öffnet die Eingabemaske im Register "Stammdaten". Die Felder und deren Funktionen sind der untenstehenden Tabelle zu entnehmen.

Im Register "Ressourcen" werden die Ressourcen hinzugefügt, für die die Gliederung verfügbar sein soll. Das heisst, dass beispielsweise Raum A ausgewählt wird und bei diesem Raum wiederum, kann die erstellte Gliederung selektiert werden (siehe z.B. Ressourcenart Raum erfassen & bearbeiten). Über den Button Ressourcen hinzufügen öffnet ein Modales Popup zur Auswahl der gewünschten Ressourcen.

Es werden nur die Ressourcen aufgelistet, die der Ressourcenart im übergeordneten Gliederungstyp entsprechen und für die genügend Berechtigung (z.B. Darf Ressourcentyp Raum verwalten) vorliegen.



Selektieren einer oder mehrerer Checkboxen und Klick auf den Button Ressource hinzufügen schliesst den Vorgang ab.

Im Register "Gruppe" werden die Gruppen hinzugefügt, für die die Gliederung verfügbar sein soll. Damit wird der Benutzerkreis eingeschränkt, der diese Gliederung später nutzen kann. Über den Button Hinzufügen öffnet ein Modales Popup zur Auswahl der gewünschten Gruppen.

Selektieren einer oder mehrerer Checkboxen und Klick auf den Button Gruppe hinzufügen schliesst den Vorgang ab.

Gliederung speichern
Nach Eingabe aller Pflichtdaten und der gewünschten optionalen Daten wird die Gliederung mittels Button Speichern gesichert und erscheint in Baumstruktur.

Gliederung bearbeiten

Navigation zu >Einstellungen>Erweiterte Basisdaten für Ressourcen>Gliederungen
In der Smartlist werden alle bereits erfassten Gliederungenstypen angezeigt. Über das Plus-Symbol öffnet sich die zweite Hirarchiestufe und zeigt alle untergeorneten Gliederungen an. Mittels Klick auf die Bezeichnung erscheinen die Icons zur Bearbeitung, Ansicht und Löschung.

Über das Editieren-Symbol öffnet sich die Gliederung im Bearbeiten Modus und alle Daten können verändert werden.

Gliederung löschen

Navigation zu >Einstellungen>Erweiterte Basisdaten für Ressourcen>Gliederungen
In der Smartlist werden alle bereits erfassten Gliederungenstypen angezeigt. Über das Plus-Symbol öffnet sich die zweite Hirarchiestufe und zeigt alle untergeorneten Gliederungen an. Mittels Klick auf die Bezeichnung erscheinen die Icons zur Bearbeitung, Ansicht und Löschung.

Mittels Klick auf das Löschen-Symbol erscheint eine Sicherheitsabfrage, ob die Gliederung wirklich entfernt werden soll. Quittieren dieser Meldung mit OK löscht die Gliederung. Im Action Message wird eine Meldung ausgegeben, dass der Eintrag erfolgreich gelöscht wurde. Die Löschung kann nur erfolgen, wenn keine Ressourcen mehr zugewiesen sind.
