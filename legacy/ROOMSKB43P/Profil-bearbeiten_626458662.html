Einleitung

Die Profil Einstellungen einer Person (Register "Stammdaten", Register "Navigation" und Register "Notifikationen") können bearbeitet werden

Vorbedingungen

  • Der Benutzer befindet sich auf seinem eigenen Profil ODER
  • Person wurde (bei entsprechender Berechtigung) im Bearbeitungsmodus geöffnet

Benötigte Rechte

Eigenes Profil:

  • keine

Fremdperson

  • Darf Sektion Listen nutzen ODER Darf Sektion Einstellungen nutzen
  • Darf Person sehen
  • Darf Person erstellen
  • Darf Person editieren intern
  • Darf Person editieren extern

Generell für Zusatzfunktionen

  • Darf Kostenträger anpassen
  • Darf Kostenträger ergänzen
  • Darf Stellvertreter editieren.
  • Darf Sektion Reservieren nutzen
  • Darf Sektion Plan nutzen
  • Darf Sektion Kalender nutzen
  • Darf Sektion Listen nutzen
  • Darf Sektion Reports nutzen
  • Darf Sektion Einstellungen nutzen

Standard Standort

Standard Standort, welcher für alle Standorte Initial übernommen wird

 

 

  • Feld kann leer sein
  • Gewählter Standort wird Initial in alle Standortpicker übernommen |

Standard Kapazität

Die Standardkapazität einer Raumbuchung, welche als Voreinstellung im Sidepanel des Menü "Reservieren" und bei einer Schnellreservation übernommen wird

5

  • Nur Ganzzahlen möglich
  • Ganzzahlen >= 0
  • Nur gültig bei der Ressourcenart „Raum“
  • Wird verwendet für Schnellreservationen als Standardwert
Stunden vorher berücksichtigen (Liste Buchungen)Setzt einen Standardwert bei der Suche nach Buchungen unter Listen (Stunden vorher berücksichtigen:)0 
  • Ganzzahlen >=0
  • Wird als Standardwert für "Stunden vorher berücksichtigen:" unter Menü LISTEN im Sidepanel Buchungen verwendet

Dauer Buchung in Min. (Vorgabe Schnellbuchung)

Der Standardwert der Schnellreservation, welcher in Minuten angegeben wird.

0

  • Nur positive Ganzzahlen möglich (>=0)
  • 0 = Schnellreservation erfolgt für den ganzen Tag oder bis zur nächsten Buchung
Anzeigedauer (in Sek.) der ActionMessagesWert wie Lange die nicht persistenten ActionMessages angezeigt werden sollen, bevor diese verschwinden  
  • Nur Ganzzahlen möglich
  • 0 = kein automatisches Verschwinden

Standard Kostenträger

Pro Person kann (bei entsprechender Berechtigung) ein oder mehrere Standard Kostenträger hinterlegt werden. Diese werden dann im Reservationsprozess übernommen

 

 

  • Rechte
    • Darf Kostenträger anpassen => erlaubt dem Benutzer den Standardkostenträger zu ändern
    • Darf Kostenträger ergänzen => erlaubt dem Benutzer eine zusätzlichen Kostenträger zu erfassen* Kostenträgerwahl über Intelibox
  • Klick auf Hinzufügen ergänzt den Kostenträger im Profil als Standardwert
  • Papierkorb Symbol löscht Kostenträger als Standard |

Farbe für Outlook-Items

Dropdown mit 25 Standardfarben zur Auswahl. Färbt während der Sync. den Outlook Kalendereintrag in der entsprechenden Farbe ein, in dem eine Kategorie erstellt wird und der Kalendereintrag dieser Kategorie zugeordnet wird.

 

 

  •  Nur wirksam, wenn Exchange Sync aktiv
  • Wichtig: In Outlook 2003 werden die alten Reservationen nach Farbände-rungen in Rooms Pro nicht ange-passt, die neuen schon. In Out-look 2007 funktioniert dies dann nach dem mind. 1 Buchung übers AddIn getätigt wurde.

Farbe für eigene Buchungen (Plan/Kalender)

Die Farbe der eigenen Buchungen wir horizontal ganz oben im Balken einer Kalenderbuchung angezeigt. Im Plan werden die Polygone von Buchungen mit der gewählten Farbe halbtransparent gefüllt

blau (0000ff)

 

 

Farbe Dritter auf Plan/Kalender einblenden

blendet die "eigenen definierten Farben" bei Buchungen von Dritten (anderer Nutzer) aus (nur gültig für den jeweils angemeldeten Benutzer)

nicht selektiert

 

 

Stellvertreter

Pro Person kann ein Stellvertreter definiert werden. Gewählte Person hat Stellvertreter Rechte = Stellvertreter darf alle Reservationen des Eigentümers lesen und editieren, bei welchen der Eigentümer Ersteller oder Organisator ist

 

 

  • Nur sichtbar mit Recht: Darf Stellvertreter editieren
  • Wahl einer Person über Intelibox |

  • Bild darf die Dateigrösse von 5 MB nicht überschreiten

Ressorucedtyp

Pro Ressourcentyp wird eine Zeile angezeigt, falls der Ressourcentyp aktiviert ist. Es stehen folgende Ressourcentypen zur Verfügung:

  • Arbeitsplatz
  • Equipment
  • Fahrzeug
  • Parkplatz
  • Raum
  • Benutzerdefinierte Ressourcen

 

 

in Abhängigkeit der aktivierten Ressorucenarten (Berechtigungen)

E-Mail aktiviert

Checkbox pro Ressourcenart für Aktivierung/Deaktivierung der E-mail Notifikation

nur Raum aktiviert

 

Sendet E-Mail Notifikationen für alle aktivierten Ressourcen

E-Mail Reminder

Zahlenpicker pro Ressourcenart für Erinnerungsfeld (mit auf/abPfeilen)
Die Erinnerungszeit wird in Minuten angegeben. Der Wert 0 bedeutet dass diese Notifikation ausgeschaltet ist.

0

0=deaktiviert
Angabe in Minuten

Outlook Sync

Checkbox pro Ressourcenart für die standardmässige Aktivierung der Synchronisation (Outlook <> ROOMS)

nicht aktiviert 
  • nur ab Exchange 2007 möglich
  • wird nur angezeigt wenn
    • die Sync im Tab "Personen Informationen" aktiviert ist
    • die entsprechende Ressourcenart für diesen Benutzer aktiv ist (benötigt Recht - z.B. darf Ressourcentyp "Arbeitsplatz" buchen)
ICS Datei versenden

Checkbox pro Ressourcenart für die standardmässige Versendung der ICS Datei

nicht aktiviert 
  • Wird in der Spalte "E-Mail aktiviert" eine Ressourcenart selektiert, kann die Ressourcenart in der Spalte ICS Datei auch selektiert werden. Ansonsten ist die Checkbox für Eingaben gesperrt.
  • Wird in der Spalte "E-Mail aktiviert" eine Ressourcenart deselektiert, wird der Hacken auch bei der ICS Datei entfernt und diese Checkbox wird wieder für Eingaben gesperrt.

Serie Reminder

0

0=deaktiviert
