Einleitung

In diversen Teilen der Applikation kann eine intelligente Suche genutzt werden. Die dafür vorgesehenen Inteliboxen, liefern nach Beginn der Eingabe vom ersten Buchstaben/Ziffer weg eine Auswahl an verfügbaren Resultaten.

Generelle Regeln für Inteliboxen

  • Globale Konfiguration (siehe Globale System Parameter konfigurieren) der Anzahl der maximalen visualisierten Einträge (falls das Suchresultat grösser ist) – Standard: 15 Konfigurationsparameter / Configuration parameter
  • Suche wird ab dem Schreiben des ersten Zeichens ausgelöst
  • Kennzeichnung:
    • Jede Intelibox wird mit einem Icon gekenzeichnet

  • Navigation:
    • Tabulator: normales Verhalten => springt aus dem Feld
    • Pfeiltasten zum Navigieren: ↑ = nach oben, ↓ = nach unten; ← innerhalb des Textes nach links und → innerhalb des Textes nach rechts
    • Entertaste der markierte Text wird selektiert (mit Mausklick wird dasselbe erreicht)
    • Backspace: Zeichen löschen und neue Suche
  • Layout:
    • Suchresultat wird unterhalb der Intelibox dargestellt – wie aktuelles ROOMS PRO

  • Falls Text zu lang ist wird er abgeschnitten und mit drei Punkten ergänzt.
    • Customized Tool Tip zeigt den ganzen Text
    • Tooltip ist auch auf dem selektierten Eintrag vorhanden.

Spezialfall Person

  • Tooltip kann um weitere Informationen angereichert werden z.B. Telefonnummer (Übersetzungen verwalten). Dies steht immer im Kontext zu dem Ort an dem die Intelibox eingesetzt wird.
  • Zusätzlich hat die Intelibox ein PersonenInfo Icon. Das Icon wird bei auch bei den nicht änderbaren Personen (z.B. Viewmode oder Ersteller angezeigt).
  • Klick auf das Icon hat folgende Funktion:
    • Ist das Feld "URL" nicht leer (siehe Personen und Gruppen verwalten), wird der unter URL angegebene Hyperlink in einem neuen Fenster geladen.
    • Ist das Feld "URL" leer wird in einem modalen Popup der Viewmodus einer Person geladen.

  • Der Inhalt dieser Intelibox kann global definiert werden (pro Kunde/Brand). Standard ist: Nachname Vorname (OE)
  • Beim Mouse-Over über die Intelibox (oder Label) werden dem Benutzer zusätzliche Informationen zur Person in Form eines Tooltips angezeigt. Dies kann global pro Kunde/Brand definiert werden. Standard ist: Telefon, Mobile, E-Mail.
  • Suche nach Personalnummer und nach Vorname Name und Name Vorname

Spezialfall Intelibox Firma/OE

  • Je nach Berechtigung kann nur nach Organisationseinheit oder nur nach Firma gesucht werden.

    Wichtig
    Wird der Inhalt oder das Mouse-Over der Intelibox mit Adressfeldern versehen, wird stets die Adresse angezeigt, die als Standard definiert ist.

Löschen einer Auswahl

  • Wurde ein oder mehrere Einträge ausgewählt, werden diese in das aktuelle Feld (Control) übernommen
  • Der Eintrag kann mit Klick auf das eingeblendete X gelöscht werden. Ebenfalls kann über das Plus-Symbol die Selektion gelöscht werden (nur bei Multiselektoren).
