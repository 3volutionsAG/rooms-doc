Knowledge Base (Deutsch) : Planlabels konfigurieren

Created by Former user, last modified by Former user on Apr 30, 2014

Artikel Nummer

Gültig ab Release

4.2

Verwandte Seiten

Folgende Platzhalter können verwendet resp. werden in folgenden Situationen auf dem Plan auch visualisiert werden:

1. Wenn der Raum frei ist:

  • Ressource.Name
  • Ressource.Bezeichnung

2. Wenn der Raum frei oder besetzt ist:

  • Ressource.NextReservation

3. Wenn der Raum besetzt ist:

  • Organisator.Nachname
  • Organisator.Vorname
  • Organisator.E-Mail
  • Organisator.Firma
  • Organisator.Telefon
  • Organisator.Kurzzeichen
  • Ersteller.Name
  • Ersteller.Vorname
  • Ersteller.E-Mail
  • Ersteller.Firma
  • Ersteller.Telefon
  • Ersteller.Kurzzeichen
  • Reservation.Titel
  • Reservation.Nummer
  • Reservation.AnzPersonen
  • Reservation.Datum_ZeitVon
  • Reservation.Datum_ZeitBis

Globales Feld heisst:
Plan Label

Ressourcen Feld heisst:
Plan Label

Folgende Syntax muss eingehalten:
[ Ersteller.Vorname\ ]
