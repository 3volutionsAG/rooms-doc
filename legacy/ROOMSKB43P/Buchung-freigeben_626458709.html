Knowledge Base (Deutsch) : Buchung freigeben

Verwandte Seiten

 

Einleitung

Ziel der Freigabe ist es, eine laufende Buchung per sofort freigeben zu können.

Vorbedingungen

Benutzer: Benötigt Rechte um das gewählte Menü zu sehen und mindestens Leserechte für die darin verfügbaren Ressourcen.

Administratoren: Finden einen detaillierten Beschrieb im KB Artikel Rechte.

Vorgehen

In den drei Menüs PLAN, KALENDER und LISTEN ist es möglich, eine Freigabe vorzunehmen.

 

Freigabe im Menü PLAN

  • Über den Standortbaum oder die Plansymbole zur gewünschten Buchung navigieren.
  • Klick auf das rote Symbol.
  • Im anschliessend erscheinenden Popup Klick auf Button Freigeben.

  • Oben rechts erscheint eine Hinweisbox, welche die erfolgreiche Freigabe bestätigt.

 

 

Freigabe im Menü KALENDER

  • In der Kalenderansicht zur gewünschten Buchung navigieren.
  • Klick auf das „+“-Symbol.
  • Im anschliessend erscheinenden Popup Klick auf Button Freigeben.

  • Oben rechts erscheint eine Hinweisbox, welche die erfolgreiche Freigabe bestätigt.

 

 

Freigabe im Menü LISTEN

  • In der Listenansicht zur gewünschten Buchung navigieren.
  • Wenn in der Spaltenreihenfolge so eingestellt, erscheint die Spalte "Freigabe" mit dem Icon Freigeben.
  • Klick auf das Icon Freigeben.

  • Es erscheint ein Popup, welches nachfragt ob man die Buchung wirklich freigeben möchte.

  • Bestätigung durch Klick auf Button OK.

  • Oben rechts erscheint eine Hinweisbox, welche die erfolgreiche Freigabe bestätigt.
