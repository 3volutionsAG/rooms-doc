Einleitung

Jede Smartlist beinhaltet immer das Personalisieren von Spalten, Einstellung des Aktualisierungs Intervall, Sortierung und Seitenwechsel (Paging) sowie optionale Checkboxen für die Mehrfachselektion.

Vorbedingungen

Keine

Vorgehen / Beschrieb

Der Benutzer kann jede Smartlist nach seinen Bedürfnissen einstellen.

Sortierung

  • Spalten mit Icons sind nicht sortierbar und haben einen schwarzen Titel
  • Eine Sortierung ist nur über jeweils 1 Spalte möglich
  • Klick auf eine Spalte sortiert aufsteigend (A-Z), beim zweiten Klick auf dieselbe Spalte ändert die Sortierung zu absteigend (Z-A)
  • Die aktuelle Sortierung wird mit einem Zeichen (Pfeil) visualisiert

Seitenwechsel (Paging)

Der Seitenwechsel (Zeilen Pro Seite) kann durch den Benutzer pro Ansicht eingestellt werden. Mittels Klick auf entsprechenden Button öffnet die Auswahlliste unterschiedlicher Zeilenanzahlen pro Seite

  • Die Auswahlmöglichkeiten werden global (pro Kunde konfiguriert)

Anzahl Spalten und Spaltenreihenfolge

Pro Listenansicht wird die Anzahl der Spalten fix vorgegeben. Der Benutzer kann jede Ansicht gemäss seinen Vorstellungen verändern. Mittels Klick auf entsprechenden Button öffnet die Auswahlliste mit allen verfügbaren Spalten.

  • Reihenfolge: Die Spaltenreihenfolge kann mittels Drag und Drop innerhalb des Popups vorgenommen werden.
  • Die Smartlist wird aktualisiert sobald das Popup nach Klick auf Button Übernehmen geschlossen wird.
  • Spaltenbreite kann durch „Maus-ziehen" im Spaltenkopf des Smartlist geändert werden.

Autorefresh

Stellt dem Benutzer die möglichen Intervalle für die Aktualisierung des Smartlist zur Verfügung.

  • Ist die Aktualisierung eingeschaltet, wird die Smartlist im angegebenen Intervall aktualisiert.
  • Benutzer kann „Aktualisieren" über den entsprechenden Button aktivieren
  • Ein Intervall kann gewählt oder die Aktualisierung ausgeschalten werden.

Navigationsbuttons

Die Navigationsbuttons sind wie folgt aufgebaut:

  • Button mit Pfeil für eine Page vor und eine zurück
  • Anzeigen der ersten Seite mit 1
  • Anzeigen der letzten Seite in Zahlenform (z.B. 3)
  • Anzeigen der maximal 5 mittleren Seiten. Beispiel: Nutzer ist auf Page 10 von 13, also werden ihm noch die Seiten 8, 9 und 11, 12 nebst der 10 visualisiert. Details unter Gesamt Layout
  • Navigationsbuttons bleiben fix positioniert.

Weitere Regeln

Grösse: Die Liste hat eine fixe minimal Grösse und verfügt über eine Scrollbar – nur Vertikal

Suchfelder: Grundsätzlich verfügen die Smartlist über Suchfelder pro Spalte (oberhalb des Spaltentitels). Es kann sein, dass in bestimmten Inhaltskontexten die Suchfelder ausgeblendet werden, um platzoptimierter arbeiten zu können. Dies erfolgt nur sofern alle Resultate auf einer Seite dargestellt werden können. Dies benötigt eine globale Einstellung (_Links einfügen): Suchfelder in Smartlist automatisch ausblenden = True
