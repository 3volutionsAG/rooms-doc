Einleitung

Die Erstellung und Verwaltung von Benutzergruppen und das Zufügen von Personen werden in diesem Artikel erläutert.

Ausserdem wird die Möglichkeit der Nutzung von Notifikationensvorlagen, welche Benutzergruppenabhängig sind, beschrieben.

Vorbedingungen

Personen sind im System vorhanden und erfasst. Siehe dazu KB Person suchen & finden und Person bearbeiten.

Benötigte Rechte

  • Darf Benutzergruppen verwalten

Vorgehen / Beschrieb

Benutzer navigiert auf >Einstellungen >Sicherheitsdaten für Personen >Benutzergruppen verwalten.

Klick auf Button Anzeigen zeigt alle vorhandenen Benutzergruppen gemäss Suchkriterien in der Smartlist:

Die Smartlist zeigt nebst der Gruppenbezeichnung auch den Verwendungszweck der Benutzergruppe (Berechtigungsgruppe, Notifikationsgruppe und Benutzerkreisgruppe).

Klick auf Bezeichnung zeigt den Anzeigemodus (Viewmode) der Benutzergruppe (Stammdaten, Anlasstypen und Notifikationen):

Klick auf Button Bearbeiten wechselt in den Bearbeitungsmodus. Dasselbe wird erreicht durch Klick auf Bearbeiten Icon oder Klick auf Button Neu in der Smartlist.

Der Benutzer befindet sich im Register "Stammdaten"

Der Benutzer klickt auf Register "Personen"

Eine Liste aller der Benutzergruppe zugewiesenen Personen wird angezeigt (Vorname, Nachname, E-Mail, Sprache, Mobile Nummer, Initialen):

Selektion einer oder mehreren Personen und Klick auf Button Entfernen entfernt die Person von der Benutzergruppe (Die Person wird nicht gelösch)

Klick auf Button Person hinzufügen öffnet modalen Dialog. Dieser zeigt alle der Benutzergruppe noch nicht hinzugefügten Personen an:
 
Eine oder mehrere Personen können selektiert werden und über Button Person hinzufügen der Benutzergruppe hinzugefügt werden.

Anmerkung: Alternativ können Personen auch direkt auf der Person einer Benutzergruppe zugeordnet werden (siehe dazu KB Benutzergruppen zur Person zuweisen)

Der Benutzer klickt auf Register "Notifikationen"

In diesem Bereich können die Notifikationstemplate hinzugefügt und konfiguriert werden. Die Details sind dem KB Notifikationen Benutzergruppen bezogen erfassen und bearbeiten zu entnehmen.
