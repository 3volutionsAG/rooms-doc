Knowledge Base (Deutsch) : Ressourcenart Fahrzeug erfassen & bearbeiten

Created by Former user, last modified by Former user (Deleted) on Feb 12, 2015

Artikel Nummer

Gültig ab Release

4.5

Verwandte Seiten

Einleitung

Damit im späteren Buchungsprozess die Ressourcenart Fahrzeug zur Verfügung steht, müssen diese entsprechend erfasst und mit allen notwendigen Stammdaten befüllt werden.

Vorbedingungen

  • Falls Gliederungen (Gliederungen erfassen & bearbeiten) für das Fahrzeug vorgesehen sind, wie z.B. die Gliederung nach Fahrzeugtypen wie z.B. Kombi, Lieferwagen etc. müssen diese initial erfasst und für die Ressourcenart Fahrzeug verfügbar sein.
  • Tarife müssen initial erfasst worden sein, falls diese später im Buchungsprozess genutzt werden wollen
  • Rollen sollten vorgängig definiert werden (Rechte & Rolle verwalten)
  • TIPP: Berechtigungen sollten bereits auf Standortebene (Standorte erfassen & bearbeiten) vor erfasst werden, da diese dann bis auf Ressourcenebene vererbt werden.

Benötigte Rechte

  • Darf Ressourcentyp Fahrzeug verwalten
  • Darf Ressourcen bearbeiten
  • Darf Sektion Einstellungen nutzen
  • Darf Ressourcentyp mit Sperrzeiten verwalten
  • Darf Berechtigungen für Ressourcen verwalten
  • Darf Feiertage verwalten (sofern die vom Standort vererbten Feiertage überschrieben oder ergänzt werden sollen)
  • Darf Ressourcentyp mit Tarifen verwalten (sofern die Tarife genutzt werden wollen)

Automatische Verlängerung

Ermöglicht im Buchungsprozess alle Buchungen auf diesem Fahrzeug jeweils automatisch um eine bestimmte Minutenzahl zu verlängern.

0 = ausgeschaltet

 

  • Wert >0 verlängert die Buchung um die eingetragene Minutenzahl
  • Prüfung erfolgt auf Ganzzahlen
  • Der Ausführungszeitpunkt ist das Buchungsende minus 5 Minuten
  • Eine automatische Verlängerung löst keine E-Mail Notifikationen aus
  • Synchronisierte Buchungen werden im Outlook nachgetragen, jedoch OHNE Aktualisierung für die Teilnehmer
  • Eine Buchung mit automatischer Verlängerung endet bei einem Checkout (siehe Handbücher 4.0 - 4.2), Konflikt mit der nächsten Buchung oder am Ende des aktuellen Kalendertages (23:59) - (Konfliktsituation: Ausgabe einer Meldung im Action Message des Erstellers und Organisators. Notifikation wird an Ersteller und Organisator versandt, sofern die korrekte Notifikationsvorlage erstellt ist)

Medien & Beilagen hinzufügen

Im Register "Medien & Beilagen" können Medien wie z.B. Bilder oder auch Dateien (Word, pdf) zum Fahrzeug hinzugefügt werden. Klick auf den Button Hinzufügen öffnet das modale Popup zum Hochladen der Dateien/Bilder.


Die Bezeichnung muss mehrsprachig (Daten mehrsprachig erfassen) erfasst werden. Mittels Button Browse wird im Dateiverzeichnis nach dem gewünschten Bild gesucht. Über den Button Upload wird das Bild neben dem Text Zugewiesenes Bild eingefügt. Der Vorgang wird mit Klick auf den Button Übernehmen abgeschlossen und das Bild erscheint im Register "Medien & Beilagen".

Medien & Beilagen entfernen

Im Register "Medien & Beilagen" sind alle bereits hochgeladenen Bilder und Datein aufgelistet. Durch Selektion einer oder mehrerer Checkboxen und anschliessender Bestätigung über den Button Löschen erscheint eine Sicherheitsabfrage, ob das Medienelement wirklich gelöscht werden soll. Wird diese Meldung mit OK quittiert, ist der Löschvorgang abgeschlossen und im Action Message wird die Anzahl der gelöschten Einträge ausgegeben.

Sperrzeiten hinzufügen

Im Register "Sperrzeiten" erfolgt die Definition von Zeiträumem, in denen das Fahrzeug nicht gebucht werden kann und später im Buchungsprozess (Farbgebung im Kalender) als nicht verfügbar visualisiert wird. Mittels Button Hinzufügen öffnet das modale Popup zur Erstellung einer Sperrzeit. Der Titel der Sperrzeit muss mehrsprachig erfasst werden. Die Festlegung des gesperrten Zeitraumes erfolgt über die Datum und Zeit Felder. Der Vorgang wird über den Button Speichern abgeschlossen und die Sperrzeit wird hinzugefügt.

Sperrzeiten entfernen

Im Register "Sperrzeiten" sind alle bereits definierten Sperrzeiten aufgeführt, auch diejenigen, die bereits auf Standortebene (siehe Standorte erfassen & bearbeiten) gesetzt und vererbt wurden. Durch Selektieren einer oder mehrerer Checkboxen und anschliessender Bestätigung über den Button Entfernen erscheint eine Sicherheitsabfrage, ob die Sperrzeit wirklich entfernt werden soll. Quittieren dieser Meldung entfernt alle Sperrzeiten und das Fahrzeug steht wieder zum Buchen zur Verfügung. Im Action Message wird die Anzahl der entfernten Einträge ausgegeben.

 

Berechtigungen hinzufügen

Im Register "Berechtigungen" werden die Benutzergruppen mit den gewünschten Rollen bzw. Berechtigungen (siehe Rechte & Rolle verwalten) hinzugefügt. Klick auf den Button Hinzufügen öffnet das modale Popup zur Auswahl der gewünschten Berechtigungen bzw. Rollen. Selektieren einer oder mehrerer Rollen und Auswahl der Benutzergruppe über die Intelibox. Der Vorgang wird mit Klick auf den Button Speichern abgeschlossen.

 

Werden keine Berechtigungen gesetzt, übernimmt das System diejenigen vom hierarchisch übergordneten Standort.

 

Berechtigungen entfernen

Im Register "Berechtigungen" sind alle bereits definierten Berechtigungen aufgeführt, auch diejenigen, die bereits auf Standortebene (Rechte & Rolle verwalten) gesetzt und vererbt (ist rekursiv) wurden. Durch Selektieren einer oder mehrerer Checkboxen und anschliessender Bestätigung über den Button Entfernen erscheint eine Sicherheitsabfrage, ob die Berechtigung wirklich entfernt werden soll. Quittieren dieser Meldung entfernt alle Berechtigungen. Im Action Message wird die Anzahl der entfernten Einträge ausgegeben.

 

Wichtig:
Werden alle Berechtigungen entfernt, so ist das Fahrzeug nicht mehr editierbar und es muss auf Standortebene (Standorte erfassen & bearbeiten) die Berechtigung neu gesetzt werden!

Feiertage hinzufügen

Im Register "Feiertage" werden die Feiertage für das betreffende Fahrzeug ausgewählt. Im späteren Buchungsprozess sind die Feiertage visualisiert (siehe auch Farbgebung im Kalender) und nur buchbar mit dem Recht an Feiertagen buchen zu dürfen. Klick auf den Button Hinzufügen öffnet das modale Popup zur Auswahl der gültigen Feiertage. Selektieren einer oder mehrerer Checkboxen und anschliessende Bestätigung über den Button Hinzufügen listet die Feiertage des Fahrzeugs auf.

 

In der Spalte "Ist Rekursiv" ist mit ja oder nein visualisiert, ob der Feiertag von einem höheren Standort geerbt wurde.

Feiertage entfernen

Im Register "Feiertage" sind alle bereits definierten Feiertage aufgeführt, auch diejenigen, die bereits auf Standortebene (siehe Standorte erfassen & bearbeiten) gesetzt und vererbt wurden. Durch Selektieren einer oder mehrerer Checkboxen und anschliessender Bestätigung über den Button Entfernen erscheint eine Sicherheitsabfrage, ob die Feier wirklich entfernt werden soll. Quittieren dieser Meldung entfernt alle Feiertage vom Fahrzeug. Im Action Message wird die Anzahl der entfernten Einträge ausgegeben.

Fahrzeug speichern

Nach Eingabe aller Pflichtdaten und der gewünschten optionalen Daten wird das Fahrzeug mittels Button Speichern gesichert und erscheint in der Smartlist.

Fahrzeug bearbeiten

Navigation zu >Einstellungen>Ressourcen
In der Smartlist werden alle bereits erfassten Ressourcen, auf denen der angemeldete Benutzer die nötigen Rechte besitzt, angezeigt. Mittels Klick auf die Bezeichnung wird das Fahrzeug im Anzeigemodus (Viewmode) dargestellt.


Mittels Button Bearbeiten oder über das Editieren-Symbol wechselt das System in den Bearbeiten Modus.


In allen, bereits vorgängig in diesem Artikel beschriebenen Registern, können Anpassungen vorgenommen werden. Die Änderungen werden mit dem Button Speichern übernommen.

Selektieren der Checkbox "Beim Speichern Notifikation an Betroffene senden" vor dem Speichervorgang löst eine E-Mail Notifikation an alle Organisatoren aus, die in der Zukunft liegende Buchungen auf diesem Fahrzeug haben. Dazu muss die Notifikationsvorlage (siehe Notifikationen standortbezogen erfassen & bearbeiten) Ressource - Anpassung konfiguriert und aktiviert sein.

Fahrzeug löschen

Navigation zu Einstellungen>Ressourcen und optionale Eingabe eines Suchbegriffs (siehe Suchen & Finden).
Klick auf den Button Finden zeigt in der Smartlist das Suchresultat bzw. alle bereits erfassten Räume bzw. Ressourcen an. Durch Selektieren der Checkbox (Mehrfachselektion möglich) und Bestätigung über den Button Löschen erscheint eine Sicherheitsabfrage, ob das Fahrzeug wirklich entfernt werden soll. Quittieren dieser Meldung entfernt das Fahrzeug. Die Löschung ist nur möglich, wenn keine Buchungen auf dem Fahrzeug bestehen. Im Action Message wird die Anzahl der entfernten Einträge ausgegeben.
