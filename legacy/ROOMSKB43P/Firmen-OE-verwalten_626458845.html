Einleitung

Firmen (= Organisationseinheiten und Abteilungen) können erfasst werden. Dazu können Adressen erfasst werden und beliebige Kostenträger und Personen zugeordnet werden.

Vorbedingungen

Für die Zuordnung von Personen und Kostenträger müssen die entsprechenden Daten bereits vorhanden sein (Personen erfasst oder importiert; Kostenträger  importiert)

Benötigte Rechte

  • Darf Firmen verwalten (extern)
  • Darf Firmen verwalten (intern)
  • Darf Firma mit Rabatten editieren
  • Darf Firmen mit Person verwalten

Vorgehen / Beschrieb

Benutzer navigiert auf >Einstellungen >Sicherheitsdaten für Personen >Firmen/OE

Eingabe der Suchkriterien

Klick auf Button Finden

Suchresultat mit allen verfügbaren Spalten wird in der Smartlist angezeigt

Durch Klick auf Button Entfernen können ein oder mehrere selektierte Firmen/OE gelöscht werden.

  • Die zur Firma/OE gehörenden Adressen werden dabei auch gleich gelöscht, sowie die Referenz/Verlinkung zu den Personen
  • Die Personen selber werden NICHT gelöscht

Klick auf Link des Feldes "Bezeichnung" öffnet die Firma im Anzeigemodus (Viewmode)

Standardrabatt in %

Auf einem Kunden (intern oder extern) kann ein Standardrabatt hinterlegt werden:

  • Im Reservationsprozess (Detailreservation) wird der Rabatt in Form eines Labels ausgewiesen (hinter dem Kostenfeld)
  • Bei Zeitänderung oder anderen Kostenänderung (Umstuhlung, Dienstleistungen, etc.) wird der Wert neu berechnet werden
  • Dasselbe gilt auf dem Anlass. In der Detailansicht und der Summary wird das Label angezeigt.
  • Auf der Summary Page wird dann der effektive Rabatt errechnet und ein Eintrag im Bereich der Rabatte generiert und entsprechend auf der Summary ausgewiesen.
  • Dieser Rabatt Record wird bei jeder weiteren Editierung der Reservation wieder gelöscht und bei Reservationsabschluss neu erstellt. So wird sichergestellt, dass immer der Stammrabatt über den Gesamtbetrag neu berechnet wird.

 

 

  1. Recht „Darf Rabatte auf Firma editieren“ notwendig, um in den Stammdaten das Feld abzufüllen
  2. Das Feld muss auf Ganzzahlen ≤100 validiert werden. Minuszahlen sind nicht Zulässig

Kostenträger zur Firma zuordnen oder entfernen

Nutzer navigiert auf Register "Kostenträger". Eine Liste der zugeordneten Kostenträger wird angezeigt (Felder: Nummer und Bezeichnung)

Eine oder mehrere Kostenträger können nach Selektion über den Button Entfernen gelöscht werden.

Klick auf Button Hinzufügen öffnet ein Modales Popup zur Zuordnung zusätzlicher Kostenträger.

Ein oder mehrere Kostenträger können selektiert werden und mit Klick auf Button Hinzufügen der Firma/OE hinzugefügt werden

  • Es werden ausschliesslich Kostenträger angezeigt, welche noch keiner anderen OE/Firma zugeordnet worden sind
  • Hinweis: Kostenträger, welche einer Firma/OE hinzugefügt worden sind, sind im Buchungsprozess exkl. für diese Firma sichtbar

Alle Daten (auch Adressen, Personen, Kostenträger) werden erst gespeichert, wenn die Firma/OE gespeichert wird. Über den Button Abbrechen können alle Änderungen verworfen werden
