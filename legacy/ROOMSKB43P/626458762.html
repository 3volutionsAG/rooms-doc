Knowledge Base (Deutsch) : Elektronisches Türschild auf Ressource einrichten

Verwandte Seiten

 

Einleitung

Damit das elektronische Türschild für eine Ressource genutzt werden kann, muss diese entsprechend eingerichtet werden. Der nachfolgende Artikel enthält alle wichtigen Details zur Konfiguration des Türschildes auf einem Raum.

Vorbedingungen

Elektronisches Türschild (Tablet) ist vorhanden.

Lizenzen für Türschilder wurden erworben und installiert.

IIS Konfiguration mit anonymous authentication eingerichtet.

Globale Konfiguration "Applikation Root-Url Anonymous" gemäss der IIS Konfiguration vorgenommen.

Benötigte Rechte

  • Darf Ressourcen bearbeiten
  • Darf Sektion Einstellungen nutzen
  • Darf Person editieren intern
  • Darf Person editieren extern
  • Darf Ressourcentyp Raum verwalten

Vorgehen

Aktivierung Türschild

Navigiere zu Einstellungen>Ressource 

Einstieg in den Bearbeiten Modus des gewünschten Raumes für den das Türschild eingerichtet werden soll. Wechsel in das Register "Türschild". 

Mittels Intelibox erfolgt die Auswahl des Benutzers, der die Konfiguration des Türschildes (siehe dazu auch Konfiguration Türschild) ausführen darf.

Für den Türschildbenutzer ist es sinnvoll eine eigene Gruppe/Rolle & Benutzer anzulegen, welche nicht über einen allfälligen Benutzerdatenimport angelegt wird.

Folgende Konfiguration wird vorgeschlagen:

Benutzer: On Site ROOMS (einzelner Benutzer reicht aus)

Ist der Benutzer ausgewählt wird der Button Türschild aktivieren aktiviert und mit Klick darauf, erstellt ROOMS den nötigen QR Code und blendet weitere Felder ein. Die Felder und deren Funktionen werden in der unten stehenden Tabelle erläutert.

  • Lizenzierung wird geprüft
  • Falls genügend Lizenzen verfügbar, wird folgendes Control eingeblendet
    1. Intelibox: Konfigurationsbenutzer wählen 
  • Ist eine Person selektiert, wird folgendes Control eingeblendet

Wenn keine bzw. nicht mehr genügend Lizenzen verfügbar sind, erscheint folgende Meldung: "Es sind keine Lizenzen mehr verfügbar, um dieses Türschild zu aktivieren. Bitte deaktivieren Sie ein anderes Türschild oder erwerben weitere Lizenzen."

 

Deaktivierung Türschild

Navigiere zu Einstellungen>Ressource

Einstieg in den Bearbeiten Modus des Raumes für den wie oben beschrieben das Türschild aktiviert wurde. Im Register "Türschild" muss der Button Türschild Deaktivieren geklickt werden.

Das Türschild wird deaktiviert und alle Felder werden ausgeblendet. Der QR Code wird gelöscht und der Raum, sowie die Lizenzen werden freigegeben. 

Die Daten werden beim Deaktivieren im Hintergrund gespeichert und stehen beim Erneuten Aktivieren (siehe oben) wieder zur Verfügung.
