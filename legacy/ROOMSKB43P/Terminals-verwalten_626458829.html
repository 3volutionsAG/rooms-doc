Einleitung

Verwaltung von Konfigurationen zur Ansteuerung von Terminals - z.B. SPS Steuerungen - über TCP/IP.

Vorbedingungen

Aktuell werden folgende Terminals unterstützt:

  • SPS Steuerungen

Benötigte Rechte

  • Darf Terminals verwalten

Vorgehen / Beschrieb

Benutzer navigiert auf >Einstellungen >Erweiterte Basisdaten für Ressourcen >Terminals und Klick Button Finden.

Optional können Suchkritieren eingegeben werden:

Ein Suchresultat über alle verfügbaren Terminals werden in einer Smartlist dargestellt (Spalten: Bezeichnung, Terminaltyp). Eine oder mehrere Terminals können mit Klick auf Button Entfernen gelöscht werden.

Klick auf die "Bezeichnung" zeigt das Terminal im Anzeigemodus (Viewmode). Klick auf Icon Bearbeiten wechselt in das Register "Stammdaten" des Bearbeitungsmodus:

Klick auf Register "Ressourcen" erlaubt die Zuweisung (Verlinkung) der betroffenen Ressource zum Terminal.

Klick auf Button Hinzufügen öffnet ein Modales Popup und zeigt alle verfügbaren Ressourcen an. Zur Identifikation werden die Spalten (Name, Bezeichnung, Ressourcenart, Standort Hierarchisch und Status) angezeigt.
