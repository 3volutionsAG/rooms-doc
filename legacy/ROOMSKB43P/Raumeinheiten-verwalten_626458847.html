Knowledge Base (Deutsch) : Raumeinheiten verwalten

Created by Former user (Deleted), last modified on Feb 11, 2015

Artikel Nummer

Verwandte Seiten

Einleitung

Verwaltung von Raumeinheiten, die als Raum-/Fläche Einheit genutzt werden können, um eine beliebige Fläche (Openspace, Büro, etc.) zu definieren und dieser Fläche Personen zuzuweisen.

Vorbedingungen

Keine

Benötigte Rechte

  • Darf Raumeinheit sehen
  • Darf Raumeinheit bearbeiten

Vorgehen / Beschrieb

Benutzer navigiert auf >Einstellungen>Raumeinheiten

Optional können Suchkritieren eingegeben werden:

Ein Suchresultat über alle Raumeinheiten werden in einer Smartlist dargestellt. Eine oder mehrere Raumeinheiten können selektiert und mit Klick auf Button Entfernen gelöscht werden.

Klick auf die "Bezeichnung" zeigt die Raumeinheit im Anzeigemodus (Viewmode). Klick auf Icon Bearbeiten wechselt in das Register "Stammdaten" des Bearbeitungsmodus:

 

Feld

Funktion

Standardeinstellung

Eigenschaften

Regeln

Bezeichnung

Eine sprechende Bezeichnung zur Identifikation der Raumeinheit muss mehrsprachig erfasst werden

 

 

Beschreibung

Eine sprechende Beschreibung der Raumeinheit kann erfasst werden

 

 

Standortpicker

Auswahl des Standortes, auf dem sich die Raumeinheit befindet

 

 
KostenträgerMöglichkeit zur Zuweisung eines Kostenträger auf die Raumeinheit   
Anzahl ArbeitsplätzeAngabe der Anzahl Arbeitsplätze auf der Raumeinheit  Validierung auf Ganzzahlen
FlächeAngabe der Fläche in m2  Validierung auf Dezimalzahlen
HöheAngabe der Raumhöhe in Meter  Validierung auf Dezimalzahlen
GewichtungMöglichkeit einer Einstufung z.B. nach Wichtigkeit  Validierung auf Dezimalzahlen

= Pflichtfelder
= Daten mehrsprachig erfassen

Klick auf Register "Personen" erlaubt die Zuweisung der betroffenen Personen zur Raumeinheit.

Klick auf Button Person hinzufügen öffnet ein Modales Popup und zeigt alle verfügbaren Personen an. Eine Person kann mehrmals derselben Raumeinheit und auch mehreren Raumeinheiten zugewiesen werden (auch zeitlich überschneidend)

Mit Klick auf Button Hinzufügen wird die Person der Raumeinheit hinzugefügt.

Klick auf Button Speichern schliesst den Vorgang ab
