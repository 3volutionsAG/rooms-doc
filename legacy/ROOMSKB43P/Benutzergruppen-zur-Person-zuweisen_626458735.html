Knowledge Base (Deutsch) : Benutzergruppen zur Person zuweisen

Created by Former user, last modified on Jul 05, 2013

Artikel Nummer

Gültig ab Release

4.2

Verwandte Seiten

Einleitung

Bei entsprechender Berechtigung kann einer Person eine oder mehrere Benutzergruppen zugewiesen werden

Vorbedingungen

Benutzer befindet sich im Bearbeiten Modus (siehe KB Person bearbeiten) einer Person

Benötigte Rechte

  • Darf Benutzergruppen verwalten

Vorgehen / Beschrieb

Klick auf Register "Benutzergruppen"

Smartlist zeigt Spalten:

  • Bezeichnung
  • ist Berechtigungsgruppe
  • Ist Notifikationsgruppe
  • Ist Teilnehmergruppe
  • Ist Benutzerkreis

Klick auf Button Hinzufügen

  1. Modales Popup öffnet sich
  2. Alle verfügbaren Benutzergruppen stehen zur Auswahl
  3. Es werden dieselben Spalten angezeigt, wie im ersten Schritt
  4. es kann eine oder mehrere Benutzergruppen selektiert werden


Klick auf Button Hinzufügen fügt die Benutzergruppe zur Person (Smartlist wird aktualisiert)

Button Entfernen, entfernt alle gewählten Benutzergruppen aus der Smartliste

Erst der Klick auf den Button Speichern der Person, speichert auch die zugefügten Benutzergruppen. Bei Klick auf den Button Abbrechen werden die Änderungen verworfen
