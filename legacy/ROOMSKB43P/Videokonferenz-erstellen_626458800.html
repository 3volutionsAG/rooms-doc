Knowledge Base (Deutsch) : Videokonferenz erstellen

Created by Former user, last modified by Former user on Mar 10, 2015

Artikel Nummer

Gültig ab Release

4.4

Verwandte Seiten

Einleitung

Im Subsidepanel "Video Konferenz" wird das Buchen einer Videokonferenz unterstützt.

Vorbedingungen

VC Buchung mit oder ohne TMS (kann global konfiguriert werden: Eingabe der "TMS ROOT URL")

Benötigte Rechte

  • Darf Anlässe als Kalender sehen
  • Darf Anlässe buchen
  • Darf Anlässe mit Reservationen buchen
  • Darf Anlässe mit Teilnehmernotifikationen editieren
  • Darf Teilnehmer editieren
  • Darf VideoConferencing Buchen sehen

Vorgehen / Beschrieb

Navigation zu >Kalender>Video Konferenz oder >Buchen>Video Konferenz

 

  1. Auswahl des Host-Standortes (Standardstandort)
  2. Auswahl der Child-Standortes. Es können mehrere hinzugefügt werden.
  3. Mittels Klick auf „Anzeigen“, werden dir verfügbaren Ressourcen angezeigt
  4. Bemerkung: Nur VC Räume werden angezeigt. Ein VC Raum wird über das zugeordnete VC Equipment identifiziert.
  5. Mittels klick auf das entsprechende Symbol können Teilnehmer erfasst werden.
  6. Die Auswahlsektion hilft dabei einen geeigneten Zeitabschnitt zu finden und die Buchung anzulegen.
  7. Die für die VC-Buchung vorgesehenen Räume werden mittels Checkbox ausgewählt.
  8. Weiter zu Stammdaten erfassen. (Dieser Button wird aktiv, sobald mindestens 2 geeignete Räume gewählt wurden)

Alle Informationen werden auf die VC-Detailbuchungsmaske übernommen:
Folgende Parameter können angegeben werden:
  1. Titel*
  2. Anzahl Personen
  3. Kommentar
  4. Organisator
  5. Kostenträger
  6. Kassifikationen

*: Wird der Titel für die Buchung des Hauptstandortes angegeben, wird dieser Wert auf die Child-Standorte übernommen.


7. VC Buchen, löst den Buchungsprozess aus und übermittelt die Daten an TMS. Klick auf den Button "Abbrechen" gelangt man auf den vorangegangenen Bildschirm zurück.



Der VC-Anlass ist erstellt. Standard Anlass Funktionalität stehen zur Verfügung. Der Anlass wird als VC Anlass qualifiziert (1).

 

Catering & Services sind auf einem VC Anlass nicht verfügbar.
