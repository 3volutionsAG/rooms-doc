Knowledge Base (Deutsch) : Buchung stornieren (annullieren)

Created by Former user, last modified by Former user (Deleted) on Feb 16, 2015

Artikel Nummer

Gültig ab Release

4.4

Verwandte Seiten

 

Einleitung

Ziel des Stornierens (Annullierens) ist es, eine Buchung zu stornieren (annullieren), so dass sie für den Benutzer unsichtbar bestehen bleibt (vergl. auch Buchung löschen).

Vorbedingungen

Benutzer: Benötigt Rechte um das gewählte Menü zu sehen und mindestens Leserechte für die darin verfügbaren Ressourcen.

Administratoren: Finden einen detaillierten Beschrieb im KB Artikel Rechte.

Vorgehen

In den drei Menüs PLAN, KALENDER und LISTEN ist es möglich, eine Buchung zu stornieren (annullieren).

 

Stornieren (Annullieren) im Menü PLAN

  • Über den Standortbaum oder die Plansymbole zur gewünschten Buchung navigieren.
  • Klick auf das rote Symbol.
  • Im anschliessend erscheinenden Popup Klick auf Button Stornieren (Annullieren).

  • Es erscheint ein Popup, welches auf die entstehenden Kosten bei einer Stornierung (Annullierung) hinweist und nachfragt ob man die Buchung trotzdem stornieren (annullieren) möchte. Ebenfalls besteht die Möglichkeit die Checkbox "Annullationsmails versenden" zu deselektieren, damit der Versand von Annullationsmails unterdrückt werden kann.

  • Bestätigung durch Klick auf Button Annullieren.

 

  • Oben rechts erscheint eine Hinweisbox, welche das erfolgreiche Stornieren (Annullieren) bestätigt.




Stornieren (Annullieren) im Menü KALENDER

  • In der Kalenderansicht zur gewünschten Buchung navigieren.
  • Klick auf das „+“-Symbol.
  • Im anschliessend erscheinenden Popup Klick auf Button Stornieren (Annullieren).


  • Es erscheint ein Popup, welches nachfragt ob man die Buchung wirklich stornieren (annullieren) möchte.

  • Bestätigung durch Klick auf Button OK.

  • Oben rechts erscheint eine Hinweisbox, welche das erfolgreiche Stornieren (Annullieren) bestätigt.

 

 

 

Stornieren (Annullieren) im Menü LISTEN

  • In der Listenansicht zur gewünschten Buchung navigieren.
  • Wenn in der Spaltenreihenfolge so eingestellt, erscheint die Spalte "Stornieren (Annullieren)" mit dem Icon Stornieren (Annullieren).
  • Klick auf das Icon Stornieren (Annullieren).

  • Es erscheint ein Popup, welches nachfragt ob man die Buchung wirklich stornieren (annullieren) möchte.

  • Bestätigung durch Klick auf Button OK.

  • Oben rechts erscheint eine Hinweisbox, welche das erfolgreiche Stornieren (Annullieren) bestätigt.
