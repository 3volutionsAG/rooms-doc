Knowledge Base (Deutsch) : Buchung terminieren

Einleitung

Ziel der Terminierung ist es, eine laufende Buchung terminieren zu können.

Vorbedingungen

Benutzer: Benötigt Rechte um das gewählte Menü zu sehen und mindestens Leserechte für die darin verfügbaren Ressourcen.

Administratoren: Finden einen detaillierten Beschrieb im KB Artikel Rechte.

Vorgehen

In den drei Menüs PLAN, KALENDER und LISTEN ist es möglich, eine Buchung zu terminieren.

 

Terminieren im Menü PLAN

  • Über den Standortbaum oder die Plansymbole zur gewünschten Buchung navigieren.
  • Klick auf das rote Symbol.
  • Im anschliessend erscheinenden Popup Klick auf Button Terminieren.

  • Oben rechts erscheint eine Hinweisbox, welche das erfolgreiche Terminieren bestätigt.



Terminieren im Menü KALENDER

  • In der Kalenderansicht zur gewünschten Buchung navigieren.
  • Klick auf das „+“-Symbol.
  • Im anschliessend erscheinenden Popup gewünschten Zeitpunkt (Datum & Uhrzeit) einstellen und Klick auf Button Terminieren.

  • Oben rechts erscheint eine Hinweisbox, welche das erfolgreiche Terminieren bestätigt.

 

 

Terminieren im Menü LISTEN

  • In der Listenansicht zur gewünschten Buchung navigieren.
  • Wenn in der Spaltenreihenfolge so eingestellt, erscheint die Spalte "Terminieren" mit dem Icon Terminieren.
  • Klick auf das Icon Terminieren.

  • Es erscheint ein Popup, welches nachfragt ob man die Buchung wirklich terminieren (abbrechen) möchte.

  • Bestätigung durch Klick auf Button OK.

  • Oben rechts erscheint eine Hinweisbox, welche das erfolgreiche Terminieren bestätigt.
