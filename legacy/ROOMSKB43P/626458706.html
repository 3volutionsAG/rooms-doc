Knowledge Base (Deutsch) : Der Experten Modus (Menü Reservieren)

Gültig ab Release

Verwandte Seiten

 

Einleitung

Im Experten Modus (Menü RESERVIEREN) kann der Benutzer nach gewünschten Ressourcen suchen und diese buchen. Mit der entsprechenden Einschränkung auf die Ressourcenart, deren Standort, den Zeitraum oder die Bezeichnung, können sie zielgenau gebucht werden. Zusätzlich stehen kundenspezifische Filter- und Suchkriterien zur Verfügung.

Vorbedingungen

Benutzer: Benötigt Rechte um den Experten Modus (Menü RESERVIEREN) zu sehen und mindestens Leserechte für die verfügbaren Ressourcen.

Administratoren: Finden einen detaillierten Beschrieb im KB Artikel Rechte.

Beschrieb

 

Das Fenster für die Suche nach Kriterien ist in zwei Bereiche eingeteilt. Der linke Bereich wird vom Sidepanel besetzt. Hier können die Kriterien eingegeben und die Suche dadurch erweitert oder eingegrenzt werden. Die Resultate werden im rechten Bereich in Listenform dargestellt.

 

Sidepanel im Experten Modus (Menü RESERVIEREN)

Das Sidepanel des Menüs RESERVIEREN bietet diverse Filtermöglichkeiten.

 

Gespeicherte Listen (_links)

Eingetragene Suchkriterien können für eine Wiederverwendung gespeichert werden.

 

Ressourcenart

Die Ressourcenart bestimmt, welche Ressourcen angezeigt werden sollen. Standardmässig steht in diesem Feld immer der Ressourcentyp "Raum". Mittels des Dropdown Buttons kann die Ressourcenart ausgewählt werden. Es sind nur Ressourcenarten sichtbar, auf die man lizenziert ist.

 

Datum und Zeit Picker 

Hier kann der Zeitraum, für welchen eine Ressource gesucht wird, eingestellt werden.

 

Standort

Mittels des Standorts kann eingeschränkt werden, wo sich die Ressource geografisch befinden soll. Standardmässig wird hier der vom Benutzer eingestellte Standard-Standort angezeigt. Falls in diesem Feld nichts steht, wird global über alle Standorte gesucht. Durch Klick auf das Plus-Symbol (+) kann der Standortbaum angezeigt und der Standort ausgewählt werden. Falls der Standort bekannt ist, kann die Standortbezeichnung im Textfeld eingegeben werden. ROOMS schlägt dann direkt Standorte vor, welche auf diese Eingabe passen könnten.





Hinweis

Der Link auf die Planansicht (Button Weltkugel) steht dem Benutzer nur dann zur Verfügung, wenn ein Plan für diesen Standort vorhanden ist und der Benutzer über die notwendigen Rechte verfügt.

 

Suchen über mehrere Standorte

Mit Hilfe von mehreren Standortpickern ist es möglich mehrere Standorte anzuzeigen und Schnellbuchungen darauf zu tätigen. Der Standortpicker kann beliebig vervielfacht werden. Durch Eingabe eines Standortes wird automatisch ein weiterer Standortpicker eingeblendet. Wird im zweiten Standortpicker auch ein Standort erfasst, erscheint ein Dritter, usw.

 

 

Erweiterte Kriterien

Ressourcen-ID/Bezeichung

Durch Eingabe der Ressourcen ID (falls vorhanden und bekannt) oder der Bezeichnung der Ressource wie z.B. grosses Auditorium, kann die Suche weiter eingegrenzt werden.

 

Kapazität (Anz. Personen)

Dieses Eingabefeld erscheint nur, wenn oben die Ressourcenart "Raum" ausgewählt wurde. Hier kann die gewünschte Teilnehmerzahl eingegeben werden (als Voreinstellung erscheint hier die Standardkapazität, welche im persönlichen Profil des Benutzers erfasst wurde).

 

Ist Öffentlich

Dieses Eingabefeld erscheint nur, wenn oben die Ressourcenart "Raum" ausgewählt wurde. Mittels des Dropdown Buttons kann ausgewählt werden ob es sich beim gesuchten Raum um einen internen oder einen öffentlichen handeln soll. Spielt dies keine Rolle, kommt die Rubrik "Alle" zum Zug.

 

Dienstleistungen

Mittels Plus-Symbol (+) werden alle zur Verfügung stehenden Dienstleistungen für den/die eingetragenen Standort/e angezeigt. Falls die Dienstleistung bekannt ist, kann die Bezeichnung im Textfeld eingegeben werden. ROOMS schlägt dann direkt Dienstleistungen vor, welche auf diese Eingabe passen könnten. Legt man anschliessend die Menge fest, werden sie im Warenkorb zusammengefasst und in die Suche eingebunden. Mit einem Klick auf das Icon "Notiz an Dienstleister" kann eine kurze Nachricht an den Dienstleister erfasst werden. Wird aus der im Suchresultat angezeigten Ressource eine Reservation erzeugt, werden die vorher selektierten Dienstleistungen direkt übernommen. Dieses Suchkriterium ist nur im Experten Modus (Menü RESERVIEREN) sichtbar.

 

 

Gliederungen

Mit Hilfe von Gliederungen (Zusammenfassung bestimmter Attribute einer Ressource wie z.B. Raumtypen Sitzungszimmer, Schulungsraum, Kundenraum etc.) kann eine Eingrenzung der Suche zum Beispiel nach gewünschtem Equipment erfolgen. Die Auswahl wird mit Klick auf das Plus-Symbol (+) getätigt. Die Inhalte sind kundenspezifisch und können im Bereich der Gliederungen in der Webapplikation durch berechtigte Personen verändert werden. Sind keine Gliederungen erfasst, werden keine Gliederungen eingeblendet.

Spezialfall: Mobiles Equipment

Der häufigste Fall bei der Suche mit einem aggregierten Gliederungstyp ist die Suche nach Equipment (ich suche einen Raum mit Beamer). Dabei werden sowohl die Räume mit fix installiertem Equipment und Räume unter Berücksichtigung der Buchung von mobilem Equipment gesucht und vorgeschlagen.

Klassifikationen

Mit Hilfe von Klassifikationen (Attribute zur spezifischen Kennzeichnung von Buchungen wie z.B. internes Meeting, Beratungsgespräch etc.) kann eine Eingrenzung der Suche nach Ressourcen, denen die gewünschten Klassifikationen zugrunde liegen, erfolgen. Die Auswahl wird mit Klick auf das Plus-Symbol (+) vorgenommen. Die Inhalte sind kundenspezifisch und können im Bereich der Klassifikation durch berechtigte Personen verändert werden. Sind keine Klassifikationen erfasst, werden keine Klassifikationen eingeblendet.

 

Wenn die Filterkriterien wunschgemäss eingestellt sind, können mit dem Button Finden die Suchresultate im rechten Fensterbereich aktualisiert werden.

 

 

Darstellung der Suchresultate

Die Resultate werden im rechten Bereich in Listenform dargestellt, einer sogenannten Smartlist.

 

Vor jedem Resultat befinden sich eine Checkbox (damit können mehrere Räume zusammen oder einzeln ausgewählt werden) und das Icon "Schnellreservation erstellen" (wird nur angezeigt, wenn der Benutzer die nötigen Rechte besitzt). Sie sind wichtig für die Schnellbuchung im Experten Modus.

 

Werden mehrere Ressourcen über die Checkboxen selektiert, können diese via Button Anlass mit Schnellreservationen zusammengefasst als Anlass gebucht werden.

 

Der Button Detailbuchung (ehem. Detailreservation) führt den Benutzer weiter zur Maske der Detailbuchung.
