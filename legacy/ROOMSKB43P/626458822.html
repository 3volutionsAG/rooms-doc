Knowledge Base (Deutsch) : Anlasstypen erfassen & bearbeiten

Created by Former user, last modified on Jul 05, 2013

Artikel Nummer

Verwandte Seiten

Einleitung

Zur Nutzung der Funktion Anlässe in ROOMS zu verwalten werden sogenannte Anlasstypen zu Grunde gelegt. Damit werden diverse Formen von Anlässen wie z.B. Kundenworkshops etc. unterschieden. Zusätzlich dienen die Anlasstypen zur Erfassung der Reservationsanfragen (Platzhalter Reservationsanfrage).

Vorbedingungen

Keine

Benötigte Rechte

  • Darf Anlasstypen verwalten
  • Darf Sektion Einstellungen nutzen
  • Darf Reservationsanfragen administrieren

Wechsel auf das Register "Reservationsanfrage" öffnet die Eingabemaske des Anlasstyps. Die Felder und deren Funktionen sind der unten stehenden Tabelle zu entnehmen.

Feld

Funktion

Standardeinstellung

Eigenschaften

Regeln

Gruppe für  die Notifikation

Angabe der Benutzergruppe, die notifiziert wird sobald im Buchungsprozess die Reservationsanfrage versendet wird.

 

Damit die E-Mailnotifikation verschickt wird muss auf der entsprechenden Benutzergruppe die Notifikationsvorlage (siehe Notifikationen Benutzergruppen bezogen erfassen und bearbeiten) hinterlegt sein.

Detail der Reservationsanfrage

Freie HTML und JavaScript Eingabe für Gestaltung oder Platzierung der Controls aus der vorhanden Liste. Diese Controls werden später auf der Anfrage zur Verfügung stehen z.B. Titel, Standort etc. Für die Liste der Controls inkl. Beschreibung siehe Platzhalter Reservationsanfrage

 



= Pflichtfelder
= Daten mehrsprachig erfassen

Im Register "Benutzergruppe" werden die Benutzergruppen ausgewählt, die im späteren Buchungsprozess den Anlasstyp und somit auch die erfasste Reservationsanfrage nutzen dürfen. Selektieren einer oder mehrerer Checkboxen und bestätigen mit dem Button Gruppe hinzufügen listet die Gruppe in Smartlist auf.


Der Vorgang mit dem Button Speichern abgeschlossen und der neue Anlasstyp wird aufgelistet.

Anlasstyp bearbeiten

Navigation zu >Einstellungen>Stammdaten für Anlässe>Anlasstypen
In der Smartlist werden alle bereits erfassten Anlasstypen angezeigt. Mittels Klick auf die Bezeichnung wird der Anlasstyp im Anzeigemodus (Viewmode) dargestellt.



Mittels Button Bearbeiten oder über das Editieren-Symbol wechselt das System in den Bearbeiten Modus. In allen, bereits vorgängig in diesem Artikel beschriebenen Registern, können Anpassungen vorgenommen werden. Die Änderungen werden mit dem Button Speichern übernommen.

Anlasstyp löschen

Navigation zu >Einstellungen>Stammdaten für Anlässe>Anlasstypen
In der Smartlist werden alle bereits erfassten Anlasstypen angezeigt. Durch Selektieren der Checkbox (Mehrfachselektion möglich) und Bestätigung über den Button Entfernen erscheint eine Sicherheitsabfrage, ob der Anlasstyp wirklich entfernt werden soll. Quittieren dieser Meldung entfernt den Anlasstyp. Die Löschung ist nur möglich, wenn keine Anlässe bzw. Buchungen mit diesem Anlasstyp bestehen. Im Actionpanel wird die Anzahl der entfernten Einträge ausgegeben.
