Vorgehen / Beschrieb

Benutzer navigiert auf >Einstellungen >System >Globale Parameter

Eingabe der Suchkriterien und Klick auf Button Finden

In einer Smartlist werden alle verfügbaren Parameter angezeigt (Spalten: Bezeichnung, Wert, Beschreibung)

Die verfügbaren Werte sind in KB Konfigurationsparameter detailliert beschrieben.

Klick auf Button Bearbeiten erlaubt die Änderung der Einstellung:
