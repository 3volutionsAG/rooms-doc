Einleitung

Farben können zur Kennzeichnung von Ressourcen, Klassifikationen u.v.m über einen Farbpicker zugewiesen werden. Diese Farbe wird dann in sämtlichen Ansichten angezeigt und dient als Orientierungshilfe.

Vorbedingungen

Keine

Vorgehen / Beschrieb

Der Farbpicker wird anhand des Beispiels mit Farben für Ressourcen aufgezeigt. Dazu navigiert der Nutzer nach Einstellungen>Ressourcen>Ressourcen verwalten.

Klick auf den Button Anzeigen öffnet die Ressourcenübersicht. Mittels Button Erstellen öffnet die Eingabemaske einer neuen Ressource. Wechsel in das Register "Spezifische Daten". Entweder wird der ein HEX Wert in das Feld "Farbe" eingeben oder über den Picker (Symbol rechts neben dem Feld) wird eine entsprechende Farbe ausgewählt.


Legende
1 = Gewählte Farbe als Hex Wert
2 = Gewählte Farbe als Vorschau
3 = Farbwahl anzeigen
4 = Farbselektor
5 = Übernimmt gewählte Farbe

Die Visualisierung der vergebenen Farben kann im Menü PLAN, im Menü KALENDER und/oder unter Menü LISTEN erfolgen.

Benötigte Rechte

Rechte zur Stammdatenverwaltung (abhängig davon, in welchem Kontext eine Farbwahl erfolgt)
