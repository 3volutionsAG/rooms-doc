Einleitung

Es kann eine Standard Person erstellt werden, bei welcher Filtereinstellungen definiert wurden.

Diese werden dann bei Erstanmeldung einer neuen Person auf diese übertragen.

Vorbedingungen

Eine (Standard) Person wurde erstellt und Filtereinstellungen auf dieser Person definiert

Benötigte Rechte

keine

Vorgehen / Beschrieb

  • Navigation auf die globalen Parametern unter Einstellungen (siehe KB Globale System Parameter konfigurieren)
  • Eingabe einer Personen ID in das Feld "Default-Filtersetting" (siehe KB Konfigurationsparameter)
