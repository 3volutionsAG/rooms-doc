Use Cases : 215.001 Reservationsanfrage erfassen

Ziel & Beschreibung

Eine neue Reservationsanfrage erstellen.

Rechte

  • Darf Reservationsanfragen einreichen

Bedingungen

Vorgängig müssen Anlass Status und Anlass Typen erfasst worden sein (siehe UC 120.001 Anlasstyp verwalten und 120.002 Anlassstatus verwalten)

Hauptszenario in Schritten

  1. Benutzer navigiert auf >Reservieren >Reservationsanfrage
  2. Nutzer wählt Standort und gewünschtes Formular
  3. Reservationsanfrage Formular wird angezeigt und Daten werden erfasst - Anfrage wird verschickt

Beziehungen zu UC

215.002 Bestehende Reservationsanfrage suchen und bearbeiten
120.001 Anlasstyp verwalten


Legende

  • Rote Schrift = MUSS Felder
  • Grüne Schrift = wird in einem Folgerelease umgesetzt

Use Case Details

Schritt 1 - Benutzer navigiert auf >Reservieren >Reservationsanfrage

Controls

Sidepanel

Navigation öffnet sich entsprechend

Regeln

Folgendes Recht muss implementiert werden:

  • Darf Reservationsanfragen einreichen

Bilder

Schritt 2 - Nutzer wählt Standort und gewünschtes Formular

Controls

Subpanel „Reservationsanfrage“

  • Standortpicker: Standort
  • Dropdown: Reservationsanfrage

Button Bereich

  • Button: Erstellen

Regeln

Standort

  • Selektion eines Standorts mittels Standortpicker
  •  

Reservationsanfrage

  • wird nur angezeigt nach Wahl eines Standorts
  • Inhalt sind alle verfügbaren Request Formulare (Anlasstypen) des aktuellen und allen hierarchisch höhren Standorten

Button Bereich

  • siehe Schritt 3

Bilder

Schritt 3 - Reservationsanfrage Formular wird angezeigt und Daten werden erfasst - Anfrage wird verschickt

Controls

Inhaltsbereich

  • HTML Formular mit allen Controls wird gemäss Erfassung gerendert und visualisiert (siehe dazu UC 120.001 Anlasstyp verwalten)

Buttons

  • Anfrage senden
  • Abbrechen

Regeln

Anfrage senden

  • Das „Verschicken“ der Anfrage wird im Actionpanel bestätigt.
  • Eine E-Mail wird an die Benutzer der  Benutzergruppe, welche dem Anlasstyp hinterlegt worden ist verschickt, sofern das Template 11 - ReservationsanfrageNeu (siehe Notifikationsmatrix)  hinterlegt worden ist
  • Alle Daten werden gespeichert
  • Navigation auf
    • Sidepanel bleibt bestehen
    • Inhaltsbereich wechselt auf die Liste der erfassten Anfragen (siehe UC 215.002 Bestehende Reservationsanfrage suchen und bearbeiten)

Abbrechen

  • Sidepanel bleibt bestehen
  • Inhaltsbereich wechselt auf die Liste der erfassten Anfragen (siehe UC 215.002 Bestehende Reservationsanfrage suchen und bearbeiten)
