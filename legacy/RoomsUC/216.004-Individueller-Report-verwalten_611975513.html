Use Cases : 216.004 Individueller Report verwalten

Created by Former user, last modified by Former user (Deleted) on Oct 13, 2014

216.005

Owner

Tim Bänziger

Rev-Number

Priorität

Ziel & Beschreibung

Individuellen Reports (Word) können in GARAIO Rooms geladen
und dort verwaltet werden .

Rechte

  • Darf individuelle Dokumente verwalten

Bedingungen

  • Report Schema  (XML) wurde durch die Entwicklung erstellt und implementiert

Hauptszenario in Schritten

  1. Navigation auf >Einstellungen >Erweiterte Basisdaten für Ressourcen >Dokumente
  2. Benutzer gibt Suchkriterien ein (optional) und klickt auf Find
  3. Suchresultat über alle gefundenen Dokumente gemäss Suchkriterien wird angezeigt - ein oder mehrere Dokumente können gelöscht werden
  4. Eine Dokument kann im View Modus angezeigt werden
  5. Ein Dokument kann editiert oder neu erstellt werden

Beziehungen zu UC


Legende

  • Rote Schrift = MUSS Felder
  • Grüne Schrift = wird in einem Folgerelease umgesetzt

Schritt 2 - Benutzer gibt Suchkriterien ein (optional) und klickt auf Finden

Controls

  • Bezeichnung
  • Button „Finden“

Regeln

Schritt 3 - Suchresultat über alle gefundenen Dokumente gemäss Suchkriterien wird angezeigt – ein oder mehrere Dokumente können gelöscht werden

Controls

Smartlist "Vorlagen"

Standard Spalten:

  • Checkbox zur Mehrfachselektion
  • Bezeichnung
  • Edit Icon

Optionale Spalten:

  • Sprache

Button Bereich

  • Button „Entfernen“
  • Button „Erstellen“

Regeln

Bezeichnung
Die Vorlagenbezeichnung wird in der Sprache des angemeldeten Benutzers angezeigt

Erstellen (siehe Schritt 5)

  • Funktionsrecht „Darf Benutzergruppen administrieren“ muss vorhanden sein

Entfernen

  • Mehrfachselektion erlaubt
  • Sicherheitsabfrage
  • Eintrag wird gelöscht und Smartlist aktualisiert
  • Bestätigungsmeldung im Actionpanel, inkl. Anzahl der gelöschten Dokumente

Schritt 5 – Ein Dokument kann editiert oder neu erstellt werden

Controls

Tab „Vorlage“

  • Feld: Bezeichnung
  • Dropdown: Sprache
  • Dropdown: Typ
  • Dropdown: Export Format
  • Dropdown: Selektion für Download
  • Dropdown: Selektion für E-Mail
  • Multiselektor: Publizieren in
  • Vorlage (mit Browse + Upload Button)
  • Link: Erfasste Vorlage

Button Bereich

  • Button „Speichern“
  • Button „Abbrechen“

Regeln

Bezeichnung

  • wird in der Benutzersprache des angemeldeten Benutzers geladen
  • Vorlage muss in allen verfügbaren Sprachen bezeichnet werden

Sprache

  • Die aktivierten Sprachen werden zur Auswahl angezeigt.
  • Die Vorlage kann einer Sprache zugeordnet werden - heisst, pro Sprache können unterschiedliche Vorlagen verwendet werden.

Typ

  • Ein Datenschema muss gewählt werden.
  • Die verfügbaren Typen werden durch die Entwicklung erstellt, implementiert und zur Verfügung gestellt

Publizieren in

  • Mehrfachselektion möglich minimal ein Publikationsort muss selektiert werden
  • Folgende Werte sind möglich:
    • Reports
    • Summary Reservation
    • View Anlass

Export Format

  • Auswahlmöglichkeit für das Export Format:
    • Docx (Word)
    • PDF

Selektion für Download

  • Folgende Optionen stehen zur Verfügung:
    • Keine => Checkbox im Reservationsprozess ist nicht selektiert
    • Vorselektiert => Checkbox im Reservationsprozess ist vorselektiert
    • Pflicht => Checkbox im Reservationsprozess ist vorselektiert und nicht änderbar
  • Download steht immer zur Verfügung

Selektion für E-Mail

  • Folgende Optionen stehen zur Verfügung:
    • Keine => Checkbox im Reservationsprozess ist nicht selektiert
    • Vorselektiert => Checkbox im Reservationsprozess ist vorselektiert
    • Pflicht => Checkbox im Reservationsprozess ist vorselektiert und nicht änderbar
  • E-Mail steht nur bei Reservationsabschluss zur Verfügung (nicht Reports und nicht View Mode auf Reservation oder Anlass)

Vorlage hinzufügen

  • Eine Vorlage kann selektiert (Browse) werden und mittels Upload auf den Server geladen werden.
  • nur Word Dokumente zulässig (*.docx)

Link erfasste Vorlage

  1. Ist bereits eine Vorlage vorhanden, wird diese angezeigt (File Namen)
  2. Die erfasste Vorlage hat einen Hyperlink und kann mittels Klick wieder auf den Client (zur Bearbeitung) geladen werden.

Speichern

  • Validierung der MUSS Felder
  • Bestätigungsmeldung im Actionpanel
  • Zurück zum Ursprungspunkt

Abbrechen

  • wechselt zurück zum Ursprungspunkt
