Ziel & Beschreibung

  1. Suchen und Finden eines bestehenden Anlass
  2. Bearbeitung des Anlass

Rechte

  • Darf Sektion Listen nutzen
  • Darf Anlässe buchen

Bedingungen

Anlass wurde vorgängig erstellt

Hauptszenario in Schritten

Erstellung und Editieren eines Anlass

  1. Nutzer navigiert auf >Listen >Listen Anlässe
  2. Nutzer gibt die Suchkriterien ein (optional) und klickt „Suchen"
  3. System liefert eine Liste der erfassten Anlässe

Beziehungen von/zu UC

 

 


Legende

  • Rote Schrift = MUSS Felder

Use Case Details

Schritt 1 - Nutzer navigiert auf >Listen >Listen Anlässe

Controls

Menüpunkt

  •  >Listen >Listen Anlässe 

Regeln

Generell

  • nur verfügbar mit Recht: Darf Anlässe buchen
 

Schritt 2 - Nutzer kann Suchkriterien eingeben (optional) und klickt „Suchen“

Controls im Sidepanel

Generell

  • Favoriten Liste
  • Datumpicker: Datum/Zeit von/bis
  • Standortpicker: Standort

Anlass Spezifische Kriterien

  • Feld: Anlass Nr
  • Feld: Anlass Bezeichnung
  • Dropdown: Anlasstyp
  • Dropdown: VC-Anlass

Personenspezifische Kriterien

  • Intelibox: Verantwortlicher
  • Intelibox: Ersteller
  • Intelibox: Organisator

Button Bereich

  • Finden
  • Zurücksetzen

Regeln

Allgemeine Regeln:

Control

Regel

Favortien Liste

Favoritenliste (siehe auch UC 230.001 Liste erstellen und (als Favorit) speichern )

  • Wird eine Liste als Favorit definiert wird diese standardmässig angezeigt.
  • Ansonsten ist die Liste leer.
  • Nach Wechsel eines Eintrages, werden Sidepanel und Inhaltsbereich automatisch neu geladen
    (Achtung: Bevor Liste gespeichert wird, muss "Finden" Button geklickt werden, um alle Kriterien zu speichdern)

Anlass Nr.

Mittels Anlass-Nr. kann ein spezifischer Anlass durch seine eindeutige Nummer gefunden werden.

Anlass Bezeichnung

Mittels Anlass-Bezeichnung kann ein spezifischer Anlass durch seine eindeutige Bezeichnung gefunden werden.

Datum/Zeit Picker (von bis)

Zeitpunkt VON resp. AB wann Anlässe berücksichtigt werden sollen UND Zeitpunkt BIS wann Anlässe berücksichtigt werden sollen

Anlasstyp

Nach welchem Typ von Anlässen soll gefiltert werden (z.B. Direktionskonferenz) - siehe UC 120.001 Anlasstyp verwalten)

Standort (Anlass)

Durch Eingabe des Standorts, werden als Suchresultat nur die dem eingegebenen Standort zugeteilten Anlässe aufgeführt

Verantwortlicher

Verantwortlicher (Owner) des Anlasses

Ersteller

Ersteller des Anlasses

Organisator

Organisator des Anlasses

VC-Anlass

  • zeigt alle Reservationen eines Anlasses mit/ohne VC Buchung* nur sichtbar bei Ressourcenart "Raum"
  • nur sichtbar bei TMS aktiviert (siehe Konfigurationsparameter / Configuration parameter 
  • Dropdown mit den Werten
    • alle
    • nur Anlässe mit VC
    • nur Anlässe ohne VC
  • Standardwert: alle

Finden

  • sucht in der Sprache des angemeldeten Benutzers
  • alle Felder (Werte) stehen in einer AND Verknüpfung
  • Bei Feldern ist Wildcard Suche (*) möglich

Bilder

 

Schritt 3 - System liefert eine Liste der erfassten Anlässe

Controls

Smartlist

Standard Spalten

  • Anlassbezeichnung (mit Link in View Mode)
  • Edit Icon
  • Anlass Nummer
  • Löschen Icon
  • Verantwortlicher
  • Ersteller
  • Organisator
  • Beginn (Format: dd.mm.yyyy HH:MM)
  • Ende (Format: dd.mm.yyyy HH:MM)
  • Anlasstyp

Optinale Spalten

  • Status
  • Anlass Standort Hierarchisch
  • Catering & Services
  • Teilnehmer
  • Kostenstellen Anlass
  • Kostenstellen Nummer Organisator (aus UC 102.003 Personen_OE verwalten)
  • Kostenstellen Nummer Verantwortlicher (aus UC 102.003 Personen_OE verwalten)
  • Kostenstellen Bezeichnung Organisator (aus UC 102.003 Personen_OE verwalten)
  • Kostenstellen Bezeichnung Verantworlticher (aus UC 102.003 Personen_OE verwalten)
  • Login Organisator (Feld "Logon Name" aus UC 101.003 Logon verwalten)
  • Login Verantwortlicher (Feld "Logon Name" aus UC 101.003 Logon verwalten)
  • Firma Nr. Organisator (siehe UC 102.001 Abteilung (Firmen) verwalten)
  • Firma Nr. Verantwortlicher (siehe UC 102.001 Abteilung (Firmen) verwalten)
  • Firma Bezeichnung Organisator (siehe UC 102.001 Abteilung (Firmen) verwalten)
  • Firma Bezeichnung Verantwortlicher (siehe UC 102.001 Abteilung (Firmen) verwalten)
  • Kosten

Regeln

Generell

  • alle Werte werden in der Sprache des angemeldeten Benutzers ausegeben

View Link

  • siehe Schritt 5 in UC 211.001 Anlass erstellen

Edit Icon

  • siehe Schritt 2 in UC 211.001 Anlass erstellen 

Löschen Icon

  • erscheint nur, wenn noch keine Reservationen zu Anlass hinzugefügt
  • Löscht den Anlass
  • Bestätigungsmeldung im Actionpanel
  • Refresh der Smartlist

Catering & Services

  • Nur Dienstleistungen des Anlasses
  • Formatierung Einzel Dienstleistung
    • Dienstleister:
    • Anzahl x Angebot (Lieferzeit - Lieferort)
  • Formatierung: Paket
    • Dienstleister
    • Anzahl x Paket Name (Lieferzeit - Lieferort): Paketoption 1, Paketoption 2, ...
  • Beispiel
    • Compas Group:
    • 2x Gipfeli (12:45 - \Schweiz\Zürich)
    • 3x Getränke-Paket (12:45 - \Schweiz\Zürich): Coca, Kafi

Teilnehmer

  • Anzeige Vorname & Name
  • für jeden Teilnehmer eine neue Zeile

Kostenstellen Anlass

  • pro Kostenträger eine neue Zeile

Kosten

  • Gesamtkosen des Anlass
