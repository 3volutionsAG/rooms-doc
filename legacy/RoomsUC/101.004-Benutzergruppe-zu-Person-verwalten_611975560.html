Use Cases : 101.004 Benutzergruppe zu Person verwalten

Created by Former user (Deleted), last modified on Oct 13, 2014

Ziel & Beschreibung

Zuweisen einer oder mehreren Benutzergruppen zu einer Person

Rechte

  • Darf Benutzergruppen verwalten

Bedingungen

Benutzergruppe muss existieren

Hauptszenario in Schritten

  1. Nutzer navigiert auf Tab „Gruppen"
  2. Nutzer kann eine oder mehrere Gruppen von Person entfernen
  3. Nutzer kann neue Gruppen zur Person hinzufügen

Beziehungen von/zu UC


Legende

  • Rote Schrift = MUSS Felder
  • Grüne Schrift = wird in einem Folgerelease umgesetzt

Schritt 2 - Nutzer kann eine oder mehrere Gruppen von Person entfernen

Controls

Smartlist

Standard Spalten

  • Checkbox zur Mehrfachselektion
  • Bezeichnung
  • Ist Berechtigungsgruppe
  • Ist Notifikationsgruppe
  • Ist Teilnehmergruppe
  • Benutzerkreis

Button Bereich

  • Button: Hinzufügen
  • Button: Entfernen

Regeln

Smartlist

  • Die Smartlist verfügt Suchfeld oberhalb der Bezeichnungsspalte

Entfernen

  1. Mehrfachselektion erlaubt
  2. Warnhinweis erscheint
  3. Verbindung von Person zur Benutzergruppe wird gelöscht
  4. Erfolgreiche Entfernung: Ausgabe einer Bestätigung im Action Panel, inkl. Anzahl der entfernten Benutzergruppen

Hinzufügen

  • Benutzergruppe kann zur Person hinzugefügt werden (siehe Schritt 3)

Schritt 3 - Nutzer kann neue Gruppen zur Person hinzufügen

Controls

Modales Popup - Smartlist

Default

  • Checkbox zur Mehrfachselektion
  • Bezeichnung
  • Ist Berechtigungsgruppe
  • Ist Notifikationsgruppe
  • Ist Teilnehmergruppe
  • Benutzerkreise

Button Bereich

  • Button: Hinzufügen
  • Button: Abbrechen

Regeln

Generell

  • Filter: es werden ausschliesslich Benutzergruppen angezeigt, bei welchen der Benutzer noch nicht zugefügt worden ist
  • Ist Teilnehmergruppe zur Zeit nicht verwendet (keine Funktion)

Benutzerkreise

  • Benutzer in Benutzergruppen mit dem Flag "Benutzerkreis" aktiviert, können im Buchungsprozess nur Buchungen für Benutzer dieser Gruppe erstellen (UC 201.000 Ressourcen reservieren)

Person Hinzufügen

  • Mehrfachselektion erlaubt
  • Popup wird geschlossen
  • Smartlist wird aktualisiert, um die zugefügten Benutzergruppen
  • Erfolg: Ausgabe einer Bestätigung im Action Panel, inkl. Anzahl der hinzugefügten Benutzergruppen
  • Nicht erfolgreich: Fehlermeldung im Action Panel
