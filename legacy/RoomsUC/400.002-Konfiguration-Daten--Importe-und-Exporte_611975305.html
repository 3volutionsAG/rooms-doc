Ziel & Beschreibung

Der generische Service für Import und Export von verscheidenen Datenstämmen kann konfiguriert werden
Erlaubt Importe/Exporte aus/in CSV, XML, TXT, MS SQL, AD

Rechte

  • Darf Konfiguration Datenimport verwalten

Bedingungen

–

Hauptszenario in Schritten

  1. Nutzer navigiert auf >Einstellungen >System >Daten-Importe & -Exporte
  2. Nutzer gibt Suchkriterien ein (optional) und klickt Button „Finden“
  3. Ein Suchresultat über Konfigurationseinträge wird angezeigt – ein oder mehrere Konfigurationseinträge können gelöscht werden
  4. Nutzer kann einen Eintrag anklicken, um die Details anzuzeigen (View Modus)
  5. Nutzer editiert oder erstellt ein Konfigurationseintrag

Schritt 5 - Nutzer editiert oder erstellt ein Konfigurationseintrag

Controls

Sidepanel

  • Feld: Name
  • Datumpicker: Nächste Ausführung
  • Feld: Intervall in Minuten
  • Textarea: Konfiguration

Button Bereich

  • Button: Speichern
  • Button: Abbrechen

Regeln

Name

  • Freie Eingabe eines beliebigen Namens des Import/Export Jobs

Nächste Ausführung

  • zum Starten des Jobs muss eine nächste Ausführung angegeben werden
  • Standard = leer
  • Datum/Zeit können auch in Vergangenheit liegen => sofortige Ausführung
  • Datumpicker verfügt nur über Datum/Zeit (kein von/bis)

Intervall in Minuten

  • Standard = leer
  • Validierung auf positive Ganzzahlen
  • leer = einmalige Ausführung

Konfiguration

  • XML basierende Konfiguration des Import oder Exports
  • wird durch GARAIO Softwareentwickler definiert.

Speichern

  • Wechsel auf Listenansicht (siehe Schritt 3)
  • Bestätigungsmeldung im Actionpanel

Abbrechen

  • zurück zum Ursprungspunkt
