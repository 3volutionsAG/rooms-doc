Use Cases : 105.011 Bestuhlung bei Ressourcen verwalten

Ziel & Beschreibung

Der Nutzer kann Bestuhlungen zu einer Ressource erfassen und dabei die notwendige Konfiguration durchführen.
Dies wird im Tab „Bestuhlung“ innerhalb einer Ressource vorgenommen.

Rechte

  • Darf Ressourcetypen mit Bestuhlungen verwalten

Bedingungen

Es müssen die Schritte 1, 2, 3 und 5 von UC 105.001 Ressourcen verwalten ausgeführt werden

Hauptszenario in Schritten

  1. Navigation in den Bereich der Bestuhlungen (Tab Bestuhlung)
  2. Liste mit allen der Ressource zugeordneten Bestuhlungsvarianten wird angezeigt
  3. Bestuhlungsvarianten kann/können der Ressource hinzugefügt werden

Beziehungen zu UC

Controls

Tab "Bestuhlungen"

wird angezeigt

Regeln

  • Tab „Bestuhlung“ muss implementiert werden
  • Recht notwendig: Darf Ressourcetypen mit Bestuhlungen verwalten

Bilder

 

Schritt 2 - Liste mit allen der Ressource zugeordneten Bestuhlungsvarianten wird angezeigt – eine oder mehrere Bestuhlungen können entfernt werden

Controls

Inhaltsbereich Smartlist
Standard Spalten:

  • Checkbox: Selektionscheckbox
  • Radio: Standardbestuhlung
  • Bezeichnung

Optionale Spalten:

  • Bild-Icon oder Thumbnail
  • Max. Personenzahl
  • Min. Personenzahl
  • Vorlaufdauer
  • Nachlaufdauer
  • Dropdown: Währung + Feld: Kosten
  • Beginn (Datum/Zeit)

  • Ende (Datum/Zeit)

  • Annulierungskosten
  • Annulierung vor Beginn (in Minuten) 

Button Bereich

  • Button: „Hinzufügen“ (siehe Schritt 3)
  • Button: „Entfernen“

 

Regeln

Anmerkung:
Es gibt keinen Editiermodus. Eine Option muss entfernt werden und neu hinzugefügt, falls diese editiert werden soll

Smartlist
Jede Spalte verfügt über ein Suchfeld, sofern Funktion aktiviert (siehe 000.001 Globale Funktionen )

Bild Icon

  • Thumbnail der Bestuhlung (falls Bild erfasst wurde)
  • Klick auf Bild-Icon zeigt die Bestuhlung in einem neuen Fenster in Originalgrösse dar.

Standardbestuhlung

  • Es muss EINE Standardbestuhlung über die Radiobuttons selektiert werden.

Währung + Kosten

  • alle aktivierten Währungen
  • Standardwert = Währung der im Profil des angemeldeten gewählten Kultur
  • die Anzahl der aktiviert Währungen ist abhängig von der Anzahl der zur Verfügung gestellten Kulturen

 

Annulierungskosten

  • Kosten die bei Annulierung einer Buchung, auf einer Bestuhlung anfallen
  • Alle verfügbaren Währungen werden angezeigt
  • Eingabe eines positiven Betrags

 

Annulierung vor Beginn (in Minuten) 

  • Wieviel vor Buchungsstart werden die Annulierungskosten fällig (Beispiel: Eingabe von 120 bedeutet, dass bei einer Stornierung kleiner als 2 Stunden vor Buchungsbeginn die Kosten erhoben werden)
  • Eingabe von Minuten
  • Eingabe von postivien Ganzzahlen

 

Hinzufügen

  • öffnet ein modales Popup (siehe Schritt 3)

Button Entfernen

  • Es kann ein oder mehrere Bestuhlungen selektiert und mittels Klick auf „Entfernen“ entfernt werden:
  1. Sicherheitsabfrage
  2. Entfernt die Verlinkung zur Bestuhlungsoption und löscht die zusätzlichen Felder des Mappings (aus der Tabelle „RessourcenBestuhlung“)
  3. Bestätigungsmeldung über Anzahl der entfernten Bestuhlungen im Actionpanel

Bilder

 

Schritt 3 - Bestuhlung kann der Ressource hinzugefügt werden

Controls

Modales Popup - Smartlist

Standard Spalten

  • Checkbox: Selektionscheckbox
  • Bezeichnung (der Bestuhlung in der Benutzersprache)
  • Min. Personenzahl
  • Max. Personenzahl

Optionale Spalten

  • Vorlaufdauer
  • Nachlaufdauer
  • Bildvorschau

Allgemeiner (unterer) Bereich

  • Dropdown: Mehrwertsteuer
  • Dropdown: Währung
  • Feld: Kosten
  • Feld: Annullationskosten
  • Feld: Start Annullation (in Minuten vorher)
  • Datumpicker: Gültigkeitsdauer: (Beginn-Datum/Zeit bis Ende-Datum/Zeit)

Button Bereich

  • Button: Hinzufügen
  • Button: Abbrechen

Regeln

Generell

  1. Es werden alle verfügbaren Bestuhlungen angezeigt
  2. Es besteht die Filtermöglichkeit über die Spaltenfilter der Smartlist
  3. Eine oder mehrere Bestuhlungsoptionen können selektiert werden

Mehrwertsteuer

  • Dropdown mit allen aufgeschaltenen Mehrwertsteuersätzen
  • Standard: "Mehrwertsteuer wird nicht berücksichtigt"

Währung

  • alle aktivierten Währungen
  • Standardwert = Währung der im Profil des angemeldeten gewählten Kultur
  • die Anzahl der aktiviert Währungen ist abhängig von der Anzahl der zur Verfügung gestellten Kulturen

Kosten

  • Standardwert: leer (keine Kosten)

Annullationskosten

  • Standardwert = leer (keine Annullationskosten)
  • Wert > 0 entspricht dem Wert, welcher bei einer Annullation Pauschal erhoben wird

Start Annullation in Minuten vorher

  • Definiert ab wieviele Minuten vor Reservationsbegin Annullationskosten erhoben werden
  • 0 = Annullationskosten werden zur Laufzeit erhoben
  • Standardwert = leer (ausgeschalten)
  • Beispiel: Bei einem Wert von 60, wird bis 60 Minuten vor Reservationsstart KEINE Annullationskosten erhoben. Zwischen 60 Minuten und der Endzeit der Reservation wird der Pauschalbetrag, welcher unter "Annullationskosten" definiert wurde in Rechung gestellt.

Gültigkeitsdauer

  • Standardwert: leer (beliebige Gültigkeit)

Hinzufügen

  • Fügt die selektierten Bestuhlungen der Ressource zu
  • Popup schliesst automatisch
  • Smartlist "Bestuhlungen" wird aktualisiert
  • Bestätigungsmeldung im Actionpanel
