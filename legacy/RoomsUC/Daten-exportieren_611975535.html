Daten exportieren


In Releases 4.0 wird der Export aller Smartlists in Excel unterstützt. Dazu wird ein entsprechendes Icon und Text (siehe Abbildung) auf jeder Smartlist implementiert.

Folgende Regeln müssen beachtet werden:

  • Symbol wird nur angzeigt bei Funktionsrecht "Darf Daten exportieren"
  • Klick auf Symbol öffnet generiert den Export und öffnet diesen in einen neuen Fenster
  • Es werden alle Daten gemäss Suchkriterien angezeigt (also das Paging wird nicht berücksichtigt)
  • Es werden nur die Spalten angezeigt, welche im Grid auch aktiv sind
  • Spaltentitel werden mit Exportiert (in der Benutzersprache des Users)
  • Filename setzt sich zusammen aus „ROOMS-Data-Export_[DateTime].xls (wobei DateTime mit dem aktuellen Datum/Zeit) ersetzt wird
  • Header/Footer Informationen werden zusätzlich in die Datei geschrieben:
    • Titel (ideal ein Export Alias, welches pro List hinterlegt wird und über die Übersetzungen editiert werden kann)
    • Ersteller (aktueller Benutzer)
    • Datum/Zeit der Erstellung
