Use Cases : Ressourcen Tooltipp

Created by Former user, last modified by Former user on Apr 30, 2014

Der Inhalt des Ressourcen Tooltipp kann global definiert werden - siehe dazu Konfigurationsparameter / Configuration parameter

Dabei können die statischen Informationen mehrsprachig abgelegt werden.

Bilder

Ab Release 4.3 stehen auch Bilder 

Unterstützte Image Typen: png, gif, jpg

Beim Mouseover über das Thumb wird das Bild in Originalgrösse  angezeigt.

 

Anzeige:

die Tooltipp werden wie folgt angezeigt:

  • Mouse Over über Info Icon

  • Mouse Over über Ressourcen Bezeichnung
  • Mouse Over über Info Icon

nur wenn Ressource nicht besetzt

  • Mouse Over über Hotspot (Polygon)
