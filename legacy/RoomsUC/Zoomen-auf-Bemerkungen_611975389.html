Zoomen auf Bemerkungen    Umsetzung, Ablauf

  1. Neben jedem Bemerkungsfeld wird ein Zoom Icon implementiert (Lupe)
  2. Mit Klick auf das Zoom Icon wird ein editierbares Popup geöffnet.
  3. Der Benutzer kann den Text ändern und mittels Übernehmen Button zurück in das Originalfeld speichern.

    Regeln:
  4. Gilt für alle Bemerkungsfelder in der Reservation, Anlass, Reservationsanfrage, etc.
  5. Explizit ausgenommen sind folgende Bereiche:
    1. Notifikationen
    2. Informatioinen
    3. Action Popup
