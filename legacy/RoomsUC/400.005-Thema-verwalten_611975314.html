Ziel & Beschreibung

Die Erfassung und Verwaltung von Kundenspezifischen Layout Anpassungen. Änderungen von CSS und Bilddateien

Rechte

  • Darf das Thema verwalten

Bedingungen

–

Hauptszenario in Schritten

  1. Nutzer navigiert auf Einstellungen >System >Oberfläche
  2. Nutzer gibt Suchkriterien ein (optional) und klickt Button „Finden"
  3. Ein Suchresultat über Themaänderungen wird angezeigt - ein oder mehrere Themaänderungen können gelöscht werden
  4. Nutzer kann einen Eintrag anklicken, um die Details anzuzeigen (View Modus)
  5. Nutzer editiert oder erstellt ein Themeneintrag

Schritt 3 - Ein Suchresultat über alle erfassten Themaänderungen wird angezeigt – ein oder mehrere Themaänderungen können gelöscht werden

Controls

Smartlist "Themenliste"
Standard Spalten:

  • Checkbox (für Mehrfachselektion)
  • Datei Pfad (mit View Link)
  • Bearbeiten Icon
  • Button: Entfernen
  • Button: Neu

Regeln

Löschen

  1. Mehrfachselektionen möglich
  2. Klick auf Löschen
  3. Warnhinweis
    1. Bestätigungsmeldung im Actionpanel (inkl. Anzahl der gelöschten Einträge) ODER
    2. Fehlermeldung im Actionpanel

WICHTIG: Wird ein Eintrag gelöscht ist automatisch der Standard wieder gültig (Standardmässig hinterlegtes Bild oder CSS)

Schritt 5 - Nutzer editiert oder erstellt einen Themeineintrag

Controls

Inhaltsbereich – Tab „Stammdaten“

  • Datei Pfad (ab Content Verzeichnis)
  • File Daten** Datei-/Verzeichnis-Browser (für Bild)
    • Upload-Button (für Bild)
  • Button: Speichern
  • Button: Abbrechen

Regeln

Datei Pfad

  • Exakter Ablagepfad der zu übersteuernden CSS oder Bilddatei
  • Pfad ab (nach)  "Content" erfassen
  • Ohne Slash (/) starten
  • Gross-Kleinschreibung irrelevant

File Daten

  • Über Browse Button wird die zu ersetzende Datei selektiert
  • Über Upload Button wird die Datei auf den Server geladen
  • Der Upload ist begrenzt. Begrenzung wird global festgelegt (siehe Konfigurationsparameter / Configuration parameter)
  • Der Upload wird visualsiert
  • Die Upload Datei wird nicht überprüft ob es sich um eine gültige Datei handelt

 

Unterstützte Formate

  • CSS
  • HTML
  • JS
  • JPG, JPEG, PNG, GIF
  • ZIP (beinhaltet alle Themenfiles)

Abbrechen-Button

  • geht zurück auf die Suchresultatliste oder View Mode (zurück zum Ursprungsort)

Button Speichern

  • Das Actionpanel bestätigt die Speicherung.
  • Zurück zum Ursprungsort
