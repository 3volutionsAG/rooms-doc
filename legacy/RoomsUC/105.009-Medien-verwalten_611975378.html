Use Cases : 105.009 Medien verwalten

Ziel & Beschreibung

Zu einer Ressource sollen verschiedene Medien (PDF, JPG, DOC, etc.) hinterlegt werden können, die dem Benutzer während des Reservationsprozesses notwendige Zusatzinformationen geban kann.

Rechte

 

Bedingungen

  • UC 105.001 Ressourcen verwalten muss in den Schritten 1, 2 und 5 ausgeführt worden sein
  • Medien Editieren ist nicht möglich – diese werden gelöscht und neu angefügt.

Hauptszenario in Schritten

  1. Benutzer navigiert auf Tab „Medien & Beilagen"
  2. System zeigt Liste mit erfassten Medien - eine oder mehrere Medien können gelöscht werden
  3. Benutzer kann Medien hinzufügen

Schritt 2 – System zeigt Liste mit erfassten Medien – ein oder mehrere Medien können gelöscht werden

Controls

Smartlist

Standard Spalten

  • Checkbox zur Selektion
  • Bezeichnung (in der Sprache des angemeldeten Benutzers)

Optionale Spalten

  • Bild/Datei (Thumbnail)

Button Bereich

  • Button „Hinzufügen“
  • Button „Löschen“

Regeln

Smartlist
Die Smartlist beinhaltet über jeder Spalte ein Suchfeld, falls aktiviert (siehe 000.001 Globale Funktionen)

Bezeichnung
Wird in der Sprache des angemeldeten Benutzers ausgegeben

Bild/Datei

  • Spalte Thumbnail ist nicht sortierbar
  • Thumbnail wird angezeigt
  • Klick auf ein Thumbnail öffnet sich das Bild (nur bei webtauglichen Bildformaten) in Vollansicht (Modal)

Hinzufügen
Siehe Schritt 3

Entfernen

  • Mehrfachselektion erlaubt
  • Sicherheitsabfrage
  • Medien werden aus Liste entfernt (gelöscht)
  • Bestätigungsmeldung im Actionpanel, inkl. der Anzahl der gelöschten Medien

Schritt 3 - Benutzer kann Medien hinzufügen

Controls

Modales Popup

  • Feld „Bezeichnung“
  • Feld „Bild/Datei auswählen“
  • Button „Browse…“
  • Button „Übernehmen“
  • Button „Abbrechen“

Regeln

Bezeichnung

  • Bezeichnung muss in der Anzahl der aktivierten Sprachen erfasst werden

Bild/Datei auswählen

  1. Dateipfad muss exakt eingegeben werden.
  2. Alternativ kann dieser Pfad erstellt werden, mittels Browse Funktionalität (Button „Durchsuchen“)
  • Prüfung der maximalen Upload Grösse
  • Bei Fehler: Ausgabe als Validierungsfehler

Button „Übernehmen“

  1. Media wird in Liste hinzugefügt
  2. Liste wird aktualisiert (noch nicht sichtbar)
  3. Falls es sich um ein Bild handelt wird ein Thumbnail in quadratischer Grösse generiert
  4. Andere Dateien werden Ihrer Endung entsprechend mit einem Symbol versehen.
  5. Nicht erkannte Dateiendungen werden mit einem neutralen Datei Icon versehen
  6. Bestätigungsmeldung im Actionpanel

Gültige Datei Formate
Grundsätzlich sollen möglichst alle Date Formate unterstützt werden. Minimal JPG, GIF, PNG, AVI, MPEG, MP3, Office Formate, PDF

Folgende Icons werden zur Verfügung gestellt:

  • Pdf
  • Doc
  • Xls
  • Ppt
  • Video
  • Audio
  • neutral
