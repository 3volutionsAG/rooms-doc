Ziel & Beschreibung

Erfassung und Administration eines Plans zu einem Standort.
Erstellung von Verknüpfungen zu weiteren Standorten und Ressourcen.

Vorgängig muss bereits ein Standort erfasst worden sein.

Rechte

erbt von 103.001 Standorte verwalten

  • Darf Plan sehen

Bedingungen

  • Standort wurde erstellt
  • Benutzer befindet sich im "Bearbeiten" Modus auf einem Standort

Hauptszenario in Schritten

  1. Nutzer wählt Tab „Stammdaten" auf dem Standort
  2. Nutzer kann einen Plan hochladen und hinzufügen, oder löschen
  3. Nutzer wählt Tab "Plan" auf dem Standort
  4. Nutzer kann einen erfassten Plan bearbeiten

Schritt 2 - Nutzer kann einen Plan hochladen und hinzufügen, oder löschen

Controls:

Tab „Stammdaten“ des Standorts

  • Browse-Feld: Datei-/Verzeichnis Browser (für Plan) mit Button: Durchsuchen
  • Button: Upload
  • Button „Clear“
  • Label „Maximale Dateigrösse:“ 
  • Zugewiesener Plan

Regeln:

Browse/Upload

  • Über den Browse Button wird ein Plan im Filesystem selektiert.
  • Das Dateiformat muss webtauglich sein (gif, jpg, png)
  • Über den Upload Button wird der Plan ins System geladen
  • Ist bereits ein Plan vorhanden wird dieser Überschrieben – die Hotspots bleiben bestehen.

Maximale Dateigrösse

  • Die maximale Dateigrösse wird auf Grund der Systemkonfiguration (webconfig) angezeigt (Standard 5MB)
  • Während des Upload wird in diesem Bereich der aktuelle Status in Form bereits hochgeladener Bytes angezeigt
  • Wird die maximale Dateigrösse überschritten gibt das System einen Fehler aus

Clear

  • Löscht den Plan
  • Löscht die gezeichneten Hotspots und die Verknüpfungen zu weiteren Plänen, Ressourcen

Zugewiesener Plan

  • Nach Upload wird der Plan als Thumbnail angezeigt.
  • Klick auf Thumbnail öffnet sich der ganze Plan als Popup

Schritt 4 - Nutzer kann einen erfassten Plan bearbeiten

Controls

Verknüpfungsoptionen

  • Option „Verknüpfung auf Ressource erstellen“ mit Intelibox „Ressourenpicker“
  • Option „Verknüpfung auf Standort erstellen“ mit Intelibox „Standortpicker“
  • Option "Verknüpfung auf Raumeinheit erstellen" mit Intelibox "Raumeinheitpicker"

Toolbox

  • Icon: Kreuz - für Navigation
  • Icon: Lupe - für Zoomen
  • Icon: Hand - für Verschieben
  • Icon: Halbrunder Pfeil - für Drehen
  • Icon: Quadrat mit Pfeilen - für Grösse ändern
  • Icon: Knoten - für Knotenpunkten ändern oder hinzufügen
  • Icon: Papierkorb - für Löschen des Polygons
  • Icon: Bereich löschen
  • Icon: Freihand: für Freihand Erstellung eines Polygons
  • Icon: Dreieck: für Dreieck Zeichnung
  • Icon: Vierreck für Viereck Zeichnung
  • Icon: Kreis für Kreis Zeichnung

Option Navigation
Mit der Funktionalität "Navigieren" kann der Plan in die richtige Position gebracht werden, um diesen zu bearbeiten:

  • Verschieben mit Maus ziehen
  • Verschieben mit Pfeiltasen
  • Verschieben über Navigationselement (ganz links im Plan)
  • Zoom mit +/- Tasten
  • Zoom mit Mausrad
  • Zoom über Navigationselement (ganz links im Plan)

Button Bereich

  • Button: Speichern
  • Button: Abbrechen

Regeln

Option „Verknüpfung auf Ressource erstellen“
Nach Klick auf diese Option wird folgende Funktionalität möglich

  1. Ressourcenpicker wird eingeblendet
  2. Wahl einer Ressource über Intelibox oder Ressourcenpicker
  3. Polygone mittels Koordinatenpunkte zeichnen (Mausklicks)
  4. Beendigung des Zeichnens mittels Doppelklick

Option „Verknüpfung auf Standort erstellen“
Nach Klick auf diese Option wird folgende Funktionalität möglich

  1. Standortpicker wird eingeblendet
  2. Wahl eines Standorts über Intelibox oder Standortpicker
  3. Polygone mittels Koordinatenpunkte zeichnen  (Mausklicks)
  4. Beendigung des Zeichnens mittels Doppelklick

 

Option „Verknüpfung auf Raumeinheit erstellen“
Nach Klick auf diese Option wird folgende Funktionalität möglich

  1. Raumeinheitpicker wird eingeblendet
  2. Wahl einer Raumeinheit über Intelibox oder Picker
  3. Polygone mittels Koordinatenpunkte zeichnen  (Mausklicks)
  4. Beendigung des Zeichnens mittels Doppelklick

Weiters

  • Auf der dem Rahmen der Fläche des Hotspots, wird dann auch der Status mittels entsprechender Farbe abgebildet, in welcher sich die verknüpfte Ressource momentan befindet.
  • Ausserdem werden die entsprechend erfassten Symbole/Icons je nach Status, im Form-Körper „eingemittet“ angezeigt
  • Die Fläche des Hotspots erscheint in der Farbe der Ressource (sofern definiert – siehe UC 105.001 Ressourcen verwalten - Bereich "spezifische Daten")
  • Informationen zu den verknüpften Ressourcen und Standorten sollen via Label und Tooltip analog zu "Plan" angezeigt werden. 214.001 Reservationen im Plan suchen und buchen (Schritt 4, Tooltips,  & Anhang)
