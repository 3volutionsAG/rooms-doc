Use Cases : Rich Intelibox Ressourcenart

Created by Former user, last modified by Former user on Jan 09, 2012

Die Intelibox Ressourcenart dient zur Wahl von einer oder mehreren Ressourcen Arten:

  • Als Dropdown (Selektion maximal einer Art) bei der Ressourcenreservation
  • Als Multiselekt Rich Intelibox bei den Listen Mittels Klick auf das Pop öffnet sich ein Popup in welchem die verfügbaren Ressourcenarten zur Auswahl stehen:

  1. Es werden nur Ressourcenarten angezeigt, bei welchen der angemeldete Benutzer auch über entsprechende Funktionsrechte verfügt.
  2. Klick auf "alle auswählen" erlaubt die Selektion aller verfügbaren Ressourcenarten
  3. Klick auf Close Button bricht Vorgang ab – keine Änderungen
  4. Klick auf Übernehmen übernimmt die Auswahl in die das Listfeld (Komma separiert)

    Weitere Bemerkungen:
    *Falls im Listfeld nicht alle gewählten Ressourcenarten angezeigt werden können enden dies mit drei Punkten und im Tooltip wird die volle länge angezeigt. Beispiel: (Meetingräume; Arbei...)
