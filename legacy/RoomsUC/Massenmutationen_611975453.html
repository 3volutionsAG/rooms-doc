Massenmutationen

Bei entsprechenden Funktionsrechten (Darf Massenmutationen durchführen) wird den Benutzern in den Listenansichten einen zusätzlichen Button „Massenmutationen" angezeigt.

Durch Anklicken min zweier Checkboxen wird der Button aktiv und auf Klick wechselt der Benutzer ins entsprechende Detail (z.B. Reservationsdetail, Ressourcen Stammdaten, etc.). Im Detail werden alle Editierbaren Felder ohne Inhalt mit einer Checkbox vor dem Feld angezeigt.Mittels Klick auf die Checkbox wird das Feld aktiviert und eine Eingabe kann erfolgen. Alle zuvor gewählten Listen Items werden nun (Klick auf Button „auf alle Übernehmen" mit dem eingegebenen Wert aktualisiert.
