Inteliboxen

Generelle Regeln für Inteliboxen

  • Globale Konfiguration der Anzahl der maximalen visualisierten Einträge (falls das Suchresultat grösser ist) – Standard: 15 (siehe dazu Konfigurationsparameter / Configuration parameter)
  • Suche wird ab dem Schreiben des ersten Zeichen ausgelöst
  • Kennzeichnung:
    • Jede Intelibox wird mit einem Icon gekenzeichnet
    • Jede Intelibox hat ein integrierter Hilfetext direkt in der Intelibox (leicht grau hinterlegt): Tippen für Hilfestellung
    • Die Implementation erfolgt bei folgenden Inteliboxen:
      • Standortpicker
      • Ressourcenpicker
      • OE Picker
      • Firma Picker
      • Personen Inteliboxen
  • Navigation:
    • Tabulator: normales Verhalten => springt aus dem Feld. Wenn möglich soll der aktuelle Eintrag der Intelibox selektiert sein.
    • Pfeiltasten zum Navigieren: ↑ = nach oben, ↓ = nach unten; ← innerhalb des Textes nach links und → innerhalb des Textes nach rechts
    • ENTER der markierte wird selektiert (mit Mausklick wird dasselbe erreicht)
    • Backspace: Zeichen löschen und neue Suche
  • Layout:
    • Suchresultat wird unterhalb der Intelibox dargestellt – wie aktuelles ROOMS PRO
  • Falls Text zu lang ist wird er abgeschnitten und mit drei Punkten ergänzt. Beispiel: Rothenbühler Tho...
    • Customized Tool Tip zeigt den ganzen Text. Beispiel Rothenbühler Thomas (Unit 100)
    • Tooltip ist auch auf dem selektierten Eintrag vorhanden.

Spezialfall Person

  • Tooltip kann um weitere Informationen angereichert werden (z.B. Telefonnummer). Dies steht immer im Kontext wo die Intelibox eingesetzt wird
  • Zusätzlich hat die Intelibox ein Personen-Info Avatar. Hat die selektierte Person ein Bild hochgeladen, wird dieses Bild als Mini-Thumb. angezeigt. Das Icon wird bei auch bei den nicht änderbaren Personen (z.B. Viewmode oder Ersteller angezeigt).
  • Klick auf das Icon hat folgende Funktion:
    • Ist das Feld "URL" aus dem Use Case 101.001 (Personen verwalten) nicht leer, wird der unter URL angegebene Hyperlink in einem neuen Fenster geladen.
    • Ist das Feld "URL" leer wird in einem modalen Popup der Viewmodus einer Person geladen.
  • Inhalt Intelibox: die angezeigten Spalten können global definiert werden (pro Kunde/Brand). Default ist: Nachname Vorname (OE)
  • Beim Mouse-Over über die Intelibox (oder Label) werden dem Benutzer zusätzliche Informationen zur Person in Form eines Tooltips angezeigt. Dies kann global pro Kunde/Brand definiert werden. Default ist: Telefon, Mobile, E-MailEs müssen Felder aus der Tabelle Person, Firma und FirmaPerson definiert werden können
  • Suche nach Personalnummer und nach Vorname Name und Name Vorname

  • Je nach Funktionsrecht kann nur nach Organisationseinheit oder nur nach Firma gesucht werden.
  • Inhalt Intelibox: die angezeigten Spalten können global definiert werden (pro Kunde/Brand). Default ist: OE/Firma. Der Avatar oder das Foto werden auch in der Intelibox angzeigt
  • Beim Mouse-Over über die Intelibox (oder Label) werden dem Benutzer zusätzliche Informationen zur Person in Form eines Tooltips angezeigt. Dies kann global pro Kunde/Brand definiert werden. Default ist: Strasse, PLZ, Ort
  • Wichtig: Wird der Inhalt oder das Mouse-Over der Intelibox mit Adressfeldern versehen, wird stets die Adresse angezeigt, welche als Standard definiert ist.

Multiselektoren

  • Wurde ein oder mehrere Einträge ausgewählt, werden diese wie bis anhin ins Control übernommen
  • Neu wird neben dem Eintrag ein Löschen-X (x) eingeblendet.
  • Der Eintrag kann mit Klick auf das X oder mit der Delete/Backspace Taste gelöscht werden.
  • Dies gilt für:
    • Standortpicker
    • Ressourcenpicker
    • Angebotspicker
    • Personen Inteliboxen
    • Benutzergruppen Inteliboxen
    • Gliederungen
    • Buchungsklassifikationen
    • Ressourcenartpicker
    • Organisationseinheitpicker
