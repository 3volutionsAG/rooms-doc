Use Cases : Farbpicker

Created by Former user, last modified by Former user on Jan 09, 2012

Einer Ressource kann eine Farbe zugewiesen werden. Diese wird dann in sämtlichen Ansichten angezeigt und dient als Orientierungshilfe. Die Zuweisung erfolgt innerhalb der „Ressource verwalten" im Tab „Erweiterte Angaben". Benutzer kann entweder einen HEX Wert in das Feld „Ressourcen Farbe" eingeben oder über den Picker eine Farbe
 
Visualisierung Listen
Es existiert eine separate (sortierbare) Spalte, welche den Farbcode in Form eines kleinen Rechteckes beinhaltet (siehe auch Kalender)
Visualisierung Kalender
Vor denRaum Bezeichnungen werden die Farbcodes dargestellt. Ist keine Farbeselektiert erscheint ein weisser Farbcode (siehe unten „Bärengraben")

Visualisierung Plan
Der Rahmen der entsprechenden Ressource, wird in der gewählten Farbe eingefärbt.Ist keine Farbe selektiert, ist kein Rahmen vorhanden (Ausnahmen Projektgruppe bei AP)
