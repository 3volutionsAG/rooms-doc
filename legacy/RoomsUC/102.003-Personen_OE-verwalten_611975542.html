Ziel & Beschreibung

Zuweisen von Personen auf eine OE (intern) und zuweisen von Personen auf eine Firma (extern). Vorgängig muss der Use Case 102.001 in den Schritten 1, 2, 3, 4 und 5 ausgeführt worden sein. Alternativ können/werden Personen zu den OE/Firmen über den Benutzerdaten Import eingelesen (UC 301.1).

Rechte

  • Entsprechendes Recht aus 102.001 Abteilung (Firmen) verwalten (Darf Firmen verwalten (intern) oder Darf Firmen verwalten (extern)) 
  • Darf Firmen mit Personen verwalten

Bedingungen

–

Hauptszenario in Schritten

  1. Nutzer navigiert auf Tab „Personen"
  2. Nutzer kann Personen aus einer OE/Firma auswählen und entfernen
  3. Nutzer kann Personen zu einer OE/Firma hinzufügen

Beziehungen von/zu UC


Legende

  • Rote Schrift = MUSS Felder
  • Grüne Schrift = wird in einem Folgerelease umgesetzt

Schritt 1 - Nutzer navigiert auf Tab „Personen“

Controls

Sidepanel

Keine Änderung

Regeln

Tab ‚Personen‘ nur sichtbar, wenn Recht "Darf Firmen mit Personen verwalten" vorhanden ist

Bilder

 

Schritt 2 - Nutzer kann Personen aus einer OE/Firma auswählen und entfernen

Controls

TAB „Personen“ - Smartlist

Default

  • Checkbox zur Mehrfachselektion
  • Nachname
  • Vorname
  • E-Mail

Optional

  • Gültig von
  • Gültig bis
  • Sprache
  • MobileNummer
  • Telefon intern
  • Telefon extern
  • Fax
  • Personalnummer
  • Initialen
  • Importiert

Button Bereich

  • Button: Person hinzufügen
  • Button: Entfernen

Regeln

Smartlist
Die Smartlist verfügt Suchfeld oberhalb jeder Spalte

Entfernen

  1. Mehrfachselektion erlaubt
  2. Warnhinweis erscheint
  3. Verbindung von OE/Firma zur Person wird gelöscht
  4. Erfolgreiche Entfernung: Ausgabe einer Bestätigung im Action Panel, inkl. Anzahl der entfernten Personen

Person Hinzufügen
Person kann zu OE hinzugefügt werden (siehe Schritt 3)

Bilder

 

Schritt 3 - Nutzer kann Personen zu einer OE/Firma hinzufügen

Controls

Inhaltsbereich TAB „Personen“ (unter OE/Firma)

Smartlist – modales Popup:

Default

  • Checkbox zur Mehrfachselektion
  • Nachname
  • Vorname
  • E-Mail

Optional

  • Initialien

Fussbereich

  • Gültig Von
  • Gültig Bis
  • Telefon intern
  • Telefon extern
  • Fax

Button Bereich

  • Button: Person Hinzufügen
  • Button: Abbrechen

Regeln

Smartlist

  • Filter: es werden Personen angezeigt, welche der gewählten OE/Firma noch NICHT zugeordnet worden sind

Person Hinzufügen

  • Mehrfachselektion erlaubt
  • Optionale Eingabe der Gültigkeitsdauer
  • Optionale Eingabe der Felder „Telefon intern“, „Telefon extern“, „Fax“
  • Popup wird geschlossen
  • Smartlist wird aktualisiert, um die zugefügten Personen
  • Erfolg: Ausgabe einer Bestätigung im Action Panel, inkl. Anzahl der hinzugefügten Personen
  • Nicht erfolgreich: Fehlermeldung im Action Panel
