Use Cases : 102.004 Kostenträger zu OE verwalten

Ziel & Beschreibung

Zuweisen von Kostenträger zu einer Firma/OE.
Zugewiesene Kostenträger sind während des Reservationsprozesses exklusiv für Organisatoren der entsprechenden Firma/OE selektierbar
Nicht zugewiesene Kostenträger sind während des Reservationsprozesses für alle Organisatoren sichtbar

Rechte

 

Bedingungen

Firma/OE muss vorgängig erstellt worden sein

Hauptszenario in Schritten

  1. Nutzer navigiert auf Tab „Kostenträger"
  2. Nutzer kann eine oder mehrere Kostenträger von Firma/OE entfernen
  3. Nutzer kann neuen Kostenträger zu Firma/OE hinzufügen

Beziehungen von/zu UC


Legende

Schritt 2 - Nutzer kann eine oder mehrere Kostenträger von Firma/OE entfernen

Controls

Smartlist

Standard Spalten

  • Checkbox zur Mehrfachselektion
  • Nummer
  • Bezeichnung

Button Bereich

  • Button: Hinzufügen
  • Button: Entfernen

Regeln

Smartlist

  • Die Smartlist verfügt Suchfeld oberhalb jeder Spalte

Entfernen

  1. Mehrfachselektion erlaubt
  2. Warnhinweis erscheint
  3. Verbindung von OE/Firma zum Kostentraeger wird gelöscht
  4. Erfolgreiche Entfernung: Ausgabe einer Bestätigung im Action Panel, inkl. Anzahl der entfernten Personen

Hinzufügen

  • Kostenträger kann zu OE hinzugefügt werden (siehe Schritt 3)

Schritt 3 - Nutzer kann neuen Kostenträger zu Firma/OE hinzufügen

Controls

Modales Popup - Smartlist

Default

  • Checkbox zur Mehrfachselektion
  • Nummer
  • Bezeichnung

Optionale Spalten

  • Beschreibung

 

Button Bereich

  • Button: Kostenträger hinzufügen
  • Button: Abbrechen

Regeln

Generell

  • Filter: es werden ausschliesslich Kostenträger angezeigt, welche noch keiner anderen OE/Firma zugeordnet worden sind
  • Kostenträger, welche einer Firma/OE hinzugefügt worden sind, sind im Buchungsprozess exkl. für diese Firma sichtbar

Person Hinzufügen

  • Mehrfachselektion erlaubt
  • Popup wird geschlossen
  • Smartlist wird aktualisiert, um die zugefügten Personen
  • Erfolg: Ausgabe einer Bestätigung im Action Panel, inkl. Anzahl der hinzugefügten Kostenträger
  • Nicht erfolgreich: Fehlermeldung im Action Panel
