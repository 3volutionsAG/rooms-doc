Ziel & Beschreibung

Verwaltet die globalen Einstellungen des Systems

Rechte

  • Darf Konfiguration System verwalten

Bedingungen

  • Systemeinträge unter 000.004 Konfigurationsparameter im Detail beschrieben

Hauptszenario in Schritten

  1. Nutzer navigiert auf >Einstellungen >System >Globale Parameter
  2. Nutzer gibt Suchkriterien ein (optional) und klickt Button „Finden“
  3. Ein Suchresultat über Konfigurationseinträge wird angezeigt
  4. Nutzer kann einen Eintrag anklicken, um die Details anzuzeigen (View Modus)
  5. Nutzer editiert oder erstellt einen Konfigurationseintrag

Schritt 5 - Nutzer editiert oder erstellt einen Konfigurationseintrag

Controls

Inhaltsbereich – TAB „Stammdaten“

  • Label: Bezeichnung
  • Label: Beschreibung
  • Control: Wert (siehe Regeln)
  • Label: Standard
  • Label: Benötigtes globales Recht

Button Bereich

  • Button: Speichern
  • Button: Abbrechen
  • Button: Zurücksetzen

Regeln

Bezeichnung

  • Interne definierte Bezeichnung des Parameters
  • Einsprachig (Deutsch)
  • nicht änderbar

Beschreibung

  • Beschreibung des Parameters
  • Einsprachig (Deutsch)
  • nicht änderbar

Wert

  • Wert des Parameters
  • Änderbar
  • Je nach Parameter stehen folgende Werte zur Verfügung
    • Feld: Eingabefeld
    • Checkbox für True/False
    • Dropdown für Selektionen
    • Textarea für grössere Textmengen

Standard

  • Ursprungswert durch den Hersteller definiert
  • nicht änderbar

Benötigtes globales Recht

  • falls der globale Parameter in direktem Zusammenhang mit einem globalen Recht steht, wird hier darauf hingewiesen
  • Standard: leer
  • nicht änderbar

Button Speichern

  • Das Actionpanel bestätigt die Speicherung.

Abbrechen-Button

  • geht zurück zum Ursprungspunkt

Zürücksetzen

  • Stellt den Standardwert (Ursprungszustand) wieder her
  • Wert wird erst gespeichert, bei Klick auf  "Speichern"
