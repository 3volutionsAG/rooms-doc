Ziel & Beschreibung

Die Verwaltung von verschiedenen Benutzergruppen. Benutzergruppen können Personen und verschiedene Rollen zugewiesen werden.

Datenrechte

–

Funktionsrechte

  • „Darf Benutzergruppen verwalten“

Bedingungen

In den globalen Einstellungen muss zuerst der Modus zur Begrenzung der Serien eingestellt werden:

  • Begrenzung Serie nach Anzahl
    • Monaten
    • Iterationen

Hauptszenario in Schritten

  1. Benutzer navigiert unter >Einstellungen >Sicherheitsdaten für Personen >Benutzergruppen
  2. Benutzer gibt Suchkriterien ein (optional) und klickt auf Finden
  3. Suchresultat über alle gefundenen Benutzergruppen gemäss Suchkriterien wird angezeigt - ein oder mehrere Benutzergruppen können gelöscht werden
  4. Eine Benutzergruppe kann angezeigt werden (View Modus)
  5. Eine Benutzergruppe kann editiert werden oder neu erstellt werden
  6. Tab „Personen" - Liste mit allen zugewiesenen Personen wird angezeigt - eine oder mehrere Personen können entfernt werden
  7. Eine oder mehrere Personen können zur Benutzergruppe hinzugefügt werden
  8. Tab Notifikationen (siehe UC 107.001 Notifikationen verwalten)

Beziehungen zu UC

108.001 Workflows verwalten
201.004 Serienbuchungen erstellen
106.002 Angebotszonen verwalten
107.001 Notifikationen verwalten

Regeln

Benutzergruppen ID

  • Optionales Feld für die Erfassung einer eindeutigen Kennzeichnung einer Benutzergruppe
  • Wird oftmals in Zusammenhang mit Import Scripts genutzt

Bezeichnung wird in der Benutzersprache des angemeldeten Benutzers geladen (über T Icon) kann das Feld übersetzt werden.

Maximale Serien Iterationen (Begrenzt die Serienbuchung. Details 201.004 Serienbuchungen erstellen ).

  • Feld hat Standardwert: leer (bedeutet ausgeschalten)
  • Feld muss auf Ganzzahl überprüft werden
    Achtung: Globale Einstellung notwendig (ROOMS Konfiguration) bezüglich Prüfung
  • Anzahl Iterationen oder auf
  • Anzahl Monate überprüft werden

Berechtigungsgruppe
Die Benutzergruppe erscheint zur Auswahl im Bereich der Datenrechte (siehe 109.001 Datenrechte verwalten )

Notifikationsgruppe
Die Benutzergruppe erscheint zur Auswahl im Bereich der Workflows (siehe 108.001 Workflows verwalten ) und Angebotszonen (siehe 106.002 Angebotszonen verwalten – Angebotszonen verwalten)

Teilnehmergruppe
Diese Funktion ist noch nicht verfügbar

Benutzerkreise

  • Benutzer in Benutzergruppen mit dem Flag "Benutzerkreis" aktiviert, können im Buchungsprozess nur Buchungen für Benutzer dieser Gruppe erstellen (siehe 201.000 Ressourcen reservieren)

Checkin Einzelbuchungen

  •  definiert ob bei einem aktivierten Checkin Prozess auf einer Ressource der Benutzer auch wirklich einchecken muss
  • Details in UC 105.001 Ressourcen verwalten und 105.015 elektronisches Türschild konfigurieren


Speichern

  • Validierung der MUSS Felder
  • Bestätigungsmeldung im Actionpanel

Abbrechen
wechselt zurück aufs Suchresultat.

Schritt 6 – Tab „Personen“ – Liste mit allen zugewiesenen Personen wird angezeigt – eine oder mehrere Personen können entfernt werden

Controls

Tab Personen - Smartlist

Standard

  • Checkbox zur Mehrfachselektion
  • Name
  • Vorname

Optional:

  • E-Mail
  • Sprache
  • MobileNummer
  • Initialen

Button Bereich:

  • Button „Person hinzufügen“
  • Button „Entfernen“

Regeln

Entfernen

  • Es können eine oder mehrere Personen aus der Benutzergruppe entfernt werden.
  • Sicherheitsabfrage erscheint.
  • Verlinkung zwischen Person und Benutzergruppe wird gelöscht
  • Smartlist wird aktualisiert
  • Bestätigungsmeldung inkl. Angabe der Anzahl der entfernten Personen wird im Actionpanel ausgegeben.

Hinzufügen (siehe Schritt 7)
Es öffnet sich ein modaler Dialog, um eine oder mehrere Personen der Benutzergruppe hinzuzufügen.

Bilder


Schritt 7 – Eine oder mehrere Personen können zur Benutzergruppe hinzugefügt werden

Controls

Modales Popup „Personen“ - Smartlist

Standard

  • Checkbox zur Mehrfachselektion
  • Name
  • Vorname
  • E-Mail

Optional:

  • Initialen

Button Bereich:

  • Button „Person hinzufügen“
  • Button „Abbrechen“

Regeln

Generell

  • Es werden alle Personen angezeigt, welche dieser Benutzergruppe noch nicht zugewiesen sind

Hinzufügen

  • Es können mehrere Personen selektiert werden
  • der Modale Dialog wird geschlossen
  • Smartlist (siehe Schritt 6) wird aktualisiert
  • Bestätigungsmeldung inkl. Angabe der Anzahl der hinzugefügten Personen wird im Actionpanel ausgegeben.

Schritt 8 – Tab "Notifikationen" (siehe UC 107.001 Notifikationen verwalten )

Generell

Dieser Bereich wird im Detail in UC 107.001 Notifikationen verwalten beschrieben:

  1. Liste aller der Benutzergruppe zugewiesenen Notifikationen wird angzeigt (Notifikationstyp, Titel)
  2. Klick auf HInzufügen: Modales Popup öffnet sich
  3. Notifikationstemplate kann selektiert werden
  4. Felder können erfasst werden:
    1. CC-Mail
    2. Sprache
    3. Titel
    4. Body Text
  5. Notifikation kann gespeichert werden

Regeln

Generell

Es stehen folgende Templates zur Verfügung:

  • Bestellung neu
  • Bestellung Anpassung
  • Bestellung Annullierung
  • Bestuhlung Neu & Anpassung
  • Reservation Statuswechsel
  • Equipment Neu & Anpassung
  • Reservationsanfrage Neu
  • Reservation Reminder (global)
  • Bestellung Anlass (neu)
  • Bestellung Anlass (Update)
  • Bestellung Anlass (Annullierung)
  • Neu registrierter Benutzer (Forms Auth.)

  • Reservation - Provisorische Reservation Reminder
