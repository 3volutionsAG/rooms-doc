Ziel & Beschreibung

Status von Anlässen verwalten

Rechte

  • Darf Anlassstatus verwalten

Bedingungen

–

Hauptszenario in Schritten

  1. Nutzer navigiert auf Einstellungen >Stammdaten für Anllässe>Anlassstatus
  2. Nutzer gibt Suchkriterien ein (optional) und klickt Button „Finden“
  3. Ein Suchresultat über alle Status wird angezeigt – ein oder mehrere Status können gelöscht werden
  4. Nutzer kann einen Eintrag anklicken, um die Details anzuzeigen (View Modus)
  5. Nutzer editiert oder erstellt ein Status

Schritt 3 - Ein Suchresultat über alle erfassten Stauts wird angezeigt – ein oder mehrere Status können gelöscht werden

Controls

Smartlist mit Spalten
Standard Spalten:

  • Checkbox (für Mehrfachselektion)
  • Bezeichnung (View Link)
  • Bearbeiten Icon

Button Bereich

  • Button: Entfernen
  • Button: Neu

Regeln

Bezeichnung
Die -Bezeichnug wird in der Sprache des Benutzers angezeigt

Entfernen

  1. Mehrfachselektionen möglich
  2. Klick auf Löschen
  3. Warnhinweis
    1. Bestätigungsmeldung im Actionpanel (inkl. Anzahl der gelöschten Einträge) ODER
    2. Fehlermeldung im Actionpanel
  4. Refresh Smartlist
