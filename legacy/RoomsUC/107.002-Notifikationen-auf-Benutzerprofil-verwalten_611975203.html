Use Case Details

Schritt 1 – Benutzer navigiert auf Tab "Notifikationen"

Controls

Sidepanel
keine Veränderungen

Regeln

 

  • Benutzer navigiert in sein Profil auf "Notifikationen"

 

Schritt 2 – Benutzer aktiviert/deaktiviert Notifikationen und Reminder

Controls

 

Ressourcentyp: 

  • Tabellarische Liste nach Ressourcenarten
  • zusätzlich wird das elektronische Türschild aufgeführt

E-Mail

  • Checkbox pro Ressourcenart für Aktivierung/Deaktivierung der Notifikation
  • Checkbox für E-Mail Aktivierung des elektronisches Türschild

E-Mail Reminder

  • Zahlenpicker pro Ressourcenart für  Erinnerungsfeld (mit up/down Pfeilen) 

Outlook Sync

  • Checkbox pro Ressourcenart für die Standardmässige Aktivierung der Synchronisation
  • Checkbox für Synchronisation Aktivierung des elektronische Türschild

ICS Datei versenden

  • Checkbox pro Ressourcenart für die Standardmässige Versendung der ICS Datei

 

Bezeichnung

  • Den Serie Reminder in Tagen angeben. Der Wert 0 bedeutet, dass der Reminder deaktiviert ist.

Serie Reminder in Tagen

  • Zahlenpicker pro Ressourcenart für  Serie Reminder (mit up/down Pfeilen)

Regeln

E-Mail Aktiviert

  • Durch Selektion einer Checkbox wird die Notifikation für die entsprechende Ressourcenart aktiviert
  • Es werden nur Checkboxen für die verfügbaren Ressourcenarten (Recht muss vorhanden sein) angezeigt
  • Soll eine am Türschild getätigte Reservation automatisch eine E-Mail an Ersteller/Organisator verschicken
    • Standard = nicht aktiviert

E-Mail Reminder

  • Durch Eingabe einer Zahl oder über Pfeiltasten kann die Minutenzahl verändert werden (Wieviele Minuten vorher wird die Erinnerung verschickt)
  • 0 bedeutet dass der Reminder ausgeschalten ist

Outlook Sync

  • nur ab Exchange 2007 möglich
  • Sync im Tab "Personen Informationen" aktiviert
  • die entsprechende Ressourcenart für diesen Benutzer aktiv ist (benötigt Recht - z.B. darf Ressourcentyp "Arbeitsplatz" buchen)
  • Soll eine am Türschild getätigte Reservation automatisch ins Outlook synchronisiert werden?

    • Standard = aktiviert

ICS Datei versenden

  • Wird in der Spalte "E-Mail aktiviert" eine Ressourcenart selektiert, kann die Ressourcenart in der Spalte ICS Datei auch selektiert werden. Ansonsten ist die Checkbox für Eingaben gesperrt.
  • Wird in der Spalte "E-Mail aktiviert" eine Ressourcenart deselektiert, muss der Hacken auch bei der ICS Datei entfernt werden und diese Checkbox wird wieder für Eingaben gesperrt.

Serie Reminder in Tagen

  • Durch Eingabe einer Zahl oder über Pfeiltasten kann die Anzahl der Tage verändert werden (Wieviele Tage vor Auslauf einer Serienbuchung, soll der Benutzer darauf hingewiesen werden)
  • 0 bedeutet dass der Reminder ausgeschalten ist
