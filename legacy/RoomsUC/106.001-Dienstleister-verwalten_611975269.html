Ziel & Beschreibung

Die Erfassung und Verwaltung von Dienstleister und deren Angebote (Angebotskategorien)

Datenrechte

–

Rechte (000.002 Recht)

  • Darf Dienstleister verwalten
  • Darf Angebote verwalten
  • Darf Dienstleister Angebote verwalten

Bedingungen

–

Hauptszenario in Schritten

  1. Nutzer navigiert auf >Einstellungen >Catering & Services >Dienstleister
  2. Nutzer gibt Suchkriterien ein (optional) und klickt Button „Finden"
  3. System liefert Suchresultat über alle erfassten Dienstleister - eine oder mehrere Dienstleister können gelöscht werden
  4. Nutzer kann einen Dienstleister anklicken, um ins Detail (Viewmodus) zu gelangen
  5. Nutzer kann Dienstleister (Stammdaten) editieren/neu erstellen
  6. Nutzer kann durch den Angebotsbaum navigieren und die weiteren Aktionen ausführen
  7. Nutzer kann eine Kategorie oder eine Angebot löschen
  8. Nutzer kann eine Kategorie editieren (bzw. neu erstellen)
  9. Nutzer kann ein Angebotspaket oder Angebot editieren (bzw. neu erstellen)

Beziehungen zu UC

Schritt 7 - Nutzer kann eine Kategorie oder ein Angebot löschen

Controls

Baumstruktur

  • Löschen Icon

Regeln

Generell
Recht „Darf Angebote verwalten” muss vorhanden sein, um zu löschen.

Bezeichnung
Wird in der Sprache des angemeldeten Benutzers ausgegeben

Löschen Icon (Kategorie und Angebot)

  1. Klickt man auf das Löschen Icon, dann erscheint zuerst ein Warnhinweis, welchen man quittieren muss.
  2. Kategorien, welche Angebote beinhalten, können nur dann gelöscht werden, wenn keine Abhängigkeiten zwischen Angeboten und Angebotszonen bestehen
  3. Die Löschung erfolgt rekursiv nach unten, insofern keine Abhängigkeiten mehr zu Angebotszonen bestehen (das heisst, nicht mehr in einer Angebotszone verwendet wird)
  4. Fehlermeldungen, bzw. Regelverletzungen werden im Actionpanel ausgegeben
  5. Bestätigungsmeldung wird im Actionpanel ausgegeben

Schritt 9 - Nutzer kann ein Angebotspaket oder ein Angebot editieren (bzw. neu erstellen)

Controls

Inhaltsbereich – Modaler Dialog

  • Feld: Bezeichnung
  • Textarea: Beschreibung
  • Angebotpicker: Angebotskategorie
  • Feld: Vorlaufdauer
  • Feld: Nachlaufdauer
  • Datei/Verzeichnisbrowser für Bild
  • Zugewiesenes Bild (Thumbnail)
  • Checkbox: Offene Menge
  • Checkbox: Überschreibbar
  • Feld: Artikel-Nr.
  • Dropdown: Kosten (mit Währung)
  • Feld: Kosten
  • Feld: Annullierungskosten
  • Feld: Start Annullierung in Minuten
  • Feld: Sortierung

Button Bereich

  • Button: Speichern
  • Button: Abbrechen

Regeln

Generell

  • Funktionsrecht „Darf Angebote verwalten” muss vorhanden sein, um zu editieren

Unterscheidung Angebotspaket und Angebot

  • Sobald einem Angebot, weitere Unterangebote zugeordnet werden, handelt es sich um ein Paket
  • Dabei gibt es nur eine Hierarchiestufe (Erfassung von Subangebot auf einem Angebot möglich, Erfassung von Subangebot auf Subangebot NICHT möglich)

Bezeichnung

  • Die Bezeichnung muss mehrsprachig erfasst werden

Beschreibung

  • Die Beschreibung muss mehrsprachig erfasst werden

Angebotspicker

  • Über den Angebotspicker kann ein Angebot (oder Paket) in eine andere Kategorie verschoben werden
  • Darstellung: Alle Kategorien und alle Angebotspakete
  • Default: Kategorie, bzw. Angebotspaket auf welchem das Angebot (Paket) erstellt wurde

Vorlaufdauer

  • Angabe in Minuten, wieviel vor Reservationsbegin ein Angebot noch bestellt werden kann
  • Im Reservationsprozess werden nicht mehr bestellbare Angebote ausgeblendet. Bei der Berechnung wird die Arbeitszeit des DL mitberücksichtigt (siehe UC 201.003 Angebote hinzufügen, entfernen)
  • Default: 0 (=keine Vorlaufdauer)

Nachlaufdauer

  • Angabe in Minuten, wieviel vor Reservationsbegin ein Angebot noch bestellt werden kann
  • Im Reservationsprozess werden nicht mehr bestellbare Angebote ausgeblendet. Bei der Berechnung wird die Arbeitszeit des DL mitberücksichtigt (siehe UC 201.003 Angebote hinzufügen, entfernen)
  • Default: 0 (=keine Vorlaufdauer)

Datei/Verzeichnisbrowser (für Bild)

  1. Mittels des „Browse“ Buttons kann nach einem Bild des Angebots im lokalen Filesystem gesucht werden
  2. Klick auf „Upload“ 
  3. Check ob max. Grösse überschritten (falls ja => Abbruch)
  4. das Bild wird in Originalgrösse in die Applikation geladen
  5. Generiert ein Thumbnail zur Vorschau (zugewiesenes Bild)
  6. Clear Button löscht das Bild

Zugewiesenes Bild
Klick auf das Thumbnail lädt das Bild in Originalgrösse in einem Vorschaufenster (popup)

Offene Menge

  • Erlaubt dem Nutzer Selektion der Dienstleistung ohne Angabe einer Menge
  • Anmerkung: Kosten bei offener Menge werden im Reservationsprozess nicht berechnet.

Überschreibbar (Details in UC 201.003 Angebote hinzufügen, entfernen)

  • Erlaubt dem Benutzer während des Bestellprozesses nach Selektion des Angebots dieses zu editieren.
  • Editert werden können Kosten und Angebotstext ändern

Artikel-Nr.

  • Alphanummerischen Eingabe von Artikelnummern möglich

Währung

  • alle aktivierten Währungen
  • Standardwert = Währung der im Profil des angemeldeten gewählten Kultur
  • die Anzahl der aktiviert Währungen ist abhängig von der Anzahl der zur Verfügung gestellten Kulturen

Kosten

  • Standardwert: leer (keine Kosten)

Annullationskosten

  • Standardwert = leer (keine Annullationskosten)
  • Wert > 0 entspricht dem Wert, welcher bei einer Annullation Pauschal erhoben wird

Start Annullation in Minuten vorher

  • Definiert ab wieviele Minuten vor Reservationsbegin Annullationskosten erhoben werden
  • 0 = Annullationskosten werden zur Laufzeit erhoben
  • Standardwert = leer (ausgeschalten)
  • Beispiel: Bei einem Wert von 60, wird bis 60 Minuten vor Reservationsstart KEINE Annullationskosten erhoben. Zwischen 60 Minuten und der Endzeit der Reservation wird der Pauschalbetrag, welcher unter "Annullationskosten" definiert wurde in Rechung gestellt.

Sortierung

  • Validierung auf nummerischen Wert
  • Gibt Sortierung im Baum auf gleicher hierarchischen Stufe vor
  • Definiert auch die Standardsortierung im Buchungsprozess (siehe auch 201.003 Angebote hinzufügen, entfernen)

Speichern

  • Validierung
  • Speichern des Eintrags
  • Wechsel auf Angebotsbaum
  • Angebotsbaum wird aktualsiert und neu geladen

Abbrechen

  • Zurück auf Baumstruktur
