Use Cases : 201.006 Kostenstellen bei Reservation zufügen und entfernen

Ziel & Beschreibung

Um während des Reservationsprozesses vordefinierte (alternative) Kostenstellen zur Auswahl zur Verfügung zu stellen, solle es möglich sein eine Liste von Kostenstellen in das System zu laden.
Dazu muss eine Kostenstellen Datei importiert werden (manuell oder durch Drittsystem).

Rechte

  • Darf Kostenträger anpassen
  • Darf Kostenträger ergänzen
  • Darf Kostenstellensplitt ändern

Bedingungen

  1. Sollen Kostenstellen zur Auswahl erscheinen, müssen diese vorgängig importiert worden sein (UC 105.007 Kostenstellen importieren und verwalten)
  2. Soll im Reservationsprozess eine Kostenstelle auf dem Organisator vorselektiert sein, muss diese im Profil der Person erfasst worden sein (UC 101.001 Personen verwalten)

Hauptszenario in Schritten

  1. Benutzer kann eine Kostenstelle einer Buchung zufügen
  2. Buchungsprozess: Benutzer kann eine zugewiesene Kostenstelle entfernen
  3. Buchungsprozess: Benutzer kann Kostensplitt der Kostenstellen verändern
  4. Buchungsprozess: Organisator wird gewechselt

Use Case Details

Schritt 1 - Benutzer kann eine Kostenstelle der Buchung zufügen

Controls

  • Intelibox „Alternative Kostenstelle“
  • Icon „Hinzufügen“

Regeln

Generell

  • Recht „Darf Kostenträger anpassen“ erlaubt die Selektion einer Alternativen Kostenstelle
  • Recht "Darf Kostenträger ergänzen" erlaubt die zusätzliche Erfassung freier Kostenstellen

Alternative Kostenstelle/Kostenträger anzeigen und auswählen

  • Feld kann leer bleiben
  • Hat der Organisator auf seinem Profil eine oder mehrere Kostenstellen zugeordnet, werden diese automatisch in die Reservation übernommen (inkl. Kostensplitt) - siehe UC 101.001 Personen verwalten
  • Gilt auch für Schnellreservationen, sofern die globale Einstellung "SchnellreservationPersönlicheKostenträger" = True ist (siehe Konfigurationsparameter / Configuration parameter)
  • Sind verschiedene "Geschäftsbereiche" vorhanden (diese Geschäftsbereiche können nur über einen Kostenträgerimport aktiviert werden) wird eine Dropdown mit der Auswahl der einzelnen Geschäftsbereiche angezeigt. Zudem steht die Auswahl "alle" zur Verfügung.
  • eine Kostenstellen kann durch (Teil-)-Eingabe eines Textes gefunden werden. Dabei werden folgende Felder durchsucht:
    • Kostenstellennummer
    • Kostenstellenbezeichnung
  • Es wird nur im vorher selektierten Geschäftsbereich gesucht - Ausnahme ist die Selektion "alle".
  • Es werden nur Kostenstellen gefunden, die keiner OE zugeordnet sind ODER der OE des gewählten Organisators zugeordnet wurden.
  • Es werden nur Kostenstellen angezeigt, welche zur vollständigen Laufdauer der Reservation auch Gültigkeit haben
  • Wird eine Kostenstelle gewählt, welche im System vorhanden ist, wird diese nach Selektion automatisch hinzugefügt
  • Hat eine Kostenstelle das Flag "istVerrechenbar" nicht selektiert wird der Kostensplitt ausgeblendet UND nicht berücksichtigt.
  • Alle selektierten und zugefügte Kostenstellen werden untereinander aufgelistet mit Kostenstellennummer, Kostenstellenbezeichnung und Geschäftsbereich

Hinzufügen Alternative Kostenstelle/Kostenträger

  • Erfassung einer zusätzlichen Kostenstelle benötigt das Recht "Darf Kostenträger ergänzen"
  • Klick auf das Hinzufügen Icon (Geld Symbol) ergänzt die Kostenstelle
  • Die zusätzliche Kostenstelle wird in der Tabelle "Kostenstellen" gespeichert und steht ab sofort zur Auswahl zur Verfügung
  • Dabei wird das Flag "istVerrechenbar" auf 1 (Standard) gesetzt (1=selektiert)

 

Speicherung erfolgt bei Buchungsabschluss

Schritt 2 - Buchungsprozess: Benutzer kann eine Kostenstelle entfernen

Controls

  • Icon „Papierkorb“ (Entfernen) – hinter jeder zugefügten Kostenstelle

Regeln

Funktionsrecht „Darf Kostenträger anpassen“ muss gewährt sein

Papierkorb

  • Klick auf Papierkorbsymbol entfernt alternative Kostenstelle von der Reservation
  • Der Kostensplitt wird neu berechnet
  • Keine Sicherheitsabfrage
  • Keine Bestätigung im Actionpanel

Bilder

 

Schritt 3 - Buchungsprozess: Benutzer kann Kostensplitt der Kostenstellen verändern

Regeln

Generell

  • Funktionsrecht: „Darf Kostenstellensplitt ändern“ muss vorhanden sein.
  • Grundsätzlich werden die Kosten  auf Grund der Anzahl der gewählten alternativen Kostenstellen prozentual aufgesplittet beim Hinzufügen einer Kostenstelle
  • Der Splitt kann manuell verändert werden, in dem die Zahlen angepasst werden.
  • Die Summe muss immer 100 ergeben, ansonsten kann die Reservation nicht abgeschlossen werden - die Fehlerausgabe erfolgt im Actionpanel
  • Es sind nur Ganz-Zahlen erlaubt. (Beim Hinzufügen rundet das System automatisch, also bei drei Kostenträger erhalten die ersten beiden den Wert 33 und der als letztes hinzugefügte den Wert 34)

Bilder

 

Schritt 4 - Buchungsprozess: Organisator wird gewechselt

Regeln

Organisator Wechsel

  • Wird der Organisator gewechselt,
  • werden die Kostenstellen auch geändert, sofern der neue Organisator NICHT dieselben Standard Kostenstellen hat.
    • alle Kostenstellen werden entfernt
    • neue Standardkostenstelle des neuen Organisators wird geladen

andernfalls bleiben die Kostenstellen bestehen (inkl. Splitt)
