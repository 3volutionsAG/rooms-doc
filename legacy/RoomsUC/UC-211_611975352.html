Wichtige Anmerkungen

Jede Liste (Grid) beinhaltet immer das Personalisieren von Spalten, Sortierung und Paging sowie optionale Checkboxen für Mehrfachselektion.

Jede Detailansicht beinhaltet immer einen Readonly-Modus, einen Editier-Modus für Einzelmutationen und einen Editier-Modus für Mehrfachmutationen.

Diese Funktionalitäten müssen deshalb beim Beschreiben der Use Cases nicht aufgeführt werden.

Zudem müssen Validierungsregeln zum aktuellen Zeitpunkt nicht beschrieben werden.

Grobbeschrieb

Bitte hier den Use Case im groben Beschreiben – auch Abhängigkeiten zu anderen Use Cases sind zu erwähnen:

Dieser Use Case befasst sich mit der Suche, Editieren und Löschen von Events:
Suchen

  1. Event muss unter „Meine Events“ des Event Owner erscheinen
  2. Event kann über eine Eventsuchmaske (analog Reservationssuche) gesucht werden, unter Verwendung der Suchkriterien der Eventstammdaten (Owner, Eventtyp, Datum/Zeit, Titel, Kostenstelle, Standort, etc.) und ev. Ressourcensuchkriterien (wie Ressourcenname, Adressat, Datum, Zeit, etc.)
  3. Event kann über die gängigen Ressourcensuche Instrumente gefunden werden (Kalender, Plan, Ressourcensuche, etc.) und dann über Ressourcendetail auf Link „Event“ klicken
    Anzeigen
  4. Suchresultat als Eventkalender (Default): zeigt den Event-Time-Stream mit allen Buchungen zu einem Event (grafische Darstellung)
    1. Klick auf Anzeigen im Event Header linkt direkt in die Details eines Events im Read-Mode
    2. Klick auf Editieren im Event Header linkt direkt in die Details eines Events im Edit-Mode
    3. Klick auf eine Buchung auf der Eventzeitlinie, wechselt direkt in den Read-Mode einer Reservation
    4. Klick auf das X einer Buchung auf der Eventzeitlinie löscht umgehend die Reservation (mit Sicherheitsabfrage)
  5. Suchresultat als Liste: zeigt eine Auflistung von Events mit den wichtigsten Angaben zu einem Event (Name, Owner, Datum/Zeit, Standort, etc.)
    1. Klick auf Anzeigen linkt direkt in die Details eines Event im Read-Mode
    2. Klick auf Editieren linkt direkt in die Details eines Events im Edit-Mode
      Detail
      Im Eventdetail
  • Kann der Event editiert werden (ev. vorgängiges Wechseln in Edit-Mode notwendig)
    • Neue Ressourcen hinzufügen
    • Ressourcen löschen
    • Dienstleistungen hinzufügen
    • Dienstleistungen löschen
    • Optional im Communication Center – Nachricht verschicken.
  • Kann der Event gelöscht werden
