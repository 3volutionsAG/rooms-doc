Use Cases : Detailansicht (View Modus)

Created by Former user, last modified by Former user on Jan 09, 2012

Im View Modus werden alle Inhalte aus dem Editermodus als nicht editierbare Übersicht dargestellt.Tabs aus dem Editier Mode werden als separate Bereiche untereinander dargestellt.Der Viewmodus blockiert den Datensatz NICHT.

Der View Mode verfügt immer über min. die folgenden Buttons:

  • Bearbeiten: Wechselt in den Edit Mode des Datensatzes
  • Abbrechen: Wechselt zurück zum Ausgangspunkt
  • Drucken: Druckt die aktuelle Seite (nur Inhaltsbereich) über die Druckfunktionalität des BrowsersBrowser Druckfunktionalität:
