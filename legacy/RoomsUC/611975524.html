Use Cases : 102.001 Abteilung (Firmen) verwalten

Ziel & Beschreibung

Finden einer OE (intern) und Ausgabe der Detailinformationen zur OE. Finden einer Firma (extern) und Ausgabe der Detailinformationen zur Firma. Bei entsprechender Berechtigung: Neuerfassung, Editieren, Löschen (inaktiv) einer OE. Bei entsprechender Berechtigung: Neuerfassung, Editieren, Löschen (inaktiv) einer Firma.
Alternativ können/werden OE/Firmen über den Benutzerdaten Import eingelesen (UC 301.01)

Rechte

  • Darf Sektion Einstellungen nutzen
  • Darf Firmen verwalten (extern)
  • Darf Firmen verwalten (intern)
  • Darf Rabatte auf Firma editieren

Bedingungen

–

Hauptszenario in Schritten

  1. Nutzer navigiert auf Menüpunkt >Einstellungen >Sicherheitsdaten für Personen >OE/Firmen
  2. Nutzer gibt Suchkriterien ein (optional) und klickt Button „Finden"
  3. System liefert Suchresultat den Kriterien entsprechend - eine oder mehrere OE können gelöscht werden.
  4. Nutzer kann Rekord anklicken, um zu den Detailinformationen zu gelangen oder einen bestehenden Eintrag löschen
  5. OE/Firma kann editiert oder neu erstellt werden

Beziehungen zu UC

Button Bereich

  • Button „Entfernen“

Regeln

Entfernen

  1. nur bei entsprechendem Recht (intern, extern)
  2. Mehrfachselektion erlaubt
  3. Warnhinweis nach Button Klick
  4. Löschen Regel:
    1. Die zur Firma/OE gehörenden Adressen werden dabei auch gleich gelöscht, sowie die Referenz/Verlinkung zu den Personen
    2. Die Personen selber werden NICHT gelöscht
  5. Aktualisierung der Smartlist
  6. Bestätigungsmeldung inkl. Anzahl der gelöschten OE/Firmen im Actionpanel

Neu

  • Eines der Rechte „Darf Firmen verwalten (extern)” oder “Darf Firmen verwalten (intern)” muss vorhanden sein.
  • Öffnet Detailbereich für die Neuerfassung einer OE (siehe Schritt 5)

Schritt 5 – OE/Firma kann editiert oder neu erstellt werden

Controls

Tab „Stammdaten“

  • Bezeichnung
  • Firmennummer
  • Art (Intern / Extern)
  • Kostenstellen-Code
  • Kostenstellen-Namen
  • Standardrabatt in %

Tab „Adressen“
Siehe 102.002 Adressen_OE verwalten – Schritt 2

Tab „Personen“
Siehe 102.003 Personen_OE verwalten – Schritt 2

Button Bereich

  1. Button „Speichern“
  2. Button „Abbrechen“

Regeln

Art

  • Steuert ob es sich um einen OE (Intern) oder Firma (extern handelt). Dieser Status gilt dann auch für die jeweils zugeordneten Personen
  • Flag kann nur geändert werden, wenn das Recht für den Neuen Wert vorhanden ist.

Standard Rabatt in %

  1. Recht „Darf Rabatte auf Firma editieren“ notwendig, um in den Stammdaten das Feld abzufüllen
  2. Das Feld muss auf Ganzzahlen ≤100 validiert werden. Minuszahlen sind nicht Zulässig
  3. Im Reservationsprozess (Detailreservation) wird der Rabatt in Form eines Labels ausgewiesen (hinter dem Kostenfeld)
  4. Bei Zeitänderung oder anderen Kostenänderung (Umstuhlung, Dienstleistungen, etc.) muss das Feld neu berechnet werden
  5. Dasselbe gilt auf dem Anlass. In der Detailansicht und der Summary wird das Label angezeigt.
  6. Auf der Summary Page wird dann der effektive Rabatt errechnet und ein Eintrag im Bereich der Rabatte generiert und entsprechend auf der Summary ausgewiesen.

Dieser Rabatt Record wird bei jeder weiteren Editierung der Reservation wieder gelöscht und bei Reservationsabschluss neu erstellt. So wird sichergestellt, dass immer der Stammrabatt über den Gesamtbetrag neu berechnet wird.

Abbrechen
Wechselt zurück auf den Viewmodus oder das Suchresultat

Speichern
Es darf nur eine Adresse als Standard definiert werden, ansonsten ist die OE/Firma NICHT speicherbar (siehe 102.001 Abteilung (Firmen) verwalten )
