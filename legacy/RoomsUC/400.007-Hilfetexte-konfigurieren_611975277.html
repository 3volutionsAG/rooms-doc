Use Cases : 400.007 Hilfetexte konfigurieren

Ziel & Beschreibung

HIlfetexte können pro Control erfasst und verwaltet werden

Rechte

  • Darf Hilfetexte verwalten

Bedingungen

–

Hauptszenario in Schritten

  1. Nutzer navigiert auf >Einstellungen >System >Hilfetexte
  2. Nutzer klickt "Editmodus aktivieren"
  3. Nutzer navigiert im System und erfasst/verwaltet Hilfetexte
  4. Nutzer klickt "Editmodus deaktivieren"
  5. Nutzer sieht Hilfetext

Beziehungen zu UC

 


Legende

  • Rote Schrift = MUSS Felder

Use Case Details

Schritt 1 - Nutzer navigiert auf >Einstellungen >System >Hilfetexte

Controls

Sidepanel

Navigation öffnet sich entsprechend

Regeln

Rechte müssen implementiert werden:

  • Darf Hilfetexte verwalten
 

Schritt 2 - Nutzer klickt "Editmodus aktivieren"

Controls

Sidepanel

  • Button „Editmodus aktivieren“

Regeln

Editmodus aktivieren

  • Klick auf Button
  • Button wechselt die Beschriftung auf "Editmodus deaktivieren"
  • Edit Modus ist aktiv

Bilder

 

Schritt 3 - Nutzer navigiert im System und erfasst/verwaltet Hilfetexte

Controls

Sidepanel, Inhaltsbereich, View Mode

  • Überall in der Applikation wird im Sidepanel, Inhaltsbereich und View Mode vor einem Label (Beschriftungsfeld) ein HIlfe Icon eingeblendet

Regeln

Hilfe Icon

  • Klick auf Icon
  • Modales Popup "Feldbezogene Hilfe" wird geöffnet
  • Pro aktivierte Sprache wird ein mehrzeiliges Eingabefeld angezeigt - keine Pflichtfelder
  • Klick auf Speichern
    • Speicherung des Hilfetextes
    • Schliessen des Popups

Bilder

 

Schritt 4 - Nutzer klickt "Editmodus deaktivieren"

Controls

Sidepanel

  • Button „Editmodus deaktivieren“

Regeln

Generell

  • Schritt 1 wurde nochmals ausgeführt

Editmodus aktivieren

  • Klick auf Button
  • Button wechselt die Beschriftung auf "Editmodus aktivieren"
  • Edit Modus ist nicht mehr aktiv

Bilder

 

Schritt 5 - Nutzer sieht Hilfetext

Controls

Sidepanel, Inhaltsbereich, View Mode

  • Überall in der Applikation
  • Wo ein Hilfetext in der Sprache des angemeldeten Benutzers hinterlegt wurde, wird das HIlfe Icon eingeblendet

Regeln

Hilfe Text ansehen

  • Klick auf Button (Hilfe Icon)
  • Modales Popup "Feldbezogene Hilfe" wird geöffnet
  • Hilfetext wird in der Sprache des angemeldeten Benutzers ausgegeben
