Diese Dokumentation beschreibt die Einrichtung und den Betrieb von Türschildern mit ROOMS.

{{% alert title="Voraussetzung" color="primary" %}}
Für die Verwendung von Türschildern ist eine gültige Türschild-Lizenz erforderlich.
{{% /alert %}}

## Einrichtung

Die Einrichtung eines Türschildes erfolgt in drei Schritten:

{{< cardpane >}}
  {{< card header="**1. System Person**" >}}
  Konfigurieren Sie eine System Person mit den notwendigen Rechten für den Türschild-Betrieb.
  
  [Zur Anleitung]({{< ref "tuerschildbenutzer" >}})
  {{< /card >}}
  {{< card header="**2. Ressource**" >}}
  Aktivieren Sie das Türschild für die gewünschte Ressource und konfigurieren Sie die Einstellungen.
  
  [Zur Anleitung]({{< ref "konfiguratonressource" >}})
  {{< /card >}}
  {{< card header="**3. Kopplung**" >}}
  Koppeln Sie das physische Türschild mit der konfigurierten Ressource.
  
  [Zur Anleitung]({{< ref "kopplung" >}})
  {{< /card >}}
{{< /cardpane >}}
