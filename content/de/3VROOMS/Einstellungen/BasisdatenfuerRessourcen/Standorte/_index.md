## Ansicht Standorte


Im rechten Feld sehen Sie eine Übersicht über die angelegten Standorte. Mit dem **+** im Pfad, öffnen Sie weitere Standorte.

{{< imgproc Standorte_erstellen Resize "960x" >}}
Standort erstellen
{{< /imgproc >}}

Mit dem untenstehenden Button erstellen Sie einen neuen Hauptstandort.

Klicken Sie auf einen Standortnamen erscheint neben dem Standort im Pfad eine Icon Toolleiste.

{{< imgproc Standort_Toolleiste Resize "960x" >}}
Toolleiste neben Standort
{{< /imgproc >}}

### Toolleiste neben dem Standort


Mit der Toolleiste können Sie einen bereits bestehenden Standort bearbeiten, die Daten einsehen oder löschen.

{{< bootstrap-table "table table-striped" >}}
| Feld         | Funktion         | Icon |
| ------------- |-------------  | -------------  |
| Plus         | Unterstandort zu bestehendem Standort hinzufügen | {{< imgproc Icon_hinzufügen Resize "100x" >}}{{< /imgproc >}} |
| Auge    |  Ansehen der Informationen zu gespeichertem Standort   |   {{< imgproc Icon_Ansehen Resize "100x" >}}{{< /imgproc >}} |
| Stift  |  Bearbeiten des bestehenden Standortes    |   {{< imgproc Icon_bearbeiten Resize "100x" >}}{{< /imgproc >}} |
| Papierkorb   | Löschen des aktuellen Standortes      |   {{< imgproc Icon_löschen Resize "100x" >}}{{< /imgproc >}} |
{{< /bootstrap-table >}}

### Unterstandort zu Hauptstandort hinzufügen


Mit Klick auf das entsprechende Icon öffnet sich das Feld, in welchem Sie die Informationen für einen neuen Unterstandort eintragen können.
Untenstehend finden Sie die Beschreibungen zu den einzelnen Feldern.

{{< imgproc Standort_hinzufügen Resize "960x" >}}
Unterstandort zu Hauptstandort hinzufügen
{{< /imgproc >}}

### Informationen und Daten zu Standort einsehen


Klicken Sie auf das Icon Auge, um die Informationen zu einem Standort einzusehen. Mit dem untenstehenden Button _Bearbeiten_ können Sie diese ebenfalls wieder bearbeiten.

Untenstehend finden Sie die Beschreibungen zu den einzelnen Feldern.

{{< imgproc Standort_ansehen Resize "960x" >}}
Informationen zu einem Standort einsehen
{{< /imgproc >}}

### Bestehenden Standort bearbeiten


Klicken Sie auf das Icon Stift, um die Informationen zu einem Standort zu bearbeiten.

Untenstehend finden Sie die Beschreibungen zu den einzelnen Feldern.

{{< imgproc Standorte_bearbeiten Resize "960x" >}}
Bestehenden Standort bearbeiten
