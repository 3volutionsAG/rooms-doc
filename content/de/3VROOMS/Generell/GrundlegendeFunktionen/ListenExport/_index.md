Alle Suchresultate und Listen können in einer .csv-Datei, z.B. für die Weiterverarbeitung in Excel, exportiert werden.
Dafür klicken Sie auf das Export-Symbol unten rechts unter dem Anzeigenbereich:

{{< imgproc Listenexport_Position Resize "1280x" >}}
Position der Schaltfläche um Listen als .csv-Datei zu exportieren
{{< /imgproc >}}

Das Actionpanel informiert Sie über den Start des Exports sowie des Bereitliegendes der Datei zum Downloaden.

{{< imgproc Listenexport_Downloadinfo_beschriftet Resize "1280x" >}}
Das Actionpanel informiert über den laufenden Export der Liste
{{< /imgproc >}}
{{< imgproc Listenexport_Downloadlink_beschriftet Resize "1280x" >}}
Das Actionpanel zeigt den Link für den Download der Liste an
{{< /imgproc >}}

Mit einem Klick auf den angezeigten Link im Actionpanel können Sie die .csv-Datei herunterladen und zur weiteren Verarbeitung auf Ihrem Rechner speichern. Sie können entweder direkt einen Speicherort auswählen oder die Datei wird in Ihrem Download-Ordner abgelegt.

Das Erstellen der Export-Datei läuft asynchron. Das bedeutet, dass weiterhin mit ROOMS gearbeitet werden kann, ohne dass der Export behindert wird. Dies bedeutet aber auch, dass während der Erstellung der Export-Datei weitere Einträge in ROOMS gespeichert werden könnten, die dann jedoch noch nicht in der Export-Datei aufgeführt werden würden.
